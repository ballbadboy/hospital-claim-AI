# Surgical Procedure Code Families (ICD-9-CM)

| ICD-9-CM Range | Category | Common Procedures |
|----------------|----------|-------------------|
| 01-05 | Nervous system (Neuro) | Craniotomy, VP shunt, DBS, spine decompression |
| 06-07 | Endocrine | Thyroidectomy, parathyroidectomy |
| 08-16 | Eye | Cataract, vitrectomy, retinal detachment repair |
| 18-22 | Ear/Nose/Sinus | Tympanoplasty, FESS, septoplasty |
| 25-29 | Mouth/Pharynx | Tonsillectomy, cleft repair |
| 30-34 | Respiratory | Lobectomy, VATS, chest tube, tracheostomy |
| 35-39 | Cardiovascular | CABG, valve replacement, vascular graft, PCI |
| 42-54 | Digestive (GI) | Appendectomy, cholecystectomy, colectomy, hernia repair |
| 55-59 | Urinary | Nephrectomy, TURP, PCNL, ureterscopy |
| 60-64 | Male genital | Prostatectomy, orchidectomy |
| 65-71 | Female genital (OB/GYN) | Hysterectomy, C-section, oophorectomy |
| 72-75 | Obstetric | C-section (74.x), repair lacerations |
| 76-84 | Musculoskeletal (Ortho) | TKR, THR, fracture ORIF, spine fusion, arthroscopy |
| 85-86 | Integumentary | Mastectomy, skin graft, wound debridement |
