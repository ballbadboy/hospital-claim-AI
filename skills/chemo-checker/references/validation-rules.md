# Chemo — 8 Checkpoints (Detail)

## Checkpoint 1: Cancer Diagnosis
- ICD-10 C00-C97 ต้อง specific ถึง site (C50.9 breast, C34.1 lung upper lobe)
- Morphology code ถ้ามี

## Checkpoint 2: Protocol Matching
- Regimen ต้องตรงกับ NHSO-approved protocol สำหรับ tumor type
- เช่น Breast C50 → AC-T, CMF, Herceptin (ถ้า HER2+)

## Checkpoint 3: Drug Fee Schedule
- ยาเคมีบำบัดเบิกตาม Fee Schedule (FS) ไม่ใช่ DRG
- มี ceiling ราคาต่อ regimen/cycle
- ยาที่ไม่อยู่ใน FS = เบิกไม่ได้

## Checkpoint 4: Cycle Tracking
- Cycle number documented (Cycle 1, 2, 3...)
- Cumulative dose recorded
- Interval between cycles appropriate

## Checkpoint 5: Pathology
- Biopsy/pathology report confirming cancer
- Morphology type documented

## Checkpoint 6: Staging
- TNM staging or disease-specific staging
- Staging must support chemotherapy indication

## Checkpoint 7: Lab Clearance
- CBC (ANC ≥ 1500, Plt ≥ 100K)
- LFT (bilirubin, AST/ALT within limits)
- RFT (creatinine clearance)

## Checkpoint 8: Special Tests
- Breast: HER2 test before Herceptin
- Lung: EGFR mutation / ALK rearrangement before targeted therapy
- Colorectal: KRAS/RAS before anti-EGFR
