# UCEP Policy — Universal Coverage Emergency Patients

## หลักการ
- ผู้ป่วยฉุกเฉินวิกฤต (Red/Orange triage) เข้า รพ.ไหนก็ได้
- รพ. ห้ามปฏิเสธ ห้ามเรียกเก็บเงิน
- สปสช. จ่ายตาม DRG ไม่เกิน 72 ชม. แรก

## เงื่อนไข
1. ต้องเข้าเกณฑ์ฉุกเฉินวิกฤต (triage Red/Orange)
2. Coverage 72 ชม. แรกนับจาก admission
3. หลัง 72 ชม. → refer กลับ รพ.ต้นสังกัด
4. ต้องขอ Emergency Authen Code ทันที
5. ใช้ UCEP project code (ไม่ใช่ UC ปกติ)

## การจ่ายเงิน
- จ่ายตาม DRG (AdjRW × Base Rate)
- ค่า implant/prosthesis จ่ายตามจริงถ้าอยู่ใน Fee Schedule
- ค่ายานอกบัญชี: เบิกไม่ได้

## After 72 Hours
- Refer กลับ รพ.ต้นสังกัดถ้าย้ายได้
- ถ้าย้ายไม่ได้ (unstable) → document เหตุผล + ขออนุมัติต่อ
- Billing หลัง 72 ชม. ใช้สิทธิ UC/SSO ปกติ
