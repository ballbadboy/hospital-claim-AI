# ER/UCEP — 8 Checkpoints

## Checkpoint 1: Basic Data
- HN, AN, arrival time, admit time, discharge time
- Chief complaint documented

## Checkpoint 2: UCEP Eligibility
- Triage level = Red or Orange → UCEP eligible
- Emergency criteria documented in medical record
- ถ้า Yellow/Green → ไม่เข้า UCEP (ใช้สิทธิ UC/SSO ปกติ)

## Checkpoint 3: 72-Hour Rule
- UCEP coverage = 72 hours from admission ONLY
- After 72 hrs → must refer to home hospital
- Calculate: discharge_time - admit_time ≤ 72 hours
- ถ้าเกิน 72 ชม. → split billing (UCEP 72hrs + regular after)

## Checkpoint 4: Emergency Authen Code
- Emergency Authen Code ต้องขอทันที (ไม่ต้องรอ)
- ถ้าไม่ได้ขอ → deny แน่นอน

## Checkpoint 5: Project Code
- UCEP ต้องใช้ UCEP project code โดยเฉพาะ
- ห้ามใช้ UC project code ปกติสำหรับ UCEP cases

## Checkpoint 6: ER→IPD Transition
- ER แล้ว admit ภายใน 24 ชม. → เบิกเป็น IPD เท่านั้น
- ห้ามเบิก ER + IPD แยกกัน (duplicate claim)

## Checkpoint 7: Documentation
- [ ] Triage assessment form (level + criteria)
- [ ] Vital signs on arrival
- [ ] Emergency intervention records
- [ ] Trauma scoring (ISS, RTS) ถ้ามี trauma
- [ ] D2B time ถ้าเป็น STEMI

## Checkpoint 8: Refer Protocol
- หลัง 72 ชม. → refer กลับ รพ.ต้นสังกัด
- Refer form ครบถ้วน
- Transport arrangement documented
