# Revenue Tracking Methodology

## Data Sources
1. FDH submission records
2. สปสช. payment statements
3. Hospital billing system (HIS)

## Claim Lifecycle
Submit → Accepted/Rejected → Processing → Paid/Denied → Appeal → Recovered/Final Deny

## Tracking Frequency
- Daily: new submissions, payments received
- Weekly: denial review, appeal status
- Monthly: full reconciliation, KPI report
