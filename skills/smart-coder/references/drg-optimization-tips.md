# DRG Optimization Tips

## 1. PDx Selection Matters
- Same patient, different PDx → different DRG → different RW
- Choose the condition that best represents the reason for admission

## 2. Code Specificity
- E11.9 (DM unspecified) → low weight
- E11.65 (DM with hyperglycemia) → slightly higher
- E11.10 (DM with ketoacidosis) → MCC = much higher

## 3. Sequencing Strategy
1. PDx → determines MDC (Major Diagnostic Category)
2. Procedures → determines surgical/medical DRG
3. CC/MCC → determines severity within DRG
4. Age → some DRGs split by age

## 4. Common Missed CC/MCC
- Malnutrition (E44.x) — often documented but not coded
- Acute kidney injury (N17.x) — Cr rise often missed
- Delirium (F05) — common in elderly, often not coded
- Pressure ulcer (L89.x) — stage 3-4 = MCC
