# Thai Clinical Coding Guidelines

## PDx Selection Rules
1. เลือก condition ที่เป็นเหตุผลหลักของ admission
2. ถ้ามาด้วย complication → complication เป็น PDx
3. ถ้าผ่าตัด → condition ที่เป็นเหตุผลของการผ่าตัดเป็น PDx

## SDx Ordering for DRG
1. MCC conditions first (highest impact on RW)
2. CC conditions next
3. Other conditions last
4. Order within same level: by clinical significance

## Common Mistakes
- ใช้ unspecified code (เช่น I25.9) แทน specific code (I25.10)
- ไม่ code post-op complications
- ไม่ code chronic conditions ที่ affect management
- PDx selection ไม่ตรง → DRG grouping ผิด
