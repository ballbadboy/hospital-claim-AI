# Common CC/MCC Optimization Patterns

## Pattern 1: Sepsis Documentation → MCC
- ถ้ามี positive blood culture + SIRS criteria → code A41.x + R65.20/21
- RW increase: +1.5-3.0

## Pattern 2: Acute Kidney Injury → MCC
- ถ้ามี Cr rise >1.5x baseline → code N17.x
- RW increase: +0.5-1.0

## Pattern 3: Respiratory Failure → MCC
- ถ้ามี ventilator + ABG showing failure → code J96.0x
- RW increase: +1.0-2.0

## Pattern 4: Post-op Complications → CC/MCC
- SSI (T81.4), DVT (I82.4), PE (I26.x)
- RW increase: +0.2-2.0

## Pattern 5: DM with complications → CC
- DM + CKD, DM + neuropathy, DM + retinopathy
- Use specific codes (E11.22, E11.40, E11.65)
- RW increase: +0.1-0.3
