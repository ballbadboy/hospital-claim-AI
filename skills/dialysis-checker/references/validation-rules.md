# Dialysis — 8 Checkpoints (Detail)

## Checkpoint 1: Diagnosis
- N18.5 (CKD Stage 5) or ESRD as primary/secondary Dx
- Nephrology evaluation documented

## Checkpoint 2: Modality
- HD = 39.95, CAPD = 54.98
- Must match billing modality exactly

## Checkpoint 3: Session Limit
- HD: typically 3 sessions/week, monthly limit per สปสช.
- Exceeding limit = auto deny

## Checkpoint 4: Authen Code per session

## Checkpoint 5: Drug Catalog
- EPO: dose + brand + GPUID must match
- IV Iron: type (Iron Sucrose/Ferric Carboxymaltose) + dose
- Heparin: standard vs LMWH

## Checkpoint 6: Documentation
- Vascular access type: AVF (39.27), AVG, Temporary catheter (39.93)
- Adequacy: Kt/V ≥ 1.2, URR ≥ 65%

## Checkpoint 7: Duplicate
- Same patient, same date, different hospital = deny both

## Checkpoint 8: Fee Schedule
- Dialyzer (single-use vs reprocessed)
- Blood tubing set
- All items must be in สปสช. Fee Schedule
