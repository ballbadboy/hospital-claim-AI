# NCD Diagnosis Codes & Monitoring
| Disease | ICD-10 | Annual Monitoring Required |
|---------|--------|--------------------------|
| DM Type 2 | E11.x | HbA1c, eye exam, foot exam, lipid, Cr |
| DM Type 1 | E10.x | Same as DM2 |
| Hypertension | I10-I15 | BP monitoring, basic labs, ECG |
| CKD | N18.1-5 | eGFR, urine albumin/protein |
| COPD | J44.x | Spirometry, inhalers |
| Asthma | J45.x | Peak flow, controller meds |
