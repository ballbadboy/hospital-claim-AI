## 2: Fee Schedule — OPD จ่ายตาม FS ไม่ใช่ DRG
## 3: Drug Catalog — ยา NCD match GPUID (Metformin, Amlodipine, etc.)
## 4: Authen Code — ทุก visit
## 5: Visit Frequency — บางบริการจำกัด 2 ครั้ง/คน/ปี
## 6: P&P Coding — ANC, vaccination, screening ต้อง code ADP ถูก
## 7: Annual Monitoring — DM: HbA1c + eye + foot, CKD: eGFR, COPD: spirometry
