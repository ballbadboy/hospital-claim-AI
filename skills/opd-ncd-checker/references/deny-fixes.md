# OPD/NCD — Deny Causes & Fixes

## 1: Drug Catalog Mismatch → แก้ GPUID ใน HIS ให้ตรง FS version (Recovery 90%)
## 2: Visit Frequency Exceeded → ตรวจสอบจำนวนจริง อุทธรณ์ถ้ามีเหตุผลคลินิก (Recovery 40%)
## 3: FS Item Missing → ตรวจ FS list ใช้ item code ที่ถูก (Recovery 85%)
## 4: Authen Code Missing → ขอย้อนหลัง (Recovery 50%)
## 5: P&P Coding Error → แก้ ADP TYPE + CODE ให้ตรง (Recovery 90%)
