# Drug Switch Guidelines

## Principles
1. **Therapeutic equivalence** — same active ingredient, same dose, same route
2. **Bioequivalence** — FDA/อย. approved generic
3. **Step-down approach** — start with highest-cost, highest-volume drugs
4. **Clinical committee approval** — PTC (Pharmacy & Therapeutics Committee)
5. **Monitor outcomes** — 3-month review after switch

## Cannot Switch (ยา narrow therapeutic index)
- Warfarin — stick with same brand
- Phenytoin — brand-specific dosing
- Cyclosporine — brand-specific bioavailability
- Levothyroxine — monitor TSH after switch

## Process
1. Identify top 20 highest-cost drugs
2. Check generic availability + อย. registration
3. Calculate potential savings
4. Present to PTC for approval
5. Implement switch with monitoring plan
6. Review outcomes at 3/6/12 months
