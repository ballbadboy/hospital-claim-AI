# NLEM — National List of Essential Medicines (Thailand)

## Categories
| Category | Description | Prescriber |
|----------|-------------|-----------|
| A | Hospital formulary — ใช้ได้ทุก รพ. | Any physician |
| B | Specialist drugs — ต้องสั่งโดยแพทย์เฉพาะทาง | Specialist only |
| C | Specialized center — ต้อง รพ. ที่มีศักยภาพ | Center + specialist |
| D | Research/compassionate — ต้องขออนุมัติ | Committee approval |
| E | สมุนไพร | As listed |

## Key Rules
- ยาใน NLEM (A-B) → เบิกได้ตาม สิทธิ UC/SSO
- ยานอก NLEM → เบิกไม่ได้ (ยกเว้นได้อนุมัติพิเศษ)
- GPO drugs = ราคาต่ำสุด guaranteed
