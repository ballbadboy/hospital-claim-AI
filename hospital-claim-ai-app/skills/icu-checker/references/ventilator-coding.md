# Ventilator Coding Rules

## ICD-9-CM Codes
| Code | Description | DRG Impact |
|------|-------------|-----------|
| 96.71 | Mechanical ventilation <96 hours | Moderate |
| 96.72 | Mechanical ventilation ≥96 hours | Very High (Pre-MDC) |

## Calculation
- Start: intubation time (or initiation of mechanical ventilation)
- End: extubation time (or discontinuation)
- Total = cumulative hours across ALL episodes
- Reintubation: ADD hours to previous total

## Critical Threshold: 96 Hours
- <96 hrs → 96.71 → regular DRG grouping
- ≥96 hrs → 96.72 → Pre-MDC tracheostomy/vent group → RW MUCH higher
- Difference can be 2-5x RW

## Documentation Required
- [ ] Exact intubation date/time
- [ ] Exact extubation date/time
- [ ] If reintubated: each episode start/end
- [ ] Ventilator settings (mode, FiO2, PEEP)
- [ ] Weaning attempts documented
- [ ] Reason for prolonged ventilation
