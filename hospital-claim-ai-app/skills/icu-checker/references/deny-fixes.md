# ICU/NICU — Common Deny Causes & Fixes

## Deny 1: ICU Admission Criteria Not Documented
**วิธีแก้:** เพิ่ม documentation ว่าทำไมต้อง admit ICU (hemodynamic instability, respiratory failure, etc.)
**Recovery:** 75%

## Deny 2: Ventilator Days Mismatch
**วิธีแก้:** ตรวจสอบ intubation/extubation time → คำนวณ cumulative hours → แก้ code 96.71/96.72
**Recovery:** 85%

## Deny 3: CC/MCC Under-coded
**วิธีแก้:** Review medical record → เพิ่ม CC/MCC codes ที่มี documentation support
**Recovery:** 90%

## Deny 4: LOS Outlier Without Justification
**วิธีแก้:** Document เหตุผลที่ LOS ยาว (complications, failed weaning, etc.)
**Recovery:** 70%

## Deny 5: NICU Birthweight Not Documented
**วิธีแก้:** ตรวจ delivery record → เพิ่ม birthweight ใน claim
**Recovery:** 95%
