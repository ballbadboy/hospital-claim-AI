# Dialysis Module — ไตเทียม HD/CAPD

## Key Deny Causes
1. Session limit exceeded per month
2. Authen Code missing per session
3. Wrong modality code (HD vs CAPD vs PD)
4. Duplicate claims between hospitals
5. Drug catalog mismatch (EPO, Iron, Heparin)

## Modality Codes
| Modality | ICD-9-CM | Payment |
|----------|----------|---------|
| Hemodialysis (HD) | 39.95 | Per session |
| CAPD | 54.98 | Per month |
| Peritoneal dialysis (acute) | 54.98 | Per session |

## Required Documentation
- CKD Stage 5 (N18.5) or ESRD diagnosis
- Nephrology evaluation
- Vascular access type (AVF/AVG/temporary catheter)
- Adequacy monitoring (Kt/V, URR)
- EPO/Iron administration records with dose

## Fee Schedule Items
- HD session fee (fixed rate per session)
- Dialyzer (single-use vs reprocessed)
- Blood tubing set
- EPO injection (specify dose + brand)
- IV Iron (specify type + dose)
- Heparin

## ฟอกไตฟรีทุกแห่ง Policy
- ผู้ป่วยเลือกฟอกไตที่ รพ. ไหนก็ได้
- ห้ามเรียกเก็บค่าบริการเพิ่มทุกกรณี
- ผู้ป่วยเก่าจ่ายเงินเอง → ยื่นอุทธรณ์เปลี่ยนวิธีได้
- ต้องผ่านการประเมินโดยคณะแพทย์ + กรรมการ

## Dx-Proc Match
- N18.5 (CKD stage 5) + 39.95 (HD) → Dialysis DRG
- ถ้ามี N17.x (AKI) → code AKI as PDx, CKD as SDx
- Vascular access procedures: 39.27 (AVF creation), 39.93 (catheter insertion)
