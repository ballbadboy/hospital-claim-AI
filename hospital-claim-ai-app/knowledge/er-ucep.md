# ER/UCEP Module — ฉุกเฉิน

## UCEP (Universal Coverage Emergency Patients)
- ผู้ป่วยฉุกเฉินวิกฤตเข้า รพ. ไหนก็ได้ **72 ชม. แรก**
- จ่ายตาม DRG ไม่เกิน 72 ชม. จาก admission
- หลัง 72 ชม. → refer กลับ รพ. ต้นสังกัด
- **ต้องใช้ UCEP project code** ไม่ใช่ UC ปกติ

## Key Deny Causes
1. ไม่เข้าเกณฑ์ฉุกเฉินวิกฤต (triage level ไม่ document)
2. เกิน 72 ชม. UCEP window
3. Refer protocol ไม่ครบ
4. Emergency Authen Code ไม่ได้ขอ
5. Wrong project code (ใช้ UC แทน UCEP)

## Triage Criteria (ต้อง document ทุกเคส)
- **Red (resuscitation):** cardiac arrest, severe shock, airway obstruction
- **Orange (emergency):** chest pain, stroke symptoms, major trauma, severe bleeding
- **Yellow (urgent):** moderate illness needing prompt care, fracture, acute abdomen

## ER → IPD Transition
- ถ้า ER แล้ว admit ภายใน 24 ชม. → เบิกเป็น IPD เท่านั้น (ไม่เบิก ER แยก)
- D2B time สำหรับ STEMI ที่มาทาง ER → document ครบ
- Trauma scoring (ISS, RTS) → document for severity justification
