## สรุปสำหรับ Cath Lab Claim Checker

### เงื่อนไขที่ต้องตรวจ (Checkpoint 3: Device Documentation)

1. **รหัสอุปกรณ์ (CODE)** ต้องตรงกับ catalog ของ สปสช.
2. **ICD-9-CM** ต้องมี procedure code ที่กำหนด (ส่วนใหญ่ต้องมี 0066)
3. **จำนวน (QTY)** ต้องไม่เกิน QTY_MAX
4. **ราคา (RATE)** ต้องไม่เกิน ceiling price
5. **BMS ใช้ 3606 / DES ใช้ 3607** — ถ้าสลับกัน = deny
6. **Stent count** ต้องมี extension code: 0045 (1 stent) / 0046 (2) / 0047 (3) / 0048 (≥4)

### ถ้าไม่ตรงเงื่อนไข → **Deny I04** (ทุกรายการ)
