# คู่มือเจ้าหน้าที่: ใช้ AI ช่วยทำ Claim Cath Lab

> สำหรับ: เจ้าหน้าที่ Billing / Medical Coder
> รพ.: พญาไทศรีราชา
> ใช้กับ: Claude Code ใน VS Code

---

## เปิดโปรแกรม

1. เปิด **VS Code**
2. กด **Ctrl+`** (เปิด Terminal)
3. พิมพ์ `claude` แล้ว Enter
4. รอจนขึ้น **Claude >** พร้อมใช้งาน

---

## งาน 1: ตรวจเคสก่อนส่ง e-Claim

### วิธีทำ

พิมพ์ในช่อง Claude แบบนี้:

```
ตรวจเคส Cath Lab:

AN: 69-03556
หญิง 65 ปี สิทธิ์ UC
Admit: 27/02/69  D/C: 01/03/69
Authen: PP1007991295

แพทย์วินิจฉัย: STEMI anterior
EKG: ST elevation V1-V4
Troponin: 15.2
Echo EF: 35%
eGFR: 45
โรคร่วม: DM, HT, CKD

ทำ PCI ใส่ DES LAD 1 ตัว Xience 3.0x28mm Lot#AB123
D2B: 78 นาที
ยา: Heparin 7500U, Clopidogrel 600mg loading
```

### สิ่งที่ต้องเตรียม (ดูจาก chart)

| ดูจากไหน | ข้อมูลอะไร |
|----------|-----------|
| **ใบ Admit** | AN, HN, เพศ, อายุ, สิทธิ์, Authen Code |
| **Discharge Summary** | แพทย์วินิจฉัยอะไร, โรคร่วม |
| **Cath Report** | ทำอะไร (PCI/Cath), stent กี่ตัว ยี่ห้อ ขนาด Lot |
| **ผล Lab** | Troponin, eGFR/Cr, HbA1c, blood sugar |
| **Echo Report** | EF กี่ % (ถ้ามี) |

> **ไม่ต้องจำ ICD code** — AI จะแนะนำให้

### ผลที่ได้

AI จะตอบ:
- **ICD codes** ที่ควรใช้ (PDx, SDx, Procedures)
- **Score** ว่าพร้อมส่งไหม
- **สิ่งที่ต้องแก้** ก่อนส่ง (ถ้ามี)
- **เงินที่คาดว่าจะได้** กี่บาท
- **CC/MCC ที่ขาด** ถ้าเพิ่มจะได้เงินเพิ่มเท่าไหร่

### ตัวอย่างผลที่ AI ตอบ

```
PDx: I21.0 (STEMI anterior)
SDx: I50.21 (HF — MCC เพิ่มเงิน 23,000 บาท)
     E11.65 (DM — CC)
     N18.3 (CKD 3)
Proc: 36.07, 37.22, 88.56
DRG: 05291 | RW: 11.48
คาดว่าได้: 95,858 บาท

⚠️ ตรวจก่อนส่ง:
1. Lot# stent ตรง GPO VMI ไหม
2. Clopidogrel TMT code ตรงไหม
```

### หลังได้ผล

```
✅ ถ้า score ดี ไม่มี ❌  → ส่ง e-Claim ได้เลย
⚠️ ถ้ามี warning        → แก้ตาม AI แนะนำ แล้วส่ง
❌ ถ้ามี critical        → ห้ามส่ง! แก้ให้เสร็จก่อน
```

## งาน 2: เคสถูก Deny — วิเคราะห์หาสาเหตุ

### วิธีทำ

เปิด REP file จาก e-Claim แล้วพิมพ์:

```
เคสถูก deny:

AN: 69-03556
DRG: 05290  RW: 8.6544
ยอดที่ควรได้: 71,322 บาท

Deny codes: HC09, IP01, HC13
รายการที่ถูก deny: CLOPIDOGREL_DRUG, INST 31,490 บาท
```

### ผลที่ได้

AI จะตอบ:
- **สาเหตุที่ถูก deny** แต่ละ code หมายความว่าอะไร
- **วิธีแก้** ทีละขั้น
- **โอกาสได้เงินคืน** กี่ %
- **ควรทำอะไร**: แก้ส่งใหม่ / อุทธรณ์ / ปล่อย

---

## งาน 3: Upload CSV ทั้งไฟล์ (หลายเคส)

### วิธีทำ

1. Export REP file จาก e-Claim (ไฟล์ .csv)
2. วางไฟล์ไว้ในโฟลเดอร์ project
3. พิมพ์ใน Claude:

```
อ่านไฟล์ REP file นี้แล้ววิเคราะห์ทุกเคสที่ถูก deny:

/path/to/eclaim_11855_IP_25690316_085002242.csv
```

### ผลที่ได้

AI จะสรุป:
- **กี่เคสถูก deny** ยอดรวมเท่าไหร่
- **เรียงตามเงิน** แก้เคสไหนก่อนได้เงินเยอะ
- **แต่ละเคส** สาเหตุ + วิธีแก้

## งาน 4: ร่างหนังสืออุทธรณ์

### วิธีทำ

```
ร่างหนังสืออุทธรณ์:

AN: 69-03556
ผู้ป่วย: นาง ลังกา เทยณี
Admit: 27/02/69  D/C: 01/03/69
วินิจฉัย: STEMI anterior  ทำ PCI ใส่ DES
ถูก deny: HC09 เรื่อง stent document
ยอด: 71,322 บาท
ตรวจแล้วข้อมูลถูกต้อง มี GPO VMI ยืนยัน
ส่งถึง: สปสช. เขต 6
```

### ผลที่ได้

AI จะร่างหนังสือภาษาราชการให้ พร้อม:
- หัวเรื่อง + เรียน
- เนื้อหาชี้แจง
- รายการเอกสารแนบ
- ช่องลงนาม

**แพทย์ตรวจ → ผอ.เซ็น → ส่ง**

## งาน 5: ถามอะไรก็ได้

Claude รู้เรื่อง claim สปสช. ทั้งหมด ถามได้เลย:

```
DRG 05290 คืออะไร ได้เงินเท่าไหร่
```

```
HC09 หมายความว่าอะไร แก้ยังไง
```

```
ถ้า EF 35% ควร code ICD อะไร
```

```
Clopidogrel ถูก deny เพราะอะไรได้บ้าง
```

```
เคสนี้ควร appeal ไหม โอกาสสำเร็จเท่าไหร่
```

---

## จำไว้ 3 ข้อ

```
1. AI แนะนำ → คนตรวจทาน → คนตัดสินใจ
2. ทุก ICD code ต้องมีหลักฐานใน chart
3. ไม่แน่ใจ → ถาม AI เพิ่ม ดีกว่าเดา
```

## ปัญหาที่พบบ่อย

| ปัญหา | แก้ไข |
|-------|------|
| พิมพ์แล้ว AI ไม่ตอบเรื่อง claim | พิมพ์ว่า "เคส Cath Lab:" นำหน้าก่อน |
| AI แนะนำ code ที่ไม่เคยเห็น | ถาม "code นี้ต้องมี documentation อะไร" |
| ไม่รู้จะกรอกอะไร | กรอกแค่ที่มี AI จะถามเพิ่มเอง |
| CSV อ่านไม่ออก | ตรวจว่าเป็น UTF-8 encoding |
| AI ตอบภาษาอังกฤษ | พิมพ์ "ตอบภาษาไทย" |
