# ICU/NICU Module — ผู้ป่วยวิกฤต

## Key Deny Causes
1. ICU admission criteria not documented → ต้องบอกว่าทำไม admit ICU
2. Ventilator days mismatch → จำนวนวัน vent ต้องตรง billing
3. CC/MCC under-coded → เคส ICU มี comorbidities เยอะแต่ไม่ได้ code ครบ
4. LOS outlier → ICU LOS ยาวเกิน trim ต้อง justify
5. Step-down timing → ย้ายออก ICU ต้อง document

## Ventilator Coding (สำคัญมาก!)
- 96.71 = Continuous mechanical ventilation <96 hrs
- 96.72 = Continuous mechanical ventilation ≥96 hrs
- **≥96 hrs = DRG weight สูงขึ้นมาก (Pre-MDC tracheostomy/vent group)**
- Document: intubation time, extubation time, total ventilator hours
- Reintubation: count total cumulative hours

## NICU-Specific
- Birthweight determines DRG (Pre-MDC grouping for neonates)
- Weight categories: <750g, 750-999g, 1000-1499g, 1500-1999g, 2000-2499g, ≥2500g
- Document: gestational age, APGAR 1min + 5min
- Procedures: surfactant (99.29), phototherapy (99.83), exchange transfusion (99.01)
- Major neonatal problems: RDS, NEC, IVH, PDA → all affect DRG

## Severity Scores (document for justification)
- APACHE II/III (adult ICU)
- SOFA score (sepsis)
- GCS (neuro)
