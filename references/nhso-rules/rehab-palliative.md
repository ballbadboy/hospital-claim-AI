# Rehab & Palliative Care Module

## Rehabilitation
- Session-based payment (PT/OT/ST)
- ต้องมี treatment plan จากแพทย์เวชศาสตร์ฟื้นฟู
- FIM (Functional Independence Measure) score ต้อง document

## Procedure Codes
| Code | Service |
|------|---------|
| 93.11-93.19 | Physical therapy (PT) |
| 93.81-93.89 | Occupational therapy (OT) |
| 93.71-93.73 | Speech therapy (ST) |
| 93.83 | Recreational therapy |

## Key Deny Causes (Rehab)
1. **Session limit exceeded** → มี cap ต่อ diagnosis
2. **No physician order** → ต้องมีใบสั่งแพทย์ทุกครั้ง
3. **FIM score not documented** → ต้องมีทั้ง admission FIM + progress FIM
4. **PT/OT/ST code mismatch** → ต้อง code ตรงกับบริการที่ให้

## Documentation
- [ ] Rehab goals (short-term + long-term)
- [ ] FIM score at admission
- [ ] FIM score at each reassessment
- [ ] Treatment sessions with progress notes
- [ ] Discharge FIM score + outcomes
- [ ] Physician order for each session type

## Palliative Care
- ดูแลผู้ป่วยระยะท้าย
- **เบิกผ่าน e-Claim ได้ตามระบบปกติ** (สปสช. ยืนยันไม่ได้ตัดงบ)
- Home visit + home care services claimable
- Document: pain management plan, advance care planning, prognosis discussion
- Opioid prescribing: document indication + dose + assessment

## Common Rehab Diagnoses
| Condition | ICD-10 | Typical Sessions |
|-----------|--------|-----------------|
| Stroke rehab | I63.x, I61.x | PT+OT+ST, 30-60 sessions |
| Post-surgical | Z96.x, M96.x | PT, 10-20 sessions |
| Spinal cord injury | S14.x, S24.x | PT+OT, long-term |
| TBI | S06.x | PT+OT+ST, varies |
| Cardiac rehab | I21.x, I25.x | PT, 12-36 sessions |
