# ใบเบิกจ่าย

เอกสารที่หน่วยบริการสร้างขึ้นในระบบเพื่อขอชดเชยค่าบริการที่ให้ผู้ป่วยแล้ว ก่อนที่ สปสช. จะตรวจสอบและโอนเงิน

## ขั้นตอนการสร้างใบเบิกจ่าย (หน่วยบริการนวัตกรรม)

1. ให้บริการตรวจสุขภาพผู้ป่วย
2. **ปิดสิทธิ์** ในระบบกรุงไทย
3. **สร้างใบเบิกจ่าย** ในระบบกรุงไทย
4. ระบบรวบรวมใบเบิกจ่ายทั้งหมด → ตัดข้อมูลเที่ยงคืนวันอาทิตย์
5. ประมวลผลและออกรายงาน → ตรวจสอบใน [[concepts/seamless-for-dms]] วันจันทร์

## รอบเวลา

| ขั้นตอน | เวลา |
|--------|------|
| ให้บริการ | จันทร์–อาทิตย์ |
| สร้างใบเบิกจ่าย | ก่อนเที่ยงคืนวันอาทิตย์ |
| ตัดข้อมูล | เที่ยงคืนวันอาทิตย์ |
| รายงานออก | วันจันทร์ถัดไป |
