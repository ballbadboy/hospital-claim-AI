# MRA — คำนิยามและคำจำกัดความ

## ประเภทผู้ป่วย

### ผู้ป่วยนอกทั่วไป/ฉุกเฉิน (General case)
ผู้ป่วยที่มารับบริการ OPD หรือฉุกเฉินด้วยโรคทั่วไป ใช้ข้อมูลจาก visit ที่ต้องการตรวจ

### ผู้ป่วยนอกโรคเรื้อรัง (Chronic case)
โรคเรื้อรังที่ประเมิน: **เบาหวาน, ความดันโลหิตสูง, COPD, หืด, หัวใจ, หลอดเลือดสมอง**

- เป็นโรค > 3 เดือน → ใช้ข้อมูลการตรวจครั้งแรกของปีเป็น **First visit date**
- เป็นโรค < 3 เดือน → ใช้ข้อมูลจากครั้งที่ตรวจพบโรค

## คำศัพท์สำคัญ

| คำศัพท์ | ความหมาย |
|--------|---------|
| **First visit date** | วันที่ผู้ป่วยโรคเรื้อรังเข้ารับการตรวจครั้งแรกของปีปฏิทิน |
| **Principal diagnosis** | การวินิจฉัยโรคหลัก — ต้องมีเพียงโรคเดียว |
| **Comorbidity** | โรคร่วม — เกิดก่อน admit และส่งผลต่อการรักษา |
| **Complication** | โรคแทรก — เกิดขึ้นระหว่างการรักษา |
| **External cause** | สาเหตุจากภายนอก (อุบัติเหตุ, สารเคมี) |
| **Discharge status** | สถานะผู้ป่วยเมื่อจำหน่าย |
| **Discharge type** | ประเภทการจำหน่าย (กลับบ้าน, ส่งต่อ, เสียชีวิต ฯลฯ) |
| **LOS** | Length of Stay — จำนวนวันนอน |
| **Leave days** | วันที่ผู้ป่วยลากลับบ้านระหว่างนอน รพ. |
| **SOAP** | Subjective, Objective, Assessment, Plan — รูปแบบ progress note |
| **Marital status** | สถานภาพสมรส (โสด/สมรส/หม้าย/หย่า/แยกกันอยู่) |
| **EMR** | Electronic Medical Record — เวชระเบียนอิเล็กทรอนิกส์ |

## Discharge Status รหัส

| รหัส | ความหมาย |
|------|---------|
| 1 | กลับบ้าน (Discharged home) |
| 2 | ส่งต่อ (Transferred) |
| 3 | เสียชีวิต (Death) |
| 4 | หนีออกจาก รพ. (Absconded) |
| 5 | ขอกลับเอง (Discharged against advice) |

## หลักการให้คะแนน

- ให้คะแนนเกณฑ์ละ **1 คะแนน**
- ต้องมีข้อมูลครบตามเกณฑ์ที่กำหนดจึงได้คะแนน
- **NA** = ไม่จำเป็นต้องมีเอกสารนั้นในเคสนี้ (ไม่หักคะแนน)
- **No** = ต้องมีแต่ไม่มี หรือไม่ผ่านเกณฑ์ (หักคะแนน)
