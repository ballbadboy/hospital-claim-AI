# IN8 / CIN008 — ข้อผิดพลาดไม่พบรหัสหัตถการ (ICD)

## Definition

**IN8** (หรือ CIN 008) คือ Error Code ที่ระบบ สปสช. แจ้งเมื่อ **ไม่พบรหัสหัตถการ (ICD Procedure Code)** ในชุดข้อมูลที่หน่วยบริการส่งเข้ามาเบิกจ่าย ปัญหานี้พบบ่อยในหน่วยบริการแผนไทย กายภาพบำบัด และร้านยาที่ส่งข้อมูลผ่านโปรแกรม EHP หรือระบบ 13 แฟมป์มาตรฐาน

## ความหมายของ Error Message

ข้อความ "ไม่มีหัตถการทำฟัน" หรือ "ไม่มีรหัสหัตถการ" ที่ปรากฏใน error อาจทำให้เข้าใจผิด ความหมายที่แท้จริงคือ **ไม่พบรหัส ICD ของหัตถการประเภทนั้นๆ** ไม่ใช่เฉพาะทันตกรรมเสมอไป เช่น ถ้าเป็นคลินิกแผนไทยจะหมายถึงไม่พบ ICD หัตถการนวด/อบ/ประคบ

## สาเหตุที่พบ

1. **ข้อมูล ICD หายไปจาก 13 แฟมป์:** ชุดข้อมูลมาตรฐาน 13 แฟมป์ที่ส่งเข้ามามีทุกฟิลด์ครบ (PID, วันที่ ฯลฯ) แต่แฟมป์ที่เก็บ ICD Procedure ว่างเปล่า — อาจเกิดจากโปรแกรม SIS/HIS ของหน่วยบริการไม่ได้ insert รหัส ICD เข้าชุดข้อมูล
2. **ลืมใส่ ICD ตอนบันทึก:** เจ้าหน้าที่บันทึกข้อมูลครบทุกอย่างแต่ไม่ได้เลือกรหัสหัตถการ
3. **แฟมป์ที่ส่งเข้ามาไม่รองรับการส่ง ICD:** โปรแกรมต้นทางมีข้อจำกัดด้านโครงสร้างข้อมูล

## วิธีแก้ไข

1. ตรวจสอบการส่งออกข้อมูลจากโปรแกรมต้นทางว่า **แฟมป์ ICD Procedure** ถูก populate ก่อนส่ง
2. ถ้าพบว่าข้อมูลหายในระบบ สปสช.: แจ้งรหัสหน่วยบริการและรายละเอียดเพิ่มเติมผ่าน 1330 กด 5 กด 1
3. สปสช.จะตรวจสอบและถ้าพบว่าเป็นปัญหาจากฝั่งระบบ จะนำมา rerun ในรอบวันจันทร์ถัดไป

## กรณีร้านยา/แผนไทย ส่งผ่าน EHP

โปรแกรม EHP ที่ใช้ส่งข้อมูลแผนไทย/ยา — ถ้าติด CIN 008 ให้ตรวจสอบว่าในขั้นตอนบันทึกบริการมีการระบุรหัส ICD หัตถการครบหรือไม่ ก่อนกด submit ส่งออก
