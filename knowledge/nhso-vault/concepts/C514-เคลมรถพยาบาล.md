# C514 — เบิกพาหนะส่งต่อผิดเงื่อนไข

## Definition

**C514** คือรหัสปฏิเสธสำหรับกรณีการเบิกค่าพาหนะรับหรือส่งต่อผู้ป่วยที่ **ไม่มีรหัสสถานพยาบาลปลายทาง** หรือ **ไม่มีสิทธิ์ในการส่งเบิก** ตามเงื่อนไขที่กำหนด

## เงื่อนไขการเบิกพาหนะส่งต่อ

การเบิกค่าพาหนะส่งต่อ (รหัส S1801 และระยะทาง) จะต้องเป็นการส่งต่อ **ข้ามจังหวัด** เท่านั้น

> **ข้อสำคัญ:** หน่วยบริการต้นทางและปลายทางอยู่ในจังหวัดเดียวกัน → เบิกไม่ได้

## โครงสร้างรหัสค่าพาหนะ

```
รหัส S1801 = ค่าสตาร์ท 500 บาท
+ ระยะทาง (กม.) × 4 บาท/กม.
```

## ตัวอย่างกรณีที่ถูกปฏิเสธ C514

- โรงพยาบาลกบินบุรี (ปราจีนบุรี) ส่งต่อไปโรงพยาบาลเจ้าพระยาอภัยภูเบศร์ (ปราจีนบุรี)
- ทั้ง 2 หน่วยอยู่ในจังหวัดปราจีนบุรีเหมือนกัน → เบิกค่าพาหนะข้ามจังหวัดไม่ได้ → ติด C514

## วิธีตรวจสอบ

1. ตรวจสอบจังหวัดของหน่วยบริการต้นทางและปลายทาง
2. ถ้าอยู่จังหวัดเดียวกัน → ไม่สามารถเบิกค่าพาหนะตาม UC ได้
3. ตรวจสอบ "สิทธิ์ย่อย ณ วันรับบริการ" ว่าเป็น UC หรือไม่
