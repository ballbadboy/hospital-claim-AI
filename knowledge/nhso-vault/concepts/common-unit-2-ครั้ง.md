# Common Unit — ข้อเสนอจำกัดบริการ 2 ครั้ง/ปี (อยู่ระหว่างประชาพิจารณ์)

## Definition

"Common unit" (Common illness) คือกิจกรรมบริการที่ 3 ของหน่วยบริการนวัตกรรม ได้แก่ **การรักษาโรคเบื้องต้น** ซึ่งมีข้อเสนอจากบอร์ด สปสช. ให้จำกัดจำนวนครั้งที่จะได้รับการชดเชย เพื่อควบคุมงบประมาณ

## สถานะปัจจุบัน (ณ ธันวาคม 2568)

**ยังไม่มีการเปลี่ยนแปลง** — อยู่ระหว่างกระบวนการรับฟังความคิดเห็น (ประชาพิจารณ์) จากหน่วยบริการที่มีผลกระทบ:
- คลินิกเวชกรรม (Innovation clinic)
- คลินิกพยาบาลและการผดุงครรภ์

อัตราการจ่ายปัจจุบัน: **150 บาท/ครั้ง** ใช้ตามประกาศเดิม จนกว่าจะมีประกาศฉบับใหม่

## ข้อเสนอที่อยู่ระหว่างพิจารณา

บอร์ด สปสช. มีมติแนวทางให้ปรับปรุงประสิทธิภาพ ซึ่งรวมถึง:
- จำกัดการให้บริการ common unit **2 ครั้ง/คน/ปีงบประมาณ** (จากเดิมที่ไม่มีการจำกัดชัดเจน)
- ค่าเฉลี่ยการใช้บริการในหน่วยนวัตกรรมที่ตรวจสอบพบ: ~2 ครั้ง/คน

## ความกังวลของหน่วยบริการ

1. **ผู้สูงอายุและกลุ่มเปราะบาง** — มีความจำเป็นต้องรับบริการมากกว่าค่าเฉลี่ย
2. **อัตราชดเชย 180 บาท** — หลายหน่วยสะท้อนว่าต้นทุนจริง (ยา + พนักงาน + บริหาร) สูงกว่า
3. **การสื่อสารล่วงหน้า** — หน่วยบริการขอให้ประกาศนโยบายล่วงหน้าชัดเจนเพื่อเตรียมตัว
4. **พื้นที่ห่างไกล** — คลินิกต้องเลือกระหว่างช่วยผู้ป่วยกับข้อกำหนด

## จำนวนครั้งถ้าประกาศออก

หากมีการจำกัด 2 ครั้ง/ปี: **นับตามปีงบประมาณ** (ตุลาคม–กันยายน)

## ช่องทางให้ความคิดเห็น

สปสช. เปิดรับฟังความคิดเห็นทางออนไลน์ — ลิงก์ประกาศผ่านช่องทางอย่างเป็นทางการของ สปสช.

## บริการในคลินิกพยาบาล 3 กิจกรรม

| กิจกรรม | รายละเอียด |
|---------|-----------|
| 1. บริการพื้นฐาน | เช่น ล้างแผล |
| 2. บริการส่งเสริมป้องกัน (PP) | วัคซีน คัดกรอง ฯลฯ |
| 3. รักษาโรคเบื้องต้น (Common Unit) | **ตัวที่มีข้อเสนอจำกัดครั้ง** |

หมายเหตุ: บริการหลายรายการในวันเดียวกันอาจมีปัญหาการปิดสิทธิ์ซ้อน ระบบตรวจสอบจำนวนครั้งรายหน่วยบริการ
