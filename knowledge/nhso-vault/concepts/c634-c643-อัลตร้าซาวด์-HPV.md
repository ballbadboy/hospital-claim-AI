# C634 และ C643 — อัลตร้าซาวด์และ Liquid Base Cytology

## Definition

รหัสปฏิเสธ C634 และ C643 เกี่ยวข้องกับการตรวจทางพยาธิวิทยาและการตรวจอัลตร้าซาวด์สตรีมีครรภ์

## How It Works

### C634 — อัลตร้าซาวด์ไม่ถูกต้อง
เกิดจาก:
1. บันทึกการตรวจอัลตร้าซาวด์มากกว่า 1 ครั้งต่อครั้งที่กำหนด
2. ไม่มีการบันทึกหัตถการอัลตร้าซาวด์ หรือบันทึกไม่ถูกต้อง

รหัสบริการที่เกี่ยวข้อง: รหัส 30010 (ตรวจอัลตร้าซาวด์สตรีมีครรภ์)

### C643 — Liquid Base Cytology ไม่มีผล HPV ก่อน
**รหัสบริการ:** การตรวจทางพยาธิวิทยา (Liquid Base Cytology)

**สาเหตุ:** การเบิกค่า Liquid Base Cytology โดยไม่มีผลการตรวจ HPV DNA test มาก่อน

**หลักการ:**
- สตรี อายุ 30–60 ปี มีสิทธิ์รับการตรวจคัดกรองมะเร็งปากมดลูก
- ขั้นตอน: ตรวจ HPV DNA test ก่อน → ถ้าผลพบเซลล์ผิดปกติ (Human Papillomavirus สายพันธุ์ 16, 18 เป็นต้น) จึงทำ Liquid Base Cytology เพื่อยืนยัน
- **ไม่สามารถ** เบิก Liquid Base Cytology โดยไม่มีผล HPV DNA test มาก่อน

**วิธีแก้ไข C643:**
- ตรวจสอบว่ามีการบันทึกผล HPV DNA test แล้วหรือยัง
- หากมีผลแล้ว ให้แนบหลักฐานหรือแก้ไขข้อมูลให้ครบ
