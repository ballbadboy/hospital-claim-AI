# FVS — Face Verification Service (ระบบยืนยันตัวตนด้วยใบหน้า)

## Definition

**FVS (Face Verification Service)** คือระบบยืนยันตัวตนของผู้รับบริการด้วยการสแกนใบหน้า ผ่านเว็บ **NHSO FVS** หรือ **New Allen** ใช้สำหรับ **ปิดสิทธิ์** (ยืนยันการรับบริการ) เมื่อสิ้นสุดการรับบริการ ณ หน่วยบริการนวัตกรรม เริ่มบังคับใช้ตั้งแต่ **12 มกราคม 2569** สำหรับคลินิกพยาบาลและการผดุงครรภ์

## วิธียืนยันตัวตน (เรียงตามลำดับความสำคัญ)

### วิธีที่ 1: สแกนใบหน้า (Face Scan)
- แอปพลิเคชันบนเว็บ NHSO FVS
- ผู้รับบริการสแกนใบหน้าผ่านกล้องอุปกรณ์
- ข้อมูลใบหน้าอ้างอิงจาก **กรมการปกครอง**
- ใช้ได้กับผู้รับบริการส่วนใหญ่ รวมถึงผู้สูงอายุที่ใบหน้าไม่เปลี่ยนแปลงมาก

### วิธีที่ 2: Line OA สปสช.
- ผู้รับบริการ **จองล่วงหน้า** ผ่าน Line OA ของสปสช.
- ได้ **QR Code** มีอายุ **6 ชั่วโมง**
- นำ QR Code มาแสดงที่คลินิก → ระบบปิดสิทธิ์อัตโนมัติ
- เหมาะสำหรับ walk-in ที่ไม่สแกนใบหน้าไม่ผ่าน และผู้ที่จอง Line ล่วงหน้า

### วิธีที่ 3: เสียบบัตรประชาชน (Smart Card ผ่าน New Allen)
- เข้าเว็บ **New Allen (AENT Service)** → เลือก "HON Point" → กรอกเลขบัตรประชาชน 13 หลัก
- ใช้สำหรับ **กลุ่มพิเศษเท่านั้น:**
  - เด็กอายุน้อยกว่า 15 ปี (ผู้ปกครองเสียบบัตรแทน + เลือกความสัมพันธ์)
  - ผู้สูงอายุที่สแกนใบหน้าไม่ผ่านจริง
  - ผู้พิการด้านใบหน้า (กะพริบตาไม่ได้)
  - กลุ่มเปราะบางที่ไม่สามารถใช้ FVS หรือ Line ได้

## ข้อยกเว้นจากการยืนยันตัวตน

ไม่บังคับทำ FVS ถ้า:
- ผู้รับบริการ **อายุน้อยกว่า 15 ปีบริบูรณ์**
- **ผู้สูงอายุ** (ใช้ Line OA แทนได้)
- **จองล่วงหน้าผ่าน Line OA** สปสช. แล้ว
- มีเหตุจำเป็นที่ต้องแจ้งสปสช. เป็นรายกรณี

## รหัสประเภทบริการในระบบ FVS

| รหัส PG | บริการ | โปรแกรมเบิก |
|---------|--------|-------------|
| **PG 019002** | รักษาโรคเบื้องต้น (CI/C) | M Care Plus |
| **PG 019003** | พยาบาลพื้นฐาน/ทำแผล (OPD Nursing) | M Care Plus |
| **PG 006001** | บริการ PP | KTB/KTP |

**สำคัญ:** เลือกรหัส PG ให้ตรงกับบริการที่ให้จริง ถ้าเลือกผิดจะติด **Error Code 0019**

## Error Code 0019

เกิดจากเลือก Service Type ใน FVS/Line ไม่ตรงกับบริการที่บันทึกในระบบ AMET/M Care Plus วิธีแก้: เลือก PG Code ให้ถูกต้องก่อนยืนยันตัวตน

## ขั้นตอน FVS ใน M Care Plus

1. ให้บริการผู้ป่วยเสร็จสิ้น
2. ผู้ป่วยทำการสแกนใบหน้า (หรือ Line / New Allen)
3. ใน M Care Plus กดปุ่มปิดสิทธิ์ตามช่องทางที่ใช้ (สีเขียว = FVS/Line, สีฟ้า = New Allen)
4. ระบบดึง token/เลขปิดสิทธิ์ (EP code) ผ่าน API
5. บันทึกลงเวชระเบียนอัตโนมัติ
6. อัปเดตสถานะ → ส่งเคลม

## อัปเดต — ครั้งที่ 96 (10 เมษายน 2569)

### เคส IN19 ก่อน 12 มกราคม 2569
- สปสช.ออกรายงานจ่ายชดเชยเมื่อ **7 เมษายน 2569** ครอบคลุม **191 หน่วยบริการ**
- Service Date ย้อนหลัง 24 ธันวาคม 2568 – 11 มกราคม 2569
- ตรวจสอบผ่าน Sim for Dest → ดู IEP Date วันที่ 7 เมษายน

### กลุ่มพิเศษ (อัปเดต)
- เด็กอายุต่ำกว่า **7 ปี**: สแกนใบหน้าเพียง **1 ครั้ง** (ไม่ต้อง 2 ครั้ง) ถ้าไม่ผ่านใช้บัตรประชาชนแทน
- เด็กอายุต่ำกว่า **7 ปี ไม่ได้อยู่กับผู้ปกครอง**: ใช้ Smart Card + Proxy ยืนยันตัวตนแทน
- เด็กอายุต่ำกว่า **1 ปี**: ยังต้องผ่านกระบวนการ FVS (1 ครั้ง) ถ้าไม่ผ่านระบบบันทึก Log ไว้

### FVS ล่ม — นโยบาย Waive อัตโนมัติ
- สปสช.มีระบบสำรอง **4 ระบบ**
- หากระบบ สปสช.ล่มเอง จะ **Waive อัตโนมัติ** ไม่ต้องอุทธรณ์
- สปสช.จะย้อนหลังดูว่าช่วงไหนระบบล่ม แล้วดึงข้อมูลขึ้นมาจ่ายให้

### DS027 — ปิดสิทธิ์ย้อนหลังปัญหา
- หน่วยบริการแจ้งว่าปิดสิทธิ์ย้อนหลัง 30 วันแล้วติด **DS027** (ถูกปฏิเสธ)
- สปสช.รับทราบ กำลังตรวจสอบ → ดู [[concepts/DS027-ข้อผิดพลาด]]

## อัปเดต — ครั้งที่ 88 (13 กุมภาพันธ์ 2569)

### Smart Card Proxy สำหรับกลุ่มพิเศษ (คลินิกพยาบาล)
- คลินิกพยาบาล 2,000+ หน่วยเริ่มให้บริการ 9 ก.พ. 2569 วิธียืนยันตัวตนมี 3 ช่องทาง
- กลุ่มที่ใช้ Smart Card Proxy ได้: เด็ก ≤15 ปี, ผู้สูงอายุ ≥60 ปีที่ Face Scan ไม่ผ่าน, ผู้พิการด้านใบหน้า
- ขั้นตอน: เข้า New Auth (authenservice.go.th) → ยืนยันตัวตนแทน → เสียบบัตรญาติ + เลือกความสัมพันธ์ → เลือกประเภทบริการนวัตกรรม → บันทึก
- ดูรายละเอียดเพิ่มเติม: [[concepts/Smart-Card-Proxy-ยืนยันตัวตนแทน]]

### Service Code สำหรับคลินิกพยาบาล
- **PG 019002** = CI / รักษาโรคเบื้องต้น (C150)
- **PG 019003** = พยาบาลพื้นฐาน/ทำแผล/ล้างตา/ล้างจมูก
- **PG 006001** = PP (ทุกวิชาชีพทำได้)
- ต้องเลือกให้ตรงกับบริการที่ให้จริง ทั้งตอนยืนยันตัวตนและตอนส่งเบิก

## อัปเดต — ครั้งที่ 85 (23 มกราคม 2569)

### ปัญหาระบบ FVS ล่มวันที่ 22 ม.ค. 2569
- ระบบสแกนใบหน้ามีปัญหา หน่วยบริการแจ้งเข้ามาจำนวนมาก
- สปสช.แนะนำ: ใช้ **Line OA แทน** ในช่วงที่ FVS ล่ม
- บริการที่ยืนยันตัวตนแล้วแต่ FVS ไม่สำเร็จ → สปสช.จะ **waive และ reprocess** ให้อัตโนมัติในรอบถัดไป
- อยู่ระหว่างทำหนังสือเวียนแจ้งหน่วยบริการนวัตกรรมอย่างเป็นทางการ

### การเลือกประเภทบริการใน FVS
- **สำคัญ:** ต้องเลือกประเภทให้ตรงกับวิชาชีพที่ให้บริการจริง
- ตัวอย่างผิด: คลินิกกายภาพเลือก OPD/IPD/PP → ถูกปฏิเสธ
- กรณีเลือกผิด: สปสช.จะนำข้อมูลมา **reprocess ใหม่**

### ระบบ FVS ในบริบทหน่วยนวัตกรรมเฟส 2
- ก่อนใช้ FVS แอป หน่วยต้อง **Reactivate ใน OSS** ให้สำเร็จก่อน
- Login ผ่าน OSS Username/Password → ระบบรู้พิกัดและรหัสร้าน
- QR ที่ร้านสร้างให้ประชาชนสแกนมีอายุ **10 นาที** (ต่างจาก QR Line OA 6 ชั่วโมง)
