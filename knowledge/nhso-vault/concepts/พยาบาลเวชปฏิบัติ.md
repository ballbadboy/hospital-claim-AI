# พยาบาลเวชปฏิบัติ (Nurse Practitioner)

## Definition

**พยาบาลเวชปฏิบัติ** คือพยาบาลวิชาชีพที่ผ่านการอบรมเพิ่มเติมและได้รับการรับรองจาก **สภาการพยาบาล** ให้สามารถ**ตรวจวินิจฉัยและรักษาโรคเบื้องต้น (Primary Care)** ได้โดยอิสระ โดยไม่ต้องมีใบสั่งแพทย์

ในระบบสปสช. ปี 2569 พยาบาลเวชปฏิบัติเป็น **คุณสมบัติบังคับ** สำหรับการให้บริการ **รักษาโรคเบื้องต้น (CI/C)** ที่คลินิกพยาบาล

## ความแตกต่างจากพยาบาลวิชาชีพทั่วไป

| คุณสมบัติ | พยาบาลวิชาชีพทั่วไป | พยาบาลเวชปฏิบัติ |
|-----------|---------------------|-------------------|
| ให้ยาตามแผนการรักษาแพทย์ | ✓ | ✓ |
| บริการพยาบาลพื้นฐาน (ทำแผล) | ✓ | ✓ |
| **รักษาโรคเบื้องต้น (CI)** | **✗ (ปี 2569)** | **✓** |
| บันทึก CI ใน M Care Plus | ✗ | ✓ |

## การขึ้นทะเบียนในระบบสปสช.

- ต้องลงทะเบียนในระบบ OSS ของสปสช. ประเภท **4.2.26** (พยาบาลเวชปฏิบัติ)
- ระบบยิง API ไปตรวจสอบกับ**สภาการพยาบาล** เพื่อยืนยันสถานะ
- พยาบาลวิชาชีพทั่วไปลงทะเบียนประเภท **4.1** (พยาบาลทั่วไป)
- ข้อมูลประเภทต้องถูกต้องในระบบ reactivate มิฉะนั้น M Care Plus จะล็อคสิทธิ์การบันทึก CI

## ข้อกำหนด Full-Time / Part-Time

- **Part-time:** 1 คน ไม่สามารถลงทะเบียนให้บริการในคลินิกพยาบาลมากกว่า 1 แห่งพร้อมกันได้
- **Full-time:** ทำงานประจำสถานที่เดียว

## ผลต่อการเบิกจ่าย

- ถ้า User ใน M Care Plus ไม่ได้ขึ้นทะเบียนเป็นพยาบาลเวชปฏิบัติ ระบบจะ**ไม่แสดงเมนู CI** ให้บันทึก
- การบันทึก CI โดยพยาบาลที่ไม่มีคุณสมบัติจะทำให้เกิดปัญหาในการเบิกจ่าย
