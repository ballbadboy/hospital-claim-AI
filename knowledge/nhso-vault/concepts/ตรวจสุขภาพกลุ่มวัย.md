# ตรวจสุขภาพที่จำเป็นตามกลุ่มวัย

บริการตรวจสุขภาพเชิงป้องกันที่ [[entities/สปสช]] จัดให้ผู้มีสิทธิ์บัตรทอง ตามช่วงอายุ/กลุ่มวัย รับบริการได้ที่หน่วยบริการในเครือข่าย รวมถึงหน่วยบริการนวัตกรรม (คลินิก/หน่วยเชิงรุกในกทม. และปริมณฑล)

## ตัวอย่างรายการบริการที่ขอเบิก (จากวิดีโอ)

- ตรวจคัดกรอง **HIV**
- ยาเม็ดเสริมธาตุ**เหล็ก**
- ยาเม็ด**คุมกำเนิด**
- รายการตรวจสุขภาพตามกลุ่มวัยอื่นๆ (เบาหวาน, ความดัน ฯลฯ)

## อัตราชดเชย (RBF)

| รายการ | ยอดชดเชย |
|--------|---------|
| รายการตรวจ (ตัวอย่างจาก 3 รายการในวิดีโอ) | 80.00 บาท |
| รายการตรวจ (บางประเภท) | 120.00 บาท |

การจ่ายเป็นแบบ **[[concepts/RBF-Result-Based-Financing]]** — จ่ายตามผลงาน ราคากำหนดล่วงหน้า ไม่ใช่ fee-for-service

## ระบบที่เกี่ยวข้อง

| ขั้นตอน | ระบบ |
|--------|------|
| บันทึกข้อมูล + ปิดสิทธิ์ | **กรุงไทย** |
| สร้างใบเบิกจ่าย | **กรุงไทย** |
| ตรวจสอบยอดชดเชย | **Seamless for DMIS** → Krungthai DHP → RBF |

## หน่วยบริการนวัตกรรม

หน่วยบริการที่ให้บริการตรวจสุขภาพเชิงรุก (เช่น คลินิกเอกชนในกทม.) อยู่ในระบบ **กรุงไทย Digital Health Platform** โดยเฉพาะ ต่างจากโรงพยาบาลทั่วไปที่ใช้ระบบอื่น

## วิธีตรวจสอบยอดเงินโอนเข้า

1. เข้า `seamlessforme.nhso.go.th`
2. คลิกไอคอน **Krungthai Digital Health Platform**
3. ไปที่ **ข้อมูล → RBF → SUMMARY**
4. ดูรายงาน IEP ที่ต้องการ → คลิกดูรายละเอียดรายบุคคล
5. ดูกิจกรรมที่ขอเบิกและยอดชดเชยที่ได้รับ (PS CODE ประจำแต่ละรายการ)

→ ดูขั้นตอนเต็ม: [[concepts/Seamless-DMIS]]
