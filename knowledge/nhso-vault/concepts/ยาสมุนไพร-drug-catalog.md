# ยาสมุนไพร — ดักแคตาลอกและการเบิกจ่าย TTMT

## Definition

ตั้งแต่วันที่ **16 มีนาคม 2568** สปสช. ปรับระบบการเบิกจ่ายยาสมุนไพรในบัญชียาหลักแห่งชาติให้ใช้ **รหัสมาตรฐาน TTMT (Thai Traditional Medicine)** แทนรหัส Working Code เดิม และต้องผ่านกระบวนการ **Drug Catalog (ดักแคตาลอก)** ก่อนส่งเบิก

## ระบบที่เกี่ยวข้อง

| ระบบ | บทบาท |
|------|-------|
| Drug Catalog (ดักแคตาลอก) | ทะเบียนยาของหน่วยบริการพร้อมรหัสมาตรฐาน TTMT |
| eClaim (e-Claim) | บันทึกและส่งข้อมูลการเบิกจ่าย |
| NHSO Digital Platform | ประมวลผลและออก Statement |

## กระบวนการเบิกจ่ายยาสมุนไพร (3 ขั้นตอน)

### ขั้นตอนที่ 1: ทำดักแคตาลอก TTMT

1. เข้าระบบ eClaim → เมนู **"จัดการข้อมูล"** → **"ยาของหน่วยบริการ"** (ดักแคตาลอก)
2. ใส่รายการยาสมุนไพรพร้อม:
   - รหัสมาตรฐาน **TTMT** (ไม่ใช่ Working Code)
   - ราคาขาย (ราคาที่ขอเบิกจาก สปสช.)
3. ส่งให้หน่วยงานที่เกี่ยวข้องตรวจสอบและอนุมัติ (สสมท. หรือหน่วยงานที่กำกับดูแล)

> ดักแคตาลอกจะรวมทั้งยาปกติและยาสมุนไพรของหน่วยบริการทั้งหมด

### ขั้นตอนที่ 2: เชื่อมโยงยา

1. เมื่อยาได้รับการอนุมัติแล้ว → กลับมาที่เมนูดักแคตาลอก
2. กดปุ่ม **"เชื่อมโยงข้อมูล"**
3. ระบบจะแจ้งว่า "ดำเนินการยาของหน่วยบริการดักแคตาลอกเรียบร้อยแล้ว"

### ขั้นตอนที่ 3: EF (แก้ไขและส่งใหม่)

สำหรับรายการที่ได้รับชดเชย 0 บาท ตั้งแต่ 16 มีนาคม 68:
1. เข้าไปที่ eClaim → แก้ไขรายการยาสมุนไพรให้เป็นรหัส TTMT ที่ถูกต้อง
2. ยาสมุนไพรกลับบ้านต้องอยู่ใน **หมวดที่ 4** (ยากลับบ้าน)
3. ส่งข้อมูลใหม่ → ระบบจะประมวลผลจ่ายอีกครั้ง

**สำหรับหน่วยสังกัดกระทรวงสาธารณสุข:** ส่งข้อมูลชุดนั้นใหม่ผ่านระบบ Financial Data Hub

## ปัญหาที่พบบ่อย

### ยาสมุนไพรได้รับชดเชย 0 บาท

| สาเหตุ | วิธีแก้ |
|-------|--------|
| ยังไม่ทำดักแคตาลอก TTMT | ทำดักแคตาลอกให้เสร็จก่อน |
| ดักแคตาลอกแล้วแต่ยังไม่เชื่อมโยงยา | กดปุ่มเชื่อมโยงข้อมูลใน eClaim |
| บันทึกผิดหมวด | ยาสมุนไพรกลับบ้าน → หมวด 3 หรือ 4 |
| รหัสยาไม่ใช่รหัส TTMT | แก้รหัสให้เป็น TTMT มาตรฐาน |

### C305 / C435 ร่วมกับยาสมุนไพร

- **C305/C435** คือปัญหาการปิดสิทธิ์ ไม่ใช่ปัญหาของดักแคตาลอก
- ต้องแก้การปิดสิทธิ์ผ่านสมาร์ทการ์ดให้สมบูรณ์ก่อน
- จากนั้นจึงทำดักแคตาลอก TTMT และ EF ใหม่

## ช่วงเวลาที่มีผล

- **ก่อน 16 มีนาคม 68:** เบิกผ่านระบบ OP Individual (43 แฟม) — ยังใช้ได้
- **ตั้งแต่ 16 มีนาคม 68 (Statement 6803 OP 02):** ต้องผ่านดักแคตาลอก TTMT ทุกรายการ

## การตรวจสอบผลลัพธ์

- เข้า Simulate for DMIS → ดูรายงาน REP รายบุคคล
- หากยังเป็น 0 บาท → ตรวจสอบ date effective ของดักแคตาลอกและรหัส TTMT
