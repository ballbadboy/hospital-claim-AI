# PAK — Prior Authorization for Kidney (ระบบขออนุมัติก่อนบำบัดทดแทนไต)

## Definition

PAK (Prior Authorization for Kidney) คือโปรแกรมระบบขออนุมัติก่อนการให้บริการบำบัดทดแทนไต ที่สปสช.พัฒนาขึ้นเพื่อแทนโปรแกรม SMD (เดิม) โดยผู้ป่วยรายใหม่ทุกรายต้องได้รับการอนุมัติผ่านระบบ PAK ก่อนจึงจะสามารถบันทึกข้อมูลเพื่อเบิกค่าใช้จ่ายผ่านโปรแกรม DMS CKD ได้

## How It Works

### Timeline

| วันที่ | เหตุการณ์ |
|--------|----------|
| ก่อน ปีงบ 2569 | หน่วยบริการนำร่อง 18 แห่ง ใช้ PAK |
| ปีงบ 2569 | หน่วยบริการนำร่อง เพิ่มเป็น **24 แห่ง** |
| **1 มกราคม 2569** | **PAK ขยายทั่วประเทศ** สำหรับผู้ป่วยรายใหม่ |
| **1 มกราคม 2569** | **โปรแกรม SMD ปิดตัว** |

### เงื่อนไขหน่วยบริการที่ให้บริการผ่าน PAK ได้

1. เป็นหน่วยบริการรับส่งต่อเฉพาะด้านการฟอกเลือดด้วยเครื่องไตเทียม
2. ผ่านการตรวจรับรองมาตรฐานโดยแพทยสภา
3. มีกระบวนการตัดสินใจร่วมในการเลือกวิธีบำบัดทดแทนไต
4. บันทึกกระบวนการตัดสินใจร่วมไว้ในเวชระเบียน

### กระบวนการ

1. หน่วยบริการบันทึกข้อมูลผู้ป่วยรายใหม่ใน PAK
2. ระบบประมวลผลและอนุมัติ
3. หน่วยบริการบันทึกข้อมูลการให้บริการใน DMS CKD
4. เบิกค่าใช้จ่ายตามปกติ

### กรณีหน่วยบริการที่ไม่ใช่นำร่อง (ก่อน 1 มกราคม 2569)

- ให้บันทึกผ่านโปรแกรมเดิม (SMD) ก่อน
- เมื่อถึง 1 มกราคม 2569 → เปลี่ยนมาใช้ PAK ทั้งหมด
