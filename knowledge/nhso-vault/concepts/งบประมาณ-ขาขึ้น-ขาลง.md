# กระบวนการงบประมาณ "ขาขึ้น-ขาลง" สปสช.

## Definition

คำศัพท์ภายในที่ สปสช. ใช้อธิบายกระบวนการจัดสรรงบประมาณกองทุนหลักประกันสุขภาพแห่งชาติ แบ่งเป็น 2 ขา:

- **ขาขึ้น**: กระบวนการเสนอขอรับงบประมาณ (สปสช. → อนุกรรมการ → บอร์ด → สำนักงบประมาณ → ครม. → สภา)
- **ขาลง**: กระบวนการจัดสรรงบที่ได้รับกลับสู่หน่วยบริการ (ปรับเป้าหมาย, จ่ายชดเชย, กำหนดอัตรา)

## ขาขึ้น — How It Works

1. **สปสช.** รวบรวมข้อมูลผลงาน, ความต้องการ, ข้อเสนอจากหน่วยบริการและประชาชน
2. **คณะอนุกรรมการ** (อนุกองทุน, อนุเงินการคลัง) กลั่นกรองพร้อมความเห็นบอร์ดควบคุม
3. **คณะกรรมการหลักประกันสุขภาพแห่งชาติ (บอร์ด)** อนุมัติ
4. ส่ง **สำนักงบประมาณ** → **ครม.** → **รัฐสภา** บรรจุใน พรบ. งบประมาณ

**ปี 2569**: กระบวนการขาขึ้นเริ่มตั้งแต่ มีนาคม–กรกฎาคม (มีการประชุมรับฟังความเห็น 22 ครั้ง รวมโรงเรียนแพทย์, รพศ., รพท., รพช.)

## ขาลง — How It Works

หลังได้รับงบประมาณ (มักได้น้อยกว่าที่ขอ):
1. สปสช. ปรับเป้าหมายตามงบที่ได้รับ
2. หารือกับผู้มีส่วนได้ส่วนเสียทุกฝ่าย
3. ผ่านอนุกองทุน → บอร์ด ตัดสินใจ
4. ออกประกาศหลักเกณฑ์เงื่อนไข (ราชกิจจา) — **ปี 69 จะออก 1 ครั้ง ตั้งแต่ต้นปีงบประมาณ ไม่ปรับกลางปี**

## สถานการณ์ปี 2569

- **งบ New High**: ~4,100 บาท/หัวประชากร เพิ่มขึ้น 12% — สูงที่สุดตั้งแต่ก่อตั้ง (ปี 2545)
- **งบสาธารณสุขบัตรทองเติบโต 12%** ขณะที่ GDP ประเทศเติบโตเพียง 2-3% — เป็น Challenge ระยะยาว
- ในอดีตบางปีใช้เงินกองทุนเกินงบ (ใช้เงินสะสม + งบกลาง)

## ปัญหาที่แก้ไขปี 2569

| ปัญหาเดิม | แนวทางแก้ไข |
|-----------|-------------|
| ประกาศหลักเกณฑ์ล่าช้า (6–9 เดือนหลังต้นปีงบฯ) | ออกประกาศ 1 ครั้งต้นปี ไม่ปรับกลางปี |
| ศัพท์ขาขึ้น-ขาลงไม่คุ้นเคย | อธิบายชัดเจนในการประชุมชี้แจงรายปี |
