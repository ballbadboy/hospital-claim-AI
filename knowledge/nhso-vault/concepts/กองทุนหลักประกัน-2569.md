# โครงสร้างงบกองทุนหลักประกันสุขภาพแห่งชาติ ปีงบประมาณ 2569

## Definition

งบประมาณรวมของกองทุนหลักประกันสุขภาพแห่งชาติประจำปีงบประมาณ 2569 (ตุลาคม 2568 – กันยายน 2569) มีมูลค่ารวมทั้งสิ้นประมาณ **265,295.5821 ล้านบาท** ดูแลประชากรผู้มีสิทธิ์บัตรทอง ~47 ล้านคน

## โครงสร้างงบหลัก

### หมวด 1 — เหมาจ่ายรายหัว (~190,000+ ล้านบาท)

งบก้อนใหญ่ที่สุด เพิ่มจากปี 2568 ราว 16,000 กว่าล้านบาท แบ่งเป็นหมวดย่อย:

| หมวดย่อย | รายละเอียด |
|----------|------------|
| ผู้ป่วยนอก (OP) | ค่าหัว OP รวมในเหมาจ่าย อัตรา 1,440.48 บาท/หัว (ดูแยกใน [[concepts/การเปลี่ยนแปลงปีงบ69]]) |
| ผู้ป่วยใน (IP) | จ่าย RW-based, Global Budget รายเขต ([[concepts/Global-Budget-IPD]]) |
| กรณีบริการเฉพาะ (CR) | จ่าย on-top สำหรับบริการพิเศษเพิ่มเติม |
| ฟื้นฟูสมรรถภาพ | งบฟื้นฟูผู้ป่วย |
| แพทย์แผนไทย | |
| งบลงทุน/ค่าเสื่อม | เบิกในลักษณะงบลงทุน |
| PP (ส่งเสริมสุขภาพ) | 410.52 บาท/หัว (แยกจากเหมาจ่าย 2–3 ปีแล้ว) ([[concepts/PP-กองทุนส่งเสริมสุขภาพ-ปี69]]) |

### งบนอกเหมาจ่าย — บริการกลุ่มเฉพาะ

| รายการ | วงเงินปี 2569 | หมายเหตุ |
|--------|--------------|----------|
| HIV/AIDS | ~4,000+ ล้านบาท | ครอบคลุม coverage เพิ่มขึ้น |
| ผู้ป่วยไตวายเรื้อรัง (CKD) | ~16,000+ ล้านบาท | เพิ่มจาก 13,000 ล้าน (ปี 68) ([[concepts/ไตวายเรื้อรัง-69]]) |
| ป้องกันโรคเรื้อรัง NCD | ~1,700+ ล้านบาท | DM, HT, จิตเวช, Asthma, COPD |
| ปฐมภูมิ (PCC/หมอครอบครัว) | ~3,700+ ล้านบาท | จ่าย Global Budget ไม่ตัดจากค่าหัว รพ. |
| Long-term Care | ~5,000+ ล้านบาท | [[concepts/Long-Term-Care-ปี69]] |
| กองทุนท้องถิ่น (อปท.) | 45 บาท/หัวประชากร | [[concepts/กองทุนท้องถิ่น-ปี69]] |

## กลไกการจัดสรร

- **ขาขึ้น**: สปสช.เสนองบต่อสำนักงบประมาณ — ปัญหาคือขาขึ้น–ขาลงไม่สัมพันธ์กัน (TDRI พบ)
- **ขาลง**: จัดสรรตามประกาศบริหารกองทุน → ประกาศการจ่าย → หน่วยบริการเบิกจ่าย
- **กลไก 7x7**: [[concepts/7x7-กลไกตรวจสอบ]] รับประเด็นระหว่างปี ปรับแนวทางในปีงบประมาณ
- **สปสช.เขต 13 เขต**: รับผิดชอบชี้แจงและกำกับดูแลพื้นที่

## ปัญหาเชิงนโยบายปี 2569

1. **IP Underfunding** — อัตราจ่าย ~8,000–8,350 บาท/RW ขณะที่ต้นทุนจริง ~12,000–13,000 บาท/RW โรงพยาบาล สธ. ติดลบจากกองทุนนี้ บุคลากรสมองไหล
2. **งบกลาง IP** — กรณีงบ IP รายเขตไม่พอ ต้องใช้งบกลาง (รออนุมัติ ครม.)
3. **NCD escalation** — ผู้ป่วยโรคเรื้อรังเพิ่มขึ้นทุกปี กดดันงบ IP และ OP
4. **ไตวายเรื้อรัง** — งบเพิ่ม 3,000 ล้านในปีเดียว จากนโยบาย "ฟอกไตฟรีทุกแห่ง"

## เป้าหมายปีงบ 2570

- กระทรวง สธ. เสนอ IP rate 10,000 บาท/RW (เป้าระยะกลาง) และ 13,000 บาท/RW (เป้าระยะยาว)
- งบขาขึ้นปี 70 จะใช้ข้อมูลจากพื้นที่ (เขตส่งเป้าหมายขึ้นมา) แทนการคำนวณจากส่วนกลาง
