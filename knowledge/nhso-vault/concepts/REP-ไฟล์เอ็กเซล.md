# REP Excel — รายงานผลการตรวจสอบการเบิกจ่าย

## Definition

**REP (Reimbursement Evaluation Report)** หรือ **IEP** คือไฟล์ Excel ที่สปสช.ออกให้หน่วยบริการ เพื่อแจ้งผลการตรวจสอบข้อมูลการส่งเบิก ว่า**ผ่านหรือไม่ผ่าน** พร้อมรหัสเหตุผล (Error Code / C-Code / Deny Code) และยอดชดเชยสุทธิ

- ออกทุก **วันจันทร์ พุธ ศุกร์**
- ดาวน์โหลดผ่านระบบ **e-Claim** (eclient) → เมนูบริการ → รับข้อมูล → ผลการตรวจสอบ REP

## Flow การเบิกจ่ายและตำแหน่ง REP

```
หน่วยบริการ
  → ส่งข้อมูลผ่าน Financial Data Hub / e-Claim
  → ประมวลผลรายวัน (CD9 + เงื่อนไขต่าง ๆ)
  → REP ออก (จันทร์/พุธ/ศุกร์)
  → Statement ออก (OP ทุก 15 วัน | IP ทุกสิ้นเดือน)
  → โอนเงิน
```

## โครงสร้าง REP Excel (6 ชีท)

| ชีท | ชื่อ | เนื้อหา |
|-----|------|---------|
| 1 | **Detail** | รายละเอียดทุกรายการ — ยอดชดเชยสุทธิ, Error Code, DRG/RW (IP) |
| 2 | **Summary** | ยอดสรุปรวม OP/IP, จำนวนเคส, ยอดชดเชยทั้งหมด |
| 3 | **Data Instrument** | รายการเบิกอุปกรณ์/อวัยวะเทียม (แสดงเฉพาะเมื่อมีการเบิก) |
| 4 | **Data Drug** | รายการเบิกยา (แสดงเฉพาะเมื่อมีการเบิก) |
| 5 | **Deny** | รายการที่ถูกปฏิเสธ — รหัส C / Deny Code และคำอธิบาย |
| 6 | **Data 0 บาท** | รายการที่จ่ายชดเชย 0 บาท |

> ถ้าไม่มีการเบิกอุปกรณ์/ยา/รายการติด ระบบจะแสดงเพียง 2 ชีท (Detail + Summary)

## คอลัมน์สำคัญใน Sheet Detail (OP)

| คอลัมน์ | ความหมาย |
|---------|---------|
| ชดเชยสุทธิ | ยอดที่สปสช.จ่ายจริงสำหรับรายการนั้น |
| Error Code | รหัส C หรือ Deny ที่แจ้งเหตุผลไม่ผ่าน |
| BC–CX | ยอดชดเชยแยกตามกองทุน (High Cost, AE, อวัยวะเทียม, IP, DMIS ฯลฯ) |
| DG | กองทุน Free Schedule |
| **ชดเชยสุทธิรวม** | = ผลรวม BC–CX + DG |

## คอลัมน์สำคัญใน Sheet Detail (IP)

- มีคอลัมน์ **DRG** และ **AdjRW** เพิ่มเติม (สปสช.คำนวณให้)
- ยอดชดเชย IP **จะไม่แสดงจนกว่า Statement จะออก** (สิ้นเดือน) เพราะต้องรอคำนวณอัตราจ่าย Global Budget รายเขต
- หลัง Statement ออก → โหลด REP ใหม่ → ยอดชดเชย IP จะปรากฏพร้อมอัตราจ่าย
- Deny Column: ตัว **C** = Clean (ผ่าน), ตัวอื่น = รหัสปฏิเสธ (เช่น iC)
- Remark: กรณี ODS/MIS หรือ Home Ward จะมีหมายเหตุพิเศษในคอลัมน์ Remark

## กองทุนใน Sheet Detail (แยกตามคอลัมน์)

| ลำดับกองทุน | กองทุน |
|------------|--------|
| 1 | High Cost |
| 2 | AE |
| 3 | อวัยวะเทียมและอุปกรณ์บำบัดรักษา |
| 4 | ผู้ป่วยใน (IP) |
| 5 | DMIS — โรคเฉพาะ (Cataract, Stroke, STEMI, Palliative Care ฯลฯ) |
| 6 | ยา (Drug) |
| 7 | Free Schedule |

## ขั้นตอนดาวน์โหลด REP

1. เข้า **e-Claim** (eclient) → Login
2. เมนู **บริการ** → **รับข้อมูล** → **ผลการตรวจสอบ REP**
3. รายการสีขาว = รอประมวลผล, สีเขียว = ประมวลผลเสร็จ
4. กดรับข้อมูล (ปุ่มที่ 1) → ระบบแจ้งจำนวนรายการ (ที่ 2)
5. ดาวน์โหลด **ไฟล์ Excel REP** (ที่ 3) → ตรวจสอบ

## ความสัมพันธ์กับ Statement และการโอนเงิน

- **REP** = รายงานผลตรวจสอบรายครั้ง (ออก จันทร์/พุธ/ศุกร์)
- **Statement** = รายงานรวมยอดตามรอบ (OP 15 วัน / IP สิ้นเดือน) → เป็นฐานในการโอนเงิน
- ยอด REP สะสมจนครบรอบ → ออก Statement → โอนเงินตามปฏิทิน
