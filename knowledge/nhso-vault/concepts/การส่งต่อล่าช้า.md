# การชะลอโอนเงิน (Delay Transfer)

## Definition

การชะลอโอนเงิน คือกระบวนการที่ สปสช. **ระงับการโอนเงินชดเชย** ให้หน่วยบริการชั่วคราว เพื่อตรวจสอบข้อมูลที่ผิดปกติก่อนการโอน สถานะจะปรากฏในหน้า [[concepts/Smart-Money-Transfer-โอนเงินอัจฉริยะ]] ว่า **"ชะลอโอน"**

## เงื่อนไขที่ทำให้ถูกชะลอโอน

1. **ค่าใช้จ่ายผิดปกติ** — สูงกว่าค่าเฉลี่ยหรือเบี่ยงเบนมาก
2. **ข้อมูลที่น่าสงสัย** — ตามฐานข้อมูลที่ สปสช. มี
3. **อยู่ในรายชื่อ Post-Audit** — หน่วยที่เข้าเกณฑ์ต้องตรวจละเอียด

## กระบวนการชะลอโอน (Timeline รายสัปดาห์)

```
จันทร์   → ออก IEP (รายงานการจ่าย)
อังคาร   → วางฎีกา + ระบบ Flag ผิดปกติ
พุธ      → เขตพิจารณาชะลอโอน
พุธ 21:00 → โอนเงินปกติ OR ชะลอโอน
พฤหัส-ศุกร์ → เขตลงตรวจหน่วยที่ชะลอ
             → ถ้าผ่าน: ปลดชะลอ (โอนสัปดาห์นั้น)
             → ถ้าไม่ทัน: ข้ามสัปดาห์
```

## กรณี Post-Audit

หน่วยที่เข้าโปรแกรมตรวจสอบหลังการจ่าย ([[concepts/Post-Payment-Audit-ตรวจสอบหลังจ่าย]]):
- คณะทำงาน Auditor ลงตรวจ
- หน่วยบริการมีสิทธิ์ **อุทธรณ์ 2 ครั้ง**
- ใช้เวลานานกว่าปกติ
- ผลการตรวจ:
  - **ผ่าน** → ปลดชะลอ, ให้บริการต่อปกติ
  - **ไม่ผ่าน** → ระงับการให้บริการ (ร้ายแรงที่สุด)

## วิธีตรวจสอบสถานะ

1. เข้า nhho.go.th → Smart Money Transfer
2. ใส่รหัสหน่วยบริการ
3. ดูคอลัมน์สถานะ — ถ้าเห็น "ชะลอโอน" ให้ติดต่อสปสช. เขต

## การติดต่อเร่งรัด

- โทร **1330 กด 5 กด 1** แจ้งรหัสหน่วยและเลขงวดที่ชะลอ
- สปสช. เขตจะเป็นผู้พิจารณาปลดหรือดำเนินการตรวจต่อ
