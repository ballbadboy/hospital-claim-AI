# เขตสุขภาพ (Health Region)

## Definition

เขตสุขภาพ คือหน่วยบริหารจัดการระบบสาธารณสุขระดับภูมิภาค ประเทศไทยแบ่งเป็น 13 เขตสุขภาพ (เขต 1-12 ครอบคลุมทุกจังหวัด + เขต 13 กรุงเทพมหานคร) สปสช.ใช้เขตสุขภาพเป็นหน่วยในการจัดสรรและบริหาร Global Budget ผู้ป่วยใน

## บทบาทในระบบการจ่ายเงิน IP

- **วงเงิน Global Budget รายเขต:** แต่ละเขตมีวงเงิน IP ของตนเอง กำหนดจากผลงานประมาณการย้อนหลัง
- **อัตราจ่ายรายเขต:** คำนวณจาก วงเงินเขต ÷ ผลรวม AdjRW ในเขต → แต่ละเขตได้อัตราต่างกัน
- **อปสข. (อนุกรรมการสปสช. ระดับเขต):** ตัดสินใจเรื่อง Extrapolation และนโยบายภายในเขต
