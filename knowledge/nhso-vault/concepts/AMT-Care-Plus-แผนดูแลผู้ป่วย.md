# AMT Care Plus (M Care Plus)

## Definition

**AMT Care Plus** หรือ **M Care Plus** คือโปรแกรมบันทึกข้อมูลการเบิกจ่ายรุ่นใหม่สำหรับ **คลินิกพยาบาลและการผดุงครรภ์** (หน่วยบริการรับส่งต่อเฉพาะด้านการพยาบาลและการผดุงครรภ์) พัฒนาโดย **สวทช. (NECTEC)** ร่วมกับสปสช. และสภาการพยาบาล เริ่มใช้อย่างเป็นทางการตั้งแต่วันที่ **3 มีนาคม 2569**

## ที่มาและเหตุผล

ระบบเดิมมีโปรแกรมแยกกัน 2 ตัว:
- **AMET / Care เดิม** — สำหรับบริการรักษาโรคเบื้องต้น (CI)
- **EKAM** — สำหรับบริการพยาบาลผู้ป่วยนอก (OPD: ทำแผล, ให้ยา ฯลฯ)

M Care Plus **ยุบรวม** ทั้งสองเข้าด้วยกัน ทำให้คลินิกใช้โปรแกรมเดียวสำหรับทั้ง CI และ OPD Nursing

## ฟีเจอร์หลัก

### 1. เวชระเบียนอิเล็กทรอนิกส์
- สร้างเวชระเบียนอัตโนมัติตามมาตรฐานสภาการพยาบาล ไม่ต้องเขียนกระดาษ
- Export เป็น PDF หรือพิมพ์ออกมาได้
- ระบบ Auditor ของสปสช. และสภาวิชาชีพเข้าถึงได้ภายในระบบ

### 2. Single Sign-On (SSO)
- เชื่อมต่อกับระบบ e-Authentication ของสปสช. (SSO)
- ยืนยันตัวตน 2 ชั้น (Two-Factor Authentication) ด้วย OTP
- ล็อกอินครั้งต่อไปใช้ปุ่ม SSO เพียงคลิกเดียว

### 3. การส่งเคลม 13 แฟ้ม V2
- ส่งข้อมูลในรูปแบบมาตรฐาน 13 แฟ้ม เวอร์ชัน 2
- ตรวจสอบสถานะเคลม real-time บน dashboard (รอพิจารณา / โอนแล้ว ฯลฯ)
- **Batch Claim:** ส่งเคลมทั้งหมดพร้อมกันในคลิกเดียว

### 4. การปิดสิทธิ์ (FVS Integration)
- ดึง token จากระบบ FVS (สแกนใบหน้า) หรือ Line OA หรือ New Allen ผ่าน API
- มีปุ่มแยกสำหรับแต่ละช่องทาง

### 5. ระบบโควต้า
- ตรวจสอบโควต้าคงเหลือรายวัน แยก C และ OPD ก่อนลงทะเบียนผู้ป่วย
- โควต้าเป็นการเสนอแนะ ไม่ lock ถ้ามีการจองผ่าน Line มาก่อน

### 6. พิมพ์ฉลากยา
- พิมพ์ฉลากยาอัตโนมัติตามมาตรฐาน อย. ผ่านเครื่องพิมพ์ฉลาก
- รองรับยาชุด (บันทึกชุดยาที่ใช้บ่อยเพื่อเรียกใช้ซ้ำ)

## บทบาทผู้ใช้งาน

| บทบาท | สิทธิ์บันทึก |
|--------|--------------|
| พยาบาลวิชาชีพทั่วไป | OPD พยาบาลพื้นฐานเท่านั้น |
| พยาบาลเวชปฏิบัติ | CI (รักษาโรคเบื้องต้น) และ OPD |
| แอดมินคลินิก | ตั้งค่าเวลาเปิด-ปิด, พิกัด GPS คลินิก |

## ขั้นตอนการใช้งาน

1. **ลงทะเบียนผู้ป่วย:** เสียบบัตรประชาชน / ไทยดี / เป๋าตัง
2. **ตรวจสอบสิทธิ์และโควต้า:** ระบบเช็คอัตโนมัติ
3. **เลือกประเภทบริการ:** CI หรือ OPD
4. **บันทึกบริการ:** Vital Signs, วินิจฉัย, หัตถการ, จ่ายยา
5. **ปิดสิทธิ์:** ดึง token FVS/Line/New Allen
6. **อัปเดตสถานะ:** เสร็จสิ้น / ยกเลิก / ส่งต่อ
7. **ส่งเคลม:** ส่งทีละเคสหรือส่งทั้งหมดพร้อมกัน
