# Seamless for DMIS

**ชื่อเต็ม:** ระบบบูรณาการข้อมูลสิทธิการรักษาพยาบาล (Seamless For DMIS)

**URL:** `seamlessforme.nhso.go.th`

**Version:** 5.0.6.13 Build 2024-1.25

ระบบกลางของ [[entities/สปสช]] สำหรับให้หน่วยบริการตรวจสอบสถานะและยอดเงินการจ่ายชดเชยค่าบริการสุขภาพ ครอบคลุมทั้งโรคเรื้อรัง (CI), การตรวจสุขภาพกลุ่มวัย (RBF), และโปรแกรมอื่นๆ

## ทางเข้าระบบ

### ทางที่ 1 — ผ่านไอคอนบัตรประชาชนใบเดียว
- ใช้สำหรับ: ร้านยา, หน่วยบริการทั่วไป
- เข้าผ่านไอคอน **"บัตรประชาชนใบเดียวรักษาทุกที่"** ใน portal หน้าหลัก
- เมนูภายใน: **REP → INDIVIDUAL** (สำหรับ CI Claim)

### ทางที่ 2 — ผ่าน Krungthai Digital Health Platform
- ใช้สำหรับ: หน่วยบริการนวัตกรรม (คลินิก, หน่วยเชิงรุก)
- เข้าผ่านไอคอน **"Krungthai Digital Health Platform"** (วงกลมสีน้ำเงินเข้ม) ใน portal
- เมนูภายใน: **ข้อมูล → RBF → SUMMARY/INDIVIDUAL** (สำหรับตรวจสุขภาพกลุ่มวัย)

## เมนูและรายงาน

| Tab | หน้าที่ |
|-----|--------|

| ผู้ดูแล | Admin |

### ประเภทรายงาน

**REP (ผ่าน path บัตรใบเดียว) — สำหรับ CI Claim:**
- **REP SUMMARY** — ยอดรวมรายงวด
- **REP INDIVIDUAL** — รายละเอียดรายบุคคล

**RBF (ผ่าน Krungthai DHP) — สำหรับตรวจสุขภาพกลุ่มวัย:**
- **RBF : ราคา RBF และ SUMMARY** — ยอดรวม
- **RBF : ราคา RBF และ INDIVIDUAL** — รายบุคคล

### รายงานสุดท้าย (drill-down)
ชื่อ: **รายงาน รายการการจ่ายเงินชดเชยตามรายการตรวจสุขภาพ**

| คอลัมน์ | คำอธิบาย |
|---------|---------|
| รายการตรวจสุขภาพที่จำเป็น | ชื่อรายการ |
| จำนวน | จำนวนครั้ง/ราย |
| ราคากลาง | ราคามาตรฐาน |
| ราคาเบิก | ราคาที่ขอเบิก (อาจแสดง 0.00) |
| รวมเงิน | ยอดชดเชยจริงที่จ่าย |
| PS CODE | รหัสรายการบริการ |

ตัวอย่างยอดชดเชย: **80.00 บาท** และ **120.00 บาท** ต่อรายการ

## วิธีตรวจสอบ CI Claim (ร้านยา — Path 1)

1. Login ด้วย user ของร้านยา
2. คลิกไอคอน **"บัตรประชาชนใบเดียวรักษาทุกที่"**
3. ไปที่เมนู **REP → Individual** (รายบุคคล)
4. กำหนดช่วงวันที่ → กดค้นหา (ปุ่มสีส้ม)
5. ดูว่ามีข้อมูลบริการตามวันที่หรือไม่
6. Export รายงานเพื่อตรวจสอบความครบถ้วน

→ รายละเอียดเพิ่มเติม: [[sources/ร้านยาส่งเคลม-CI-ไม่ได้รับการประมวลผล-ต้องทำอย่างไร-GsGeW-E1g4Y]]

## วิธีตรวจสอบตรวจสุขภาพกลุ่มวัย (หน่วยบริการนวัตกรรม — Path 2)

1. Login ด้วย user ของหน่วยบริการ
2. คลิกไอคอน **"Krungthai Digital Health Platform"** (วงกลมสีน้ำเงิน)
3. เข้า DMIS → เมนู **ข้อมูล** → **RBF**
4. เลือก **RBF SUMMARY** ก่อน → ใส่รหัสหน่วยบริการ → ค้นหา
5. ถ้าไม่พบ → ลอง Path 1 (บัตรใบเดียว) แทน
6. คลิก IEP ที่ต้องการ → ดูรายละเอียดรายบุคคลและกิจกรรมที่ขอเบิก

→ รายละเอียดเพิ่มเติม: [[sources/วิธีการตรวจสอบยอดเงินโอนเข้ารายการตรวจสุขภาพที่จำเป็นตามกลุ่มวัย-ZCKCYr0bO8U]]

## การตรวจสอบสำหรับหน่วยนวัตกรรม (ข้อมูลเพิ่มเติม ครั้งที่ 84)

หน่วยนวัตกรรมที่เข้าระบบใหม่กลางปี 68 อาจยังไม่คุ้นเคยกับการตรวจสอบ ให้ทำตามลำดับนี้:

1. **ตรวจสอบสถานะหน่วยใน Hospital Search** — ดูว่า "วันที่เข้า" ตรงกับวันที่ขึ้นทะเบียนหรือไม่ บริการที่ส่งก่อนวันที่เข้าจะไม่ถูกประมวลผล
2. **ตรวจสอบ OneSOP Service** — ดูสถานะความพร้อมรับเงิน และเลขบัญชีธนาคารที่ลงทะเบียนไว้
3. **เข้า Seamless for DMIS** → ไอคอน "บัตรประชาชนใบเดียวรักษาทุกที่" → REP Summary → ดูรายงานวันจันทร์

### เนื้อหารายงาน REP Summary

| คอลัมน์ | ความหมาย |
|--------|---------|
| วันที่ออกรายงาน | วันจันทร์หลังสิ้นสัปดาห์บริการ |
| REP Number | รหัสชุดรายงานการจ่าย |
| จำนวนรายการ | รายการที่ส่งเบิก |
| ยอดเรียกเก็บ | ยอดที่ขอชดเชย |
| ยอดชดเชย | ยอดที่จ่ายจริง (หักรายการไม่ชดเชย) |
| **เลขงวด (IEP Number)** | ใช้โยงกับ Smart Money Transfer |

### การเลื่อนวันโอนเงิน (ม.ค. 2569)

ตั้งแต่กลางเดือนมกราคม 2569 การโอนเงินปรับจาก**วันพุธ**เป็น**วันพฤหัสบดี** ≤ 21:00 น. เนื่องจากมีการปรับระบบประมวลผล (ชั่วคราว)

## วิธีตรวจสอบยอดเงิน Disability (ฟื้นฟูคนพิการ)

สำหรับหน่วยบริการที่ต้องการตรวจสอบว่าค่าบริการฟื้นฟูคนพิการเข้ามาแล้วหรือไม่:
1. Login ด้วย user คลินิก/หน่วยบริการ
2. เข้าระบบ Seamless for DMIS (seamlessforme.nhso.go.th)
3. ดู **เลขงวด** ในแบบสรุป — งวดที่ลงท้ายด้วย **"012"** จะเป็นข้อมูลบริการช่วง 29 ก.ย.–4 ต.ค. 2568
4. คลิกเลขงวด → ดูวันที่บริการของผู้รับบริการในงวดนั้น
