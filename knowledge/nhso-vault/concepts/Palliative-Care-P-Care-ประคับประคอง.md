# Palliative Care (P Care) — บริการดูแลผู้ป่วยระยะสุดท้าย

## Definition

P Care หรือ Palliative Care คือ**บริการดูแลแบบประคับประคองสำหรับผู้ป่วยระยะสุดท้าย** ที่ได้รับการวินิจฉัยและส่งต่อให้ดูแลที่บ้าน สปสช. จ่ายค่าชดเชยให้หน่วยบริการแบบเหมาจ่ายรายเดือน

## เงื่อนไขการเบิกจ่าย (ปัจจุบัน ปีงบ 68)

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| การวินิจฉัย | ผู้ป่วยต้องมีกลุ่มโรคตามประกาศสปสช. (มีรหัส ICD-10 กำกับ) |
| แผนการดูแล | ต้องมีแผนดูแลแบบประคับประคองระบุเป็นรหัสโรค |
| ผู้ส่งต่อ | โรงพยาบาลแม่ข่ายวินิจฉัยและส่งคนไข้มา |
| การลงทะเบียน | บันทึกผ่านระบบ eClaim |

## อัตราการจ่าย

| สถานการณ์ | ยอดจ่าย |
|----------|---------|
| ดูแลที่บ้านรายเดือน (เดือนที่ 1–6) | 1,000 บาท/เดือน |
| เมื่อผู้ป่วยเสียชีวิต | 3,000 บาท (ครั้งเดียว) |
| ถ้าเสียชีวิตในเดือนแรก | 1,000 + 3,000 = 4,000 บาท |
| หลังเดือนที่ 6 (ยังมีชีวิตอยู่) | หยุดจ่ายรายเดือน รอจนเสียชีวิต |

## ขั้นตอนการบันทึก

1. โรงพยาบาลแม่ข่ายวินิจฉัยและส่งคนไข้ไปดูแลที่บ้าน
2. หน่วยบริการ (รวมถึง รพ.สต.) บันทึกการเยี่ยมผ่าน **eClaim**
3. สปสช. ตรวจสอบ: กลุ่มโรคตามประกาศ + แผนดูแล + วันที่เริ่ม
4. IEP ออกรวมกับโรงพยาบาลแม่ข่าย → ตรวจสอบใน Statement UC (ประเภท H)

## รพ.สต. กับ P Care

รพ.สต. สามารถบันทึกเบิก P Care ได้ หากอยู่ในเครือข่ายบริการและโรงพยาบาลแม่ข่ายส่งคนไข้มา โดย IEP จะออกรวมกับแม่ข่าย ให้ตรวจสอบจาก **Statement UC ประเภท H**

## หมายเหตุ

เงื่อนไข P Care กำลังอยู่ระหว่างการปรับปรุง (ณ พ.ย. 2567) จะมีประกาศใหม่เร็วๆ นี้เพื่อให้ครอบคลุมการดูแลแบบองค์รวมมากขึ้น
