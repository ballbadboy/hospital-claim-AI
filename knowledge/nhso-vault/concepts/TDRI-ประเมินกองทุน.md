# TDRI — การประเมินการบริหารจัดการกองทุนหลักประกันสุขภาพแห่งชาติ

## Definition

สถาบันวิจัยเพื่อการพัฒนาประเทศไทย (TDRI) ได้รับมอบหมายให้ศึกษาและประเมินการบริหารจัดการกองทุนหลักประกันสุขภาพแห่งชาติ นำเสนอในการประชุมชี้แจงระดับประเทศปีงบ 2569 (ช่วงเช้า) โดยใช้กรอบการวิเคราะห์ 2 หลักการ ได้แก่ **Principal-Agent Theory** และ **Triple E Framework**

## กรอบการวิเคราะห์

### 1. Principal-Agent Theory (ทฤษฎีตัวการ–ตัวแทน)

| บทบาท | ผู้รับบทบาท |
|-------|------------|
| **ตัวการ (Principal)** | ผู้มีสิทธิ์ (ประชาชน), หน่วยบริการ, องค์กรวิชาชีพ, ผู้มีส่วนได้เสีย |
| **ตัวแทน (Agent)** | สปสช. + บอร์ดหลัก + บอร์ดควบคุม |

คำถามหลัก: ตัวแทน (สปสช.) ทำงานแทนตัวการได้ดีเพียงใด? มีผลประโยชน์ทับซ้อนหรือไม่?

**ปัญหาที่พบ — บอร์ดปีงบ 4–6**:
- กรรมการ >3 คณะ ก่อให้เกิด Role Conflict (บางกรรมการนั่งทั้งบอร์ดหลักและบอร์ดควบคุม)
- บางส่วนอ่านมากกว่า 11 หน่วยใน 2 ปี / 18 หน่วยใน 3 ปี
- ตาม พ.ร.บ. ปีที่ 5 บอร์ดกรรมการควรลดเหลือ 3 คณะ

### 2. Triple E Framework

| E | ความหมาย | สิ่งที่ดู |
|---|----------|----------|
| **Efficiency** | ประสิทธิภาพ | งบขาขึ้น–ขาลง, การเบิกจ่าย, ความซับซ้อน |
| **Evidence-based** | ใช้ข้อมูล | รายงานการประชุม, เว็บไซต์, ข้อมูลในระดับเขต |
| **Execution** | การนำไปปฏิบัติ | นโยบายออกแล้วดำเนินการได้ดีแค่ไหน |

ใช้ **Scoring Rubric** ให้คะแนน (ต่ำสุด–สูงสุด) ในแต่ละมิติ ลงไปถึงระดับเขตสุขภาพ

## ผลการประเมิน (สรุป)

| ด้าน | ผลการประเมิน |
|------|-------------|
| งบขาขึ้น | พอโอเค — แต่ disclosure ข้อมูลยังไม่เพียงพอ, บางขั้นตอนควรลดความซับซ้อน |
| งบขาลง–ขาขึ้น ความสัมพันธ์ | **ยังเป็นปัญหา** — ขาลงไม่สัมพันธ์กับขาขึ้น (ถูกวิจารณ์มาตลอด) |
| การเบิกจ่าย–ตรวจสอบ | ทำได้ค่อนข้างดี |
| การจัดซื้อจัดจ้าง | ทำได้ค่อนข้างดี |
| การควบคุมงบประมาณ | **ยังเป็นปัญหา** — หลายกรณีต้องใช้งบกลาง |
| คุณภาพบริการ (ผ่าน 1330) | ทำได้ค่อนข้างดี — แต่กลุ่มเปราะบาง (ผู้สูงอายุ/ชนบท) ยังไม่รู้ช่องทาง |
| การจัดการร้องเรียน | ทำได้ดี |

## ข้อเสนอแนะหลัก

1. ลดความซับซ้อนขั้นตอนงบขาขึ้น
2. เพิ่ม disclosure ข้อมูลงบประมาณให้ชัดเจน
3. แก้ปัญหาความไม่สัมพันธ์ขาขึ้น–ขาลง
4. ลดจำนวนคณะกรรมการ/บทบาทซ้อนทับ
5. เพิ่มความตระหนักช่องทางสำหรับกลุ่มเปราะบาง

## บริบท

- งานศึกษาชิ้นนี้เป็น External Assessment — TDRI เป็นองค์กรวิจัยอิสระ
- สปสช.ให้ข้อมูลระดับเขต (Micro data) แก่ TDRI เพื่อวิเคราะห์
- ค่าใช้จ่ายสุขภาพต่อ GDP ของไทยยังต่ำกว่า OECD มาก; ค่าใช้จ่ายต่อหัวต่ำกว่าสิงคโปร์มาก
- รัฐบาลควรเพิ่มงบสุขภาพ แต่ต้องมีข้อมูลบริการจริงรองรับ
