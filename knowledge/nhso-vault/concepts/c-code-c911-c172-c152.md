# รหัส C-Code: C911, C172, C152 — ข้อผิดพลาดการเบิกจ่าย

## Definition

รหัส C ที่เกิดบ่อยในการเบิกจ่ายสำหรับหน่วยบริการที่ให้บริการผู้ป่วยนอก (OPD) และ/หรืออุปกรณ์/อวัยวะเทียม โดยเฉพาะหน่วยบริการประเภทพิเศษ เช่น มูลนิธิ หน่วยเคลื่อนที่ และโรงพยาบาล

## C911 — ข้อมูลไม่ครบสำหรับการประมวลผล

### ความหมาย
ข้อมูลผู้ป่วยนอกที่ส่งเบิกเข้ามา **ไม่เข้าเกณฑ์การเบิกจ่ายจากส่วนกลาง** เนื่องจาก:
- ขาดรายการย่อยของค่าใช้จ่าย (มีแต่ยอดรวม ไม่มีรายการ)
- ไม่ระบุเงื่อนไขที่จำเป็น เช่น:
  - ฉุกเฉิน (Emergency)
  - Walk-in / OP Anywhere
  - ส่งต่อข้ามจังหวัด (Inter-provincial referral)
  - สิทธิ์ย้ายและมีผลทันที
  - อุบัติเหตุ

### วิธีแก้ไข
1. ตรวจสอบว่าประสงค์จะเบิก **รายการอะไร** (ไม่ใช่แค่ยอดรวม)
2. เติมเงื่อนไขที่เกี่ยวข้องให้ครบ
3. ส่งข้อมูลใหม่ที่มีรายการย่อยและเงื่อนไขครบถ้วน

### ตัวอย่างที่พบ
มูลนิธิกาญจนบารมี (หน่วยคัดกรองมะเร็งเต้านมเคลื่อนที่ด้วยแมมโมแกรม) ติด C911, C172, C152 พร้อมกัน

---

## C172 — ใช้รหัส IPD ในผู้ป่วยนอก

### ความหมาย
บันทึกรหัสเบิก **IP002–IP006** (รายการอวัยวะเทียม/อุปกรณ์บำบัดรักษาสำหรับ IPD) ในเมนูผู้ป่วยนอก (OPD) ซึ่งรหัสเหล่านี้ **ใช้ได้เฉพาะผู้ป่วยใน (IPD) เท่านั้น**

### วิธีแก้ไข
- ตรวจสอบว่าผู้ป่วยรายนั้นเป็น IPD จริงหรือไม่
- ถ้าเป็น OPD → ห้ามใช้รหัส IP002–IP006
- แยกรายการให้ถูกต้องตามประเภทผู้ป่วย

---

## C152 — เบิก OPD ซ้ำซ้อน

### ความหมาย
ส่งข้อมูลเบิกกรณีผู้ป่วยนอก **ซ้ำซ้อนในโรคเดียวกัน** — คือผู้ป่วยคนเดียวกันมี 2 รายการ OPD ในวันเดียวและโรคเดียวกันอยู่ในชุดเบิก

### วิธีแก้ไข
1. ตรวจสอบว่าสามารถ **รวม visit** ได้หรือไม่
2. ถ้ารวมได้ → เบิกเป็นรายการเดียว
3. ถ้าเป็นบริการต่างแผนก → ตรวจสอบว่าระบบอนุญาตเบิกแยกหรือไม่
