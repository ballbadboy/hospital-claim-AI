# eCam System — ระบบค้นหารหัสข้อผิดพลาด (C-Code)

## Definition

**eCam System** (Electronic Claim Audit and Management System) คือระบบเว็บของ สปสช. ที่หน่วยบริการใช้สำหรับตรวจสอบสถานะการเบิกจ่าย และดาวน์โหลดเอกสารอ้างอิงรหัสข้อผิดพลาด (C-Code) พร้อมแนวทางแก้ไข

## วิธีเข้าใช้งาน

มี 2 ช่องทาง:
1. พิมพ์ URL โดยตรง: `eclaim.nhso.go.th` แล้วกด Enter
2. เว็บกลาง สปสช.: nhso.go.th → บริการออนไลน์ → งานชดเชย → eCam

## การค้นหารหัส C (Error Code)

1. เข้าระบบ eCam System
2. คลิก **ดาวน์โหลด**
3. เลื่อนหาหัวข้อ **"เอกสารประกอบการใช้งาน eCam"**
4. คลิกไอคอน **"รายละเอียดการผิดพลาดของข้อมูลรหัสติด C"**
5. ระบบแสดงรายการรหัส C พร้อม:
   - ความหมายของรหัส
   - สาเหตุที่ติด
   - วิธีปฏิบัติและแนวทางแก้ไข

**หมายเหตุ:** สปสช. อัปเดตรายการรหัส C อย่างต่อเนื่อง ควรดาวน์โหลดเวอร์ชันล่าสุดเสมอ

## รหัส C ที่มักพบบ่อย (จากครั้งที่ 24)

| รหัส C | ความหมาย | แนวทางแก้ไข |
|--------|----------|-------------|
| C305 | ไม่ได้ปิดสิทธิ์ภายในวัน | ดู [[concepts/การปิดสิทธิ์]] |
| C438 | สิทธิ์ไม่ตรงกับที่เบิก | ตรวจสอบสิทธิ์ ณ วันรับบริการ |
| C911 | ข้อมูลไม่ครบสำหรับประมวลผล | เติมรายการย่อยและเงื่อนไข |
| C172 | ใช้รหัส IP (IP002–IP006) ใน OPD | ตรวจสอบว่าเป็น IPD จริงหรือไม่ |
| C152 | เบิก OPD ซ้ำซ้อนในโรคเดียวกัน | ตรวจสอบรวม visit ได้หรือไม่ |
| C35 | ไม่ปิดสิทธิ์ภายในวันแรกของบริการ | ปิดสิทธิ์ภายในวันที่ให้บริการ |

## ความสัมพันธ์กับระบบอื่น

- **OSR Appeal:** หลังจากค้นหาสาเหตุจาก eCam แล้ว ใช้ OSR ส่งหลักฐานอุทธรณ์
- **Seamless for DMS:** ตรวจสอบสถานะรายงาน IEP และการโอนเงิน
- **Smart Money Transfer:** ยืนยันยอดเงินโอน
