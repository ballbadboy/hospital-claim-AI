# รหัส D (Deny Code) — รูปแบบรหัสปฏิเสธใหม่

## Definition

**รหัส D** (Deny Code) คือรูปแบบรหัสปฏิเสธข้อมูลใหม่ที่ สปสช.นำมาใช้ในระบบ eClaim โดยนำ **ตัวอักษร "D"** มานำหน้ารหัส C เดิม

**หลักการ:** ความหมายของรหัสเหมือนเดิมทุกประการ เพียงแต่มี "D" นำหน้า

## ตัวอย่างการแปลงรหัส

| รหัส C เดิม | รหัส D ใหม่ | ความหมาย |
|-------------|-------------|----------|
| C415 | **D415** | เบิก Telemedicine แต่ไม่มีศักยภาพที่ขึ้นทะเบียนไว้ |
| C905 | **D905** | แสดงตนยืนยันสิทธิ์ไม่ตรงวิธีที่กำหนด |
| C349 | **D349** | เบิกบริการรักษาเบื้องต้น (C) แต่ไม่มีพยาบาลเวชปฏิบัติ |

## วิธีดูคำอธิบาย

ดาวน์โหลดตาราง C และคำอธิบายการแก้ไขได้จาก:
- **เว็บ eClaim System** — ตาราง C codes + วิธีแก้ไข
- ค้นหาด้วยรหัส C ที่ตามหลัง D (เช่น D415 → ดูที่ C415)

## วิธีแก้ไข

ใช้รูปแบบการแก้ไขเดียวกับ **รหัส C** ที่ตามหลัง D — ไม่มีขั้นตอนพิเศษเพิ่มเติม

## สาเหตุที่มีรหัส D

สปสช.นำระบบ "deny" มาใช้ในกระบวนการตรวจสอบเบื้องต้น (pre-screening) ก่อนที่ข้อมูลจะผ่านไปยังการประมวลผลหลัก ทำให้รหัสปฏิเสธมีรูปแบบใหม่ที่สามารถแยกแยะได้ว่าถูก deny ในขั้นตอนใด
