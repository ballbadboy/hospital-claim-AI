# โควต้าบริการคลินิกพยาบาล

## Definition

**โควต้าบริการ** คือจำนวนการให้บริการสูงสุดที่คลินิกพยาบาลสามารถเบิกจ่ายได้ต่อวัน (หรือต่อรอบเวลา) สำหรับบริการแต่ละประเภทในระบบสปสช. ปี 2569 โควต้าจะถูก**ปล่อยรายวัน** แยกระหว่าง **CI (C)** และ **OPD Nursing** ตรวจสอบได้ผ่านระบบ M Care Plus ก่อนลงทะเบียนผู้ป่วย

## ปัญหาที่พบในการปฏิบัติงาน

จากข้อเสนอแนะของผู้ให้บริการในที่ประชุม:
- โควต้าถูกปล่อยพร้อมกันและ**หมดภายใน 2–3 นาที** ในบางพื้นที่
- ผู้สูงอายุมารอรับบริการแล้วโควต้าหมดก่อนสแกนได้
- ระบบล่ม (downtime) ช่วงปล่อยโควต้าทำให้ผู้ป่วยไม่สามารถรับบริการได้
- ผู้ประกอบการบางรายสะท้อนว่าต้องออกค่าใช้จ่ายเองช่วง reactivate

## วิธีตรวจสอบโควต้า

- ใน M Care Plus กดปุ่ม **"เช็คโควต้า"** บนหน้าลงทะเบียนผู้ป่วย
- ระบบแสดงโควต้าคงเหลือแยก C และ OPD
- ถ้าโควต้า = 0 อาจรอรับในช่วงเวลาอื่น (เช่น ช่วงบ่าย/เย็น)

## ความสัมพันธ์กับการจอง Line OA

- ถ้าผู้ป่วย**จองล่วงหน้า**ผ่าน Line OA และมี QR Code แล้ว แม้โควต้าจะแสดงเป็น 0 ก็**ยังให้บริการได้**
- ระบบโควต้าเป็น**การเสนอแนะ ไม่ใช่ lock** ถ้ามีหลักฐานการจองที่ชัดเจน (QR Code / EP Code)

## ข้อเสนอแนะจากผู้ให้บริการ

- ขอให้สปสช. **กำหนดโควต้าต่อคลินิกอย่างชัดเจน** (เช่น 5 หรือ 10 คน/วัน) แทนการปล่อยแบบ first-come-first-served
- เพื่อให้สามารถแจ้งผู้รับบริการได้ว่าวันนี้รับได้กี่คน ไม่ต้องแย่งโควต้ากัน
