# การยืนยันตัวตนด้วยบัตรประชาชน — หน่วยบริการนวัตกรรม

## Definition

มาตรการบังคับให้หน่วยบริการนวัตกรรม (คลินิกนวัตกรรม 7 วิชาชีพ) ต้องยืนยันตัวตนผู้รับบริการด้วยบัตรประชาชนทุกครั้ง ก่อนที่ สปสช. จะจ่ายเงินชดเชย มีผลบังคับตั้งแต่ **1 พฤษภาคม 2568** หากไม่ยืนยันตัวตนตามแนวทาง สปสช. จะสงวนสิทธิ์ไม่จ่ายค่าชดเชยและไม่เปิดให้อุทธรณ์

## ประวัติมาตรการ

| วันที่ | เหตุการณ์ |
|--------|-----------|
| 14 มีนาคม 2568 | สปสช. ทำหนังสือแจ้งแนวทางไปยังหน่วยนวัตกรรมทุกแห่ง |
| 25 เมษายน 2568 | ประชุมหน่วยนวัตกรรม ครั้งที่ 1 เพื่อชี้แจง |
| 29 เมษายน 2568 | ประชุมหน่วยนวัตกรรม ครั้งที่ 2 |
| **1 พฤษภาคม 2568** | **บังคับใช้จริง — หน่วยที่ไม่ปฏิบัติตามไม่ได้รับค่าชดเชย** |

## วิธียืนยันตัวตนที่ยอมรับ

ทุกวิธีต้องใช้ **บัตรประชาชนเสียบ** เป็นหลัก:

| วิธี | คำอธิบาย |
|------|-----------|
| Smart Card Reader | เสียบบัตรประชาชนเข้าเครื่องอ่านบัตร |
| KiOS | เสียบบัตรที่เครื่อง KiOS |
| แอปพลิเคชัน สปสช. | ยืนยันตัวตนผ่านแอป |
| ไทยดี (Thaidee) | ยืนยันตัวตนผ่านแอปไทยดี |
| New AEN เว็บ | ยืนยันผ่านหน้าเว็บ New Authentication |
| เป่าตังค์ | ยืนยันตัวตนผ่านแอปเป่าตังค์ |
| KTB Digital Health Platform | ใช้ฟังก์ชันเสียบบัตรประชาชนเท่านั้น |
| แอปพลิเคชันอื่น | ต้องมีฟังก์ชันใช้งานโดยใช้บัตรประชาชนเสียบ |

> **สำคัญ:** ทุกวิธีต้องผ่านการเสียบบัตรประชาชนจริง — ไม่รับการยืนยันด้วยวิธีอื่น

## ข้อยกเว้น (ผ่อนผันไม่ต้องยืนยันตัวตน)

มีเพียง 3 กรณีเท่านั้น:

1. **Telehealth** — การให้บริการทางไกล
2. **เด็กอายุต่ำกว่า 15 ปี** — ไม่ต้องยืนยันตัวตน
3. **บริการในเรือนจำ/สถานพินิจ** — ไม่มีสัญญาณอินเทอร์เน็ตและบัตรประชาชนถูกเก็บไว้ที่ทางการ; ใช้วิธีปิดสิทธิ์โดยเจ้าหน้าที่ + รายชื่อผู้รับบริการที่ผู้บัญชาการเรือนจำเซ็นรับรองแทน

## ข้อมูลการไม่ปฏิบัติตาม (1–6 พฤษภาคม 2568)

จากข้อมูลที่ สปสช. รวบรวมหลังบังคับใช้:

| รายการ | ตัวเลข |
|--------|--------|
| จำนวนครั้งที่ไม่ยืนยันตัวตน | 877 ครั้ง |
| จำนวนคน | 802 คน |
| มูลค่าเงินชดเชยที่ถูกสงวนสิทธิ์ | 412,810 บาท |
| วิชาชีพที่พบมากที่สุด | สหคลินิก |
| เขตที่พบมากที่สุด | เขตระยอง (17 หน่วย) |

## ผลของการไม่ปฏิบัติตาม

- สปสช. **ไม่จ่ายเงินชดเชย** สำหรับครั้งที่ไม่มีการยืนยันตัวตน
- **ไม่เปิดให้อุทธรณ์** กรณีที่ไม่ได้ยืนยันตัวตนตามแนวทาง
