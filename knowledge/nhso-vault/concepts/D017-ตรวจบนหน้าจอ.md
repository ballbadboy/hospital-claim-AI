# D017 — การแก้ไขผ่าน On Screen Review (SS Audit)

## Definition

**D017** คือรหัสปฏิเสธการจ่ายชดเชย (Deny Code) ที่เกิดจากการตรวจสอบของผู้ตรวจสอบ (Auditor) สปสช. ก่อนการจ่ายชดเชย หน่วยบริการสามารถอุทธรณ์ได้ผ่านระบบ **On Screen Review (OSR)** หรือ **SS Audit**

## กระบวนการตรวจสอบก่อนจ่าย

1. หน่วยบริการส่งข้อมูลเบิกจ่ายเข้าระบบ
2. **วันถัดไปก่อน 15:00 น.** ทีม สปสช. ตรวจสอบและแจ้งผล
3. รายการที่ถูกปฏิเสธ → แสดง Deny Code พร้อมเหตุผล
4. หน่วยบริการเข้า On Screen Review เพื่อดูรายละเอียดและอุทธรณ์

## วิธีเข้าระบบ On Screen Review

1. เข้าเว็บกลาง สปสช. → ระบบงานชดเชย
2. เลือก **SS Audit** (หรือ On Screen Review)
3. Login ด้วย user หน่วยบริการ
4. ค้นหารายการที่ติด D017
5. ดูรายละเอียด: **คำอธิบายจากผู้ตรวจสอบ (Auditor)** และเหตุผลที่ปฏิเสธ
6. แก้ไขข้อมูลตามที่แนะนำ → ส่งอุทธรณ์กลับ

## ข้อมูลใน On Screen Review

ระบบ OSR มีข้อมูลละเอียดกว่า Seamless/SATE for DMIS:
- เหตุผลการปฏิเสธรายบุคคล
- คำตอบโต้จาก Auditor
- ประวัติการแก้ไข
- ช่องทางส่งเอกสารเพิ่มเติม

> หากดูรายละเอียดใน IEP ของ Seamless แล้วยังไม่พอ → ให้เข้า On Screen Review เพื่อดูข้อมูลเพิ่มเติม
