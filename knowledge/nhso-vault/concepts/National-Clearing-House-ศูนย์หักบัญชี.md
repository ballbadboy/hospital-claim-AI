# National Clearing House — การจัดการธุรกรรมสิทธิ์อื่น

## Definition

National Clearing House (NCH) คือหน่วยงานภายใน สปสช. ที่ทำหน้าที่เป็นตัวกลางในการจัดการธุรกรรมการเบิกจ่ายค่าใช้จ่ายสำหรับ **สิทธิ์อื่น** ที่ไม่ใช่บัตรทอง เช่น ประกันสังคม ข้าราชการกรมบัญชีกลาง อปท. กทม. การรถไฟฯ เป็นต้น

## บทบาทของ สปสช. ในสิทธิ์อื่น

| ประเภท | บทบาท สปสช. |
|--------|-------------|
| ข้าราชการกรมบัญชีกลาง | Clearing House — หน่วยงานเจ้าของสิทธิ์โอนเงินให้สถานพยาบาล |
| ประกันสังคม (สกส.) | Clearing House |
| กรุงเทพมหานคร (กทม.) | Clearing House |
| สป. | Clearing House |
| เจ้าหน้าที่ สปสช. | Clearing House |
| ขสมก. | Clearing House |
| การรถไฟแห่งประเทศไทย (ใหม่ปี 69) | Clearing House (เริ่ม 1 มกราคม 2569) |
| อปท. | มาตรา 9 — สปสช. บริหารจัดการโอนเงินให้สถานพยาบาลโดยตรง |
| ผู้ประกันตนคนพิการ | มาตรา 9 |

## สิ่งที่เปลี่ยนแปลงปีงบ 2569

### 1. ประกันสังคม — โรคมะเร็ง

- **การรักษาโรคมะเร็งทุกกรณี**: เปลี่ยนช่องทางส่งข้อมูลไปยัง สกส. ตั้งแต่วันที่ **19 กุมภาพันธ์ 2568** เป็นต้นไป
- **ยาเคมีบำบัด ฮอร์โมน รังสีรักษา อุปกรณ์/อวัยวะเทียม** (ICD-10 ขึ้นต้นด้วย "C"): เปลี่ยนช่องทางส่งข้อมูลไปยัง สกส. ตั้งแต่ **1 ตุลาคม 2568** เป็นต้นไป

### 2. ข้าราชการกรมบัญชีกลาง — ขยาย รพ.สต.

| ระยะ | ช่วงเวลา | ขอบเขต |
|------|---------|--------|
| ระยะแรก | ก.ย.–ธ.ค. 68 | รพ.สต. สังกัด สธ. 500 แห่ง + สังกัด อปท. 20 แห่ง |
| | | เริ่มรับข้อมูลจาก รพ.สต. วันที่ 1 ต.ค. 68 |
| | | รพ.สต. 520 แห่งแรก: ยืนยันตัวตนด้วยเครื่อง EDC ของ KTB |
| | | หน่วยที่เหลือ: ใช้ระบบปิดสิทธิ์ของ สปสช. |
| ปี 69 | ต.ค. 68–ก.ย. 69 | ขยายเพิ่มเติม |
| ปี 70 | ต.ค. 69–ก.ย. 70 | ขยายทั่วประเทศ |

- หลักเกณฑ์การเบิกจ่าย: ใช้หลักเกณฑ์ของกรมบัญชีกลางเช่นเดิม

### 3. อปท. — 2 ประเด็นใหม่

1. **การแสดงตนยืนยันสิทธิ์**: ผู้มีสิทธิ์ อปท. ต้องยืนยันตัวตนเมื่อสิ้นสุดการรับบริการ (เริ่มปีงบ 2569 รายละเอียดจะแจ้งอีกครั้ง)
2. **Elective Case โรงพยาบาลเอกชน**: บันทึกผ่านโปรแกรม Elective Case ของ สปสช. เฉพาะสถานพยาบาลที่ทำสัญญากับกรมบัญชีกลางและแจ้งความประสงค์เข้าร่วม
   - จ่ายชดเชยตามเงื่อนไขผู้ป่วยในปกติ (Base Rate ตามกรมบัญชีกลาง + ค่าห้อง ค่าอาหาร อุปกรณ์รายการ)

### 4. สิทธิ์การรถไฟแห่งประเทศไทย (ใหม่ปี 69)

- **เริ่มรับเบิกจ่ายตรง**: 1 มกราคม 2569
- หลักเกณฑ์: อ้างอิงระเบียบกรมบัญชีกลาง ยกเว้นค่าห้อง ค่าอาหาร:

| ประเภทผู้มีสิทธิ์ | ค่าห้อง/ค่าอาหาร (จ่ายตามจริง ไม่เกิน) |
|-----------------|---------------------------------------|
| พนักงานการรถไฟ | 1,500 บาท/วัน |
| บุคคลในครอบครัวพนักงาน | 1,100 บาท/วัน |
| ลูกจ้างเฉพาะงาน + ครอบครัว | 1,000 บาท/วัน |
| อดีตพนักงาน + ครอบครัว | ตามระเบียบกรมบัญชีกลาง |

## โปรแกรมที่ใช้

- ปีงบ 69 ยังคงใช้โปรแกรมเดิมในการเบิกจ่าย
- เปลี่ยนเฉพาะระบบตรวจสอบสิทธิ์ และระบบขอเลขอนุมัติสิทธิ์อื่น
- รวมถึงระบบบันทึกค่ารักษาพยาบาลด้วย Base Rate ของสิทธิ์ อปท.

## หลักเกณฑ์การเบิกจ่ายแต่ละสิทธิ์

| สิทธิ์ | หลักเกณฑ์ |
|--------|----------|
| อปท., กทม., เจ้าหน้าที่ สปสช., สป., ขสมก., การรถไฟ | ระเบียบกรมบัญชีกลาง |
| ประกันสังคม | ประกาศสำนักงานประกันสังคม |
| ผู้ประกันตนคนพิการ | ประกาศและเงื่อนไขที่ สปสช. กำหนด |
