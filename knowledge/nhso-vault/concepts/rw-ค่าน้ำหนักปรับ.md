# Adjusted RW (AdjRW / Adjusted Relative Weight)

## Definition

Adjusted RW หรือ AdjRW คือค่าน้ำหนักสัมพัทธ์ที่ปรับแล้ว ใช้วัด "แต้ม" ของผลงานการให้บริการผู้ป่วยในแต่ละราย โดยคำนวณจาก Relative Weight (RW) ของกลุ่มโรค DRG ปรับด้วยตัวคูณต่างๆ เช่น ประเภทโรงพยาบาล ความซับซ้อนของผู้ป่วย และปัจจัยอื่น

## How It Works

- แต่ละผู้ป่วยในที่รับไว้รักษาจะถูกจัดกลุ่มตามระบบ DRG (Diagnosis Related Group) และได้รับค่า RW
- ค่า RW ถูกปรับด้วยตัวคูณต่างๆ (เช่น CCUF, ปรับรถล่าช้า) เพื่อให้ได้ AdjRW ของแต่ละเคส
- ผลรวม AdjRW ของโรงพยาบาล = ตัวชี้วัดผลงาน IP ทั้งหมดของโรงพยาบาลนั้นในช่วงเวลาที่กำหนด
- สปสช.ใช้ **ผลรวม AdjRW ระดับเขต** เป็นตัวหารวงเงิน Global Budget เพื่อหาอัตราจ่าย:

```
อัตราจ่าย (บาท/AdjRW) = วงเงิน Global Budget ÷ ผลรวม AdjRW ทั้งเขต
```

## การใช้งานใน Statement

- แต่ละ Statement จะแสดงจำนวน AdjRW ของเดือนนั้น
- นำ AdjRW คูณกับอัตราจ่าย (เช่น 7,719 บาท) แล้วหักเงินเดือน หักพรบ. = ยอดชดเชยสุทธิ
- ในกรณี Random Sampling และ Extrapolation: AdjRW อาจถูกปรับเพิ่มหรือลดจากผลการตรวจสอบเวชระเบียน

## ความสัมพันธ์กับอัตราจ่าย

| สถานการณ์ | ผลต่อ AdjRW รวม | ผลต่ออัตราจ่าย |
|----------|----------------|----------------|
| ผลงานสูงกว่าประมาณการ | ตัวหารใหญ่ขึ้น | อัตราจ่ายลดลง |
| ผลงานต่ำกว่าประมาณการ | ตัวหารเล็กลง | อัตราจ่ายสูงขึ้น |
| ข้อมูลติด C หรือ D | ไม่นับใน AdjRW | อัตราจ่ายสูงขึ้นเล็กน้อย แต่โรงพยาบาลไม่ได้เงินส่วนนั้น |
