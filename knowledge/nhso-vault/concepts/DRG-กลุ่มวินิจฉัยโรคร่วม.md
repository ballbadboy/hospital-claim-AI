# DRG (Diagnosis Related Group)

## Definition

DRG หรือ Diagnosis Related Group คือระบบจำแนกกลุ่มวินิจฉัยโรคที่ใช้ในการคำนวณน้ำหนักสัมพัทธ์ (Relative Weight / RW) ของผู้ป่วยแต่ละราย เพื่อใช้เป็นฐานในการคำนวณค่าบริการผู้ป่วยในที่สปสช.จะชดเชยให้โรงพยาบาล

## How It Works

- แพทย์ลงรหัสโรคและหัตถการ (ICD-10, ICD-9-CM) สำหรับผู้ป่วยแต่ละราย
- ระบบ DRG จัดกลุ่มผู้ป่วยตามรหัสโรค ความซับซ้อน ภาวะแทรกซ้อน → ได้ "DRG Code" และ "Relative Weight (RW)"
- RW ถูกปรับเพิ่มเติม (CCUF, ปรับรถล่าช้า ฯลฯ) → ได้ Adjusted RW สำหรับแต่ละเคส

## เวอร์ชันของ DRG

ประเทศไทยใช้ DRG ที่พัฒนาโดย สวรส. มีการปรับปรุงเวอร์ชันตามระยะเวลา:
- **DRG Version 5:** ใช้ในปีงบประมาณก่อนหน้า
- **DRG Version 6:** เวอร์ชันใหม่ที่ปรับโครงสร้างการจำแนกโรคและน้ำหนัก RW

### ส่วนต่าง DRG v5 → v6

- เมื่อเปลี่ยนเวอร์ชัน DRG ค่า RW ของหลายโรคกลุ่มอาจเปลี่ยนแปลง
- สปสช.มีมติบอร์ดให้จ่ายส่วนต่างที่เกิดจากการเปลี่ยน DRG v5 เป็น v6 เพิ่มเติม
- ปีงบประมาณ 2568: ส่วนต่าง DRG v5 → v6 มูลค่า **~541 ล้านบาท** (จะโอนในสัปดาห์ถัดไปจากงวดแรก)

## ความสัมพันธ์กับการตรวจสอบ

- Auditor ในกระบวนการ Random Sampling 3% จะตรวจสอบความถูกต้องของรหัสโรคที่นำมาคำนวณ DRG
- หากพบว่ารหัสโรคผิด → AdjRW เปลี่ยน → ยอดเงินเปลี่ยน
