# การตรวจประเมินหน่วยบริการประจำปีผ่านระบบ OSS

## Definition

**การตรวจประเมินประจำปี** คือกระบวนการที่หน่วยบริการในระบบหลักประกันสุขภาพแห่งชาติต้องผ่านการตรวจสอบและขึ้นทะเบียนใหม่ **อย่างน้อยปีละ 1 ครั้ง** ตามข้อบังคับของ สปสช. โดยปีงบประมาณ 2569 เป็นปีแรกที่ดำเนินการแบบ **ดิจิทัล 100%** ผ่านระบบ One Stop Service (OSS) แทนการส่งเอกสารกระดาษ

## สถิติ (ณ 27 กุมภาพันธ์ 2569)

| รายการ | จำนวน |
|--------|--------|
| หน่วยบริการทั้งหมดในระบบ | ~30,000+ แห่ง |
| หน่วยนวัตกรรม (ยกเว้น) | ~15,000 แห่ง |
| หน่วยที่ต้องตรวจประเมิน | ~16,000 แห่ง |
| ยังไม่ได้เข้าระบบ | **9,417 แห่ง** |

## กำหนดเวลา

| รอบ | วันที่ |
|-----|--------|
| กำหนดเดิมรอบแรก | 26 กุมภาพันธ์ 2569 |
| **ขยายเวลา** | **15 มีนาคม 2569** |
| สปสช.เขตอนุมัติภายใน | 3 วันทำการ (หลังส่งข้อมูล) |

## ขั้นตอนการตรวจประเมิน

### ขั้นที่ 1: สมัครเข้าระบบ OSS
- ผู้มีอำนาจลงนาม (ผู้อำนวยการ) หรือผู้รับมอบอำนาจ เข้าระบบด้วย **ThaiD**
- หน่วยราชการ: แนบคำสั่งแต่งตั้ง/มอบหมาย
- หน่วยเอกชน: ลิงก์ข้อมูลกับกรมพัฒนาธุรกิจการค้า (DBD) และเลขสถานพยาบาล 5 หลัก (เดิม) หรือ 9 หลัก (ใหม่)

### ขั้นที่ 2: บันทึกข้อมูลหน่วยบริการ
- ข้อมูลที่ระบบดึงอัตโนมัติ (ไม่ต้องกรอก): จำนวนประชากรขึ้นทะเบียน, ประเภทหน่วยบริการ
- ข้อมูลที่ต้องกรอก: **เลขบัตรประชาชนบุคลากร** (ระบบเชื่อม API สภาวิชาชีพอัตโนมัติ — ไม่ต้องแนบใบอนุมัติ)

### ขั้นที่ 3: ส่งและรอการอนุมัติ
- สปสช.เขตอนุมัติภายใน **3 วันทำการ**
- หากเกิน 3 วัน: ติดต่อสปสช.เขต
- กรณีไม่ผ่านเกณฑ์: สปสช.เขตแจ้งให้ทำแผนพัฒนา → ตรวจประเมินซ้ำอีกครั้ง

## ผลลัพธ์ที่เป็นไปได้

| ผล | ความหมาย |
|----|----------|
| ผ่านเกณฑ์ | ขึ้นทะเบียนสำเร็จ ปีงบ 2569 |
| ไม่ผ่านเกณฑ์ | ได้เวลาปรับปรุง → ตรวจซ้ำ → ยังเบิกจ่ายได้ปกติระหว่างนี้ |

> **สำคัญ:** ไม่ผ่านเกณฑ์ ≠ ยกเลิกการเป็นหน่วยบริการ หน่วยบริการยังคงให้บริการและเบิกจ่ายได้ตามปกติระหว่างที่ดำเนินการแก้ไข

## ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ไข |
|-------|----------|
| แพทย์มีชื่อกลาง ระบบหาไม่พบ | ตรวจสอบชื่อที่แพทยสภาก่อน → ถ้าไม่ตรง แจ้งสปสช.เขตตรวจ API |
| เวลาบุคลากรซ้ำซ้อน (2 ที่) | บันทึก Full Time ที่หลัก, Part Time ที่ที่ 2 — เวลาต้องไม่ทับกัน |
| ลบ/แก้ไขข้อมูลบุคลากรที่ยืนยันแล้ว | แจ้งสปสช.เขตให้ดำเนินการ (ทำเองไม่ได้หลัง Submit) |
| บริษัทเอกชน 18 แห่ง ขึ้นไม่ได้ | ส่งอีเมล register@nhso.go.th พร้อม capture หน้าจอ + รหัสสถานพยาบาล |
| CPP กับตรวจประเมินประจำปี | ทำแยกกัน — ตรวจประเมินประจำปีทำก่อน CPP ทำหลัง |

## ประเภทหน่วยที่ต้องประเมินกี่ระบบ

- โรงพยาบาลชุมชนที่มี 3 ประเภท (ปฐมภูมิ, ประจำ, รับส่งต่อ): ต้องประเมิน **ทั้ง 3 ประเภท**
- หน่วยฟอกไต outsource: แยกประเมินแต่ละหน่วย

## ความสัมพันธ์กับการเบิกจ่าย

ข้อมูลบุคลากรที่บันทึกใน OSS จะใช้ประกอบการ **ขึ้นทะเบียนศักยภาพ (CPP)** เพื่อเบิกค่าบริการเฉพาะทาง (เช่น ยาละลายลิ่มเลือด, CRRT, CAPD) โดยอัตโนมัติ — ไม่ต้องส่งเอกสารเพิ่มเติม
