# NHO Budget Portal — เว็บไซต์ตรวจสอบการโอนเงินกองทุน

## Definition

NHO Budget Portal คือระบบสารสนเทศของ สปสช. ที่ให้หน่วยบริการตรวจสอบ**รายงานการโอนเงินกองทุน** ทั้งรายวัน รายบุคคล และยอดรวมทั้งปี รวมถึงใบเสร็จรับเงินอิเล็กทรอนิกส์ เข้าถึงผ่าน nhho.go.th (เมนู NHO Budget)

## วิธีเข้าใช้งาน

### รายงานการโอนเงินรายวัน (ไม่ต้อง Login)

1. เข้า **nhho.go.th**
2. เลือก "สำหรับหน่วยบริการ" → "บริการออนไลน์"
3. เลือก **NHO Budget** → "รายงานการจ่ายเงินกองทุน (2566 ถึงปัจจุบัน)"
4. ใส่รหัสหน่วยบริการ → กดค้นหา

### รายงานรายบุคคล (ต้อง Login)

- ต้องสมัครขอ username/password ผ่านผู้บริหารหน่วยบริการ
- ผู้อำนวยการ/หัวหน้าหน่วยลงนามอนุมัติ + แนบหนังสือมาที่ สปสช.
- สมัครออนไลน์ได้ที่หน้าเว็บ NHO Budget
- ล็อคอินผิด 3 ครั้ง → ระบบล็อค ต้องติดต่อเจ้าหน้าที่

## รายงานที่มีในระบบ

| รายงาน | รายละเอียด | Login |
|--------|-----------|-------|
| รายงานการโอนเงินรายวัน | ยอดรวมกองทุน + งวด + วันที่โอน | ไม่ต้อง |
| รายงานรายบุคคล | รายละเอียดผู้รับบริการแต่ละราย | ต้อง |
| ยอดรวมทั้งปี | ภาพรวมการโอนตั้งแต่ 1 ต.ค. | ต้อง |
| ยอดคงค้าง | ยอด ณ วันที่ระบุ (ใช้ยืนยันบัญชีปลายปี) | ต้อง |
| รายงานผู้บริหาร | สรุปการโอนรายกองทุน แยกเขต/จังหวัด | ต้อง |
| ใบเสร็จรับเงินอิเล็กทรอนิกส์ | ออกโดยธนาคารกรุงไทย ดาวน์โหลดได้ | ต้อง |

## ข้อมูลที่เห็นในรายงานโอนเงินรายวัน

- **ยอดเงินพึงรับ** — เงินที่ควรโอนในรอบนั้น
- **ยอดเงินที่โอนจริง** — อาจต่างกันถ้ามีหักกลบหนี้
- **ชะลอโอน** — กรณีมีหนี้หรือถูก Flag ผิดปกติ
- **รอหักกลบ** — มีหนี้เกินยอดพึงรับ รอรอบถัดไป
- **ภาษีหัก ณ ที่จ่าย** — สำหรับหน่วยเอกชน (e-withholding tax ส่งกรมสรรพากรอัตโนมัติ)
- **หลักประกัน** — กรณีหน่วยที่มีวางหลักประกัน

## ใบเสร็จรับเงินอิเล็กทรอนิกส์

- ธนาคารกรุงไทยออกใบเสร็จแทน สปสช.
- ส่งทางอีเมลที่แจ้งไว้โดยอัตโนมัติ
- ดาวน์โหลดได้ที่หน้าเว็บ NHO Budget
- ไฟล์ PDF ใช้รหัส 9 หลัก (username) เปิด
- หน่วยเอกชน: ใช้สแกน QR Code ชำระหนี้ได้ทันทีหากต้องการ

## กรณีหน่วยบริการเอกชน — ใบตอบรับ

เมื่อรับเงินแล้ว ต้องส่ง**ใบตอบรับการโอนเงิน/ใบเสร็จรับเงิน**กลับมาที่ สปสช. ตามระเบียบ (ระบบอำนวยความสะดวก — ใช้หนังสือแนบท้ายลงนามส่งกลับได้เลย)

## ตรวจสอบยอดคงค้างสำหรับผู้สอบบัญชี

ใช้ตรวจสอบประกอบ [[concepts/หนังสือยืนยันการตรวจสอบ]]:
1. Login → เมนูยอดคงค้าง
2. ระบุปีงบประมาณ (เช่น 2568) → เลือกวันที่ 30 กันยายน
3. ระบบแสดงยอดที่ สปสช. บันทึกบัญชีไว้ → เปรียบเทียบกับหนังสือ

หมายเหตุ: ข้อมูลรีเฟรชตลอดวัน — ควรดูตอนเย็นเพื่อให้ยอดตรงกับบัญชี
