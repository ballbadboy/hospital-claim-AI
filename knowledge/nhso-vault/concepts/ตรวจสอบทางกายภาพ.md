# Physical Verification — การยืนยันตัวตนทางกายภาพสำหรับกลุ่มเปราะบาง

## Definition

**Physical Verification** คือวิธียืนยันตัวตน/ปิดสิทธิ์ด้วย **บัตรประชาชน + Smart Card Reader** สำหรับกลุ่มประชาชนที่ไม่สามารถใช้ระบบ FVS (สแกนใบหน้า) หรือ Line Application ได้ ตามประกาศ สปสช. ลงวันที่ **20 กุมภาพันธ์ 2569**

## กลุ่มเป้าหมาย 4 กลุ่ม

| กลุ่ม | ลักษณะ |
|-------|--------|
| 1 | อายุน้อยกว่า 15 ปี ที่ไม่พบข้อมูลใบหน้าในระบบ หรือไม่มี Smartphone |
| 2 | ผู้สูงอายุที่ไม่สามารถใช้เทคโนโลยี หรือมีข้อจำกัดด้านลายนิ้วมือ/อุปกรณ์ดิจิทัล |
| 3 | ผู้ป่วยที่มีข้อจำกัดทางร่างกาย/พยาธิสภาพ เช่น บาดเจ็บที่ใบหน้า ความผิดปกติทางใบหน้า |
| 4 | กลุ่มเปราะบางที่มีข้อจำกัดในการเข้าถึงเทคโนโลยีหรืออุปกรณ์ดิจิทัล |

## ลำดับขั้นตอน (ต้องทำตามลำดับ)

```
1. พยายาม FVS (สแกนใบหน้า) ก่อน
   ↓ ไม่ผ่าน
2. พยายาม Line Application ก่อน
   ↓ ไม่ผ่าน/ไม่สามารถทำได้
3. ใช้วิธี Physical Verification
   → เสียบบัตรประชาชนผู้รับบริการ (SMC)
   → หรือเสียบบัตรผู้ปกครอง/ผู้ดูแล
   → หรือบัตรเจ้าหน้าที่ (กรณีสุดท้าย — ถูกเพ่งเล็งตรวจสอบเข้มขึ้น)
```

> **สำคัญ:** ต้องมี transaction ผลปฏิเสธ FVS อยู่ในระบบก่อน จึงจะใช้ Physical Verification ได้

## ประเภทบริการที่ต้องเลือก

เมื่อใช้ Physical Verification ให้เลือกประเภทบริการใน FVS/ระบบปิดสิทธิ์เป็น:

> **"บริการนวัตกรรมโดยการเสียบบัตรประชาชน (เฉพาะเด็ก ผู้สูงอายุ ผู้มีปัญหาด้านใบหน้า และกลุ่มเปราะบางที่ไม่สามารถยืนยันตัวตนโดยผ่านระบบ FVS หรือ LINE)"**

## กรณีพิเศษ: เด็กต่างด้าวอายุต่ำกว่า 7 ปี

- ตรวจสอบสิทธิ์ด้วยเลข 13 หลักผ่านระบบ New Authentication ก่อน
- หากมีสิทธิ์ UC → ใช้ FVS หรือ Line ก่อน
- หากทำไม่ได้ → ใช้บัตรผู้ปกครอง/ผู้ดูแล (ตาม ประกาศ 20 ก.พ. 2569)
- กรณีแม่เป็นต่างด้าวไม่มีบัตร → สุดท้ายใช้บัตรเจ้าหน้าที่ (ถูกเพ่งเล็ง)

## กรณีผู้สูงอายุ >60 ปี

- สแกนใบหน้า **1 ครั้ง** หากไม่ผ่าน → ไม่ต้องสแกนซ้ำ
- ใช้บัตรประชาชนผู้สูงอายุเสียบแทน หรือบัตรผู้ดูแล
- เลือกประเภทบริการให้ถูกต้อง

## ฐานกฎหมาย

- **ประกาศ สปสช.** เรื่องการแสดงยืนยันสิทธิ์เมื่อสิ้นสุดการรับบริการด้วยระบบ FVS
- ลงวันที่ **20 กุมภาพันธ์ 2569**
- เผยแพร่ใน Facebook Live ครั้งที่ 89 และทุกช่องทาง สปสช.
