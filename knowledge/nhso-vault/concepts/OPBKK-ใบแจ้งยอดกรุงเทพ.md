# OPBKK Statement — ผู้ป่วยนอก กทม. (เขต 13)

## Definition

**OPBKK** คือบริการผู้ป่วยนอก (OP) ในเขตกรุงเทพมหานคร (เขตสุขภาพที่ 13) ซึ่งมีกระบวนการตรวจสอบพิเศษก่อนการออก Statement แตกต่างจาก OP ทั่วไปในภูมิภาค

## กระบวนการออก Statement ของ OPBKK

```
หน่วยบริการส่งข้อมูลบริการ OP (กทม.)
    ↓
Auditor ตรวจสอบ
    ↓
หน่วยบริการต้นสังกัดทวนสอบ
    ↓
ผ่านการทวนสอบ → ออก Statement (ช้ากว่าปกติ)
    ↓
Statement ชั่วคราวรหัส "99" ออกก่อน
    ↓
Update เป็น Statement จริง → หน่วยบริการอุทธรณ์ได้
```

## เปรียบเทียบกับ OP ปกติ

| รายการ | OP ภูมิภาค | OPBKK (เขต 13) |
|--------|-----------|----------------|
| ระยะเวลาออก Statement | 15 วัน | นานกว่า 15 วัน |
| การตรวจสอบก่อนจ่าย | ปกติ | Auditor + หน่วยต้นสังกัด |
| Statement ชั่วคราว | ไม่มี | รหัส 99 (ก่อนออก Statement จริง) |
| ช่องทางอุทธรณ์ | ปกติ | ต้องรอ Statement จริงก่อน |

## ลักษณะเทียบกับบริการอื่น

คล้ายกับ **OP ข้ามจังหวัด (Inter-provincial OP)** ในภูมิภาค ที่ต้องให้หน่วยบริการประจำเข้ามาตรวจสอบก่อนออก Statement

## ข้อควรระวัง

- หน่วยบริการที่ได้รับ Statement ชั่วคราวรหัส 99 — ยังต้องรอ Statement จริงอีกรอบ
- **ไม่มีการประเภทบริการนี้ให้อุทธรณ์** จนกว่า Statement จริงจะออก
- เมื่อ Statement จริงออกแล้ว — ตรวจสอบรายการและอุทธรณ์ได้ตามปกติ
