# กายภาพบำบัดและการเบิกจ่าย

## Definition

บริการกายภาพบำบัดและฟื้นฟูสมรรถภาพในระบบ UC สามารถเบิกได้หลายรูปแบบขึ้นอยู่กับประเภทของหน่วยบริการและลักษณะผู้ป่วย

## How It Works

### ประเภทการบันทึก
1. **กายภาพบำบัด OPD ทั่วไป:** บันทึกผ่านระบบ EClaim (EM)
2. **การฝังเข็ม:** บันทึกผ่านระบบ EM ส่วนหนึ่ง
3. **Disability System:** คลินิกกายภาพบำบัดที่เป็นหน่วยนวัตกรรม บันทึกผ่านระบบ Disability
4. **ผู้ป่วยระยะกลาง (Intermediate Care):** บันทึกแยกต่างหากตามประกาศ

### DS032 — ปฏิเสธจากการตรวจสอบศักยภาพ
- C/DS032 เกิดจากระบบตรวจสอบศักยภาพในการให้บริการเทียบกับจำนวนบริการที่เกิดขึ้น
- **การอุทธรณ์:** ต้องทำเป็นเอกสาร (หนังสืออุทธรณ์) ส่งให้ สปสช.
- ระบบ Simulate for DMIS ยังไม่มีระบบอิเล็กทรอนิกส์รองรับการอุทธรณ์
- แนบหลักฐานศักยภาพผู้ให้บริการ (ผู้รักษา/นักกายภาพ) ในหนังสือ

### DS905 — ยืนยันสิทธิ์ไม่สมบูรณ์
- เกิดจากไม่ได้ใช้บัตรประชาชน Smart Card หรือเทียบเท่า/Thai D ยืนยันสิทธิ์เมื่อสิ้นสุดการรับบริการ
- ต้องยืนยันสิทธิ์ทั้งก่อนและหลังรับบริการ

### ปัญหาการปิดสิทธิ์ซ้ำ 2 รูปแบบ
หน่วยบริการกายภาพบำบัดบางแห่งพบปัญหาข้อมูลไม่ถูกประมวลผลเพราะ:
1. ปิดสิทธิ์วิธีที่ 6 (Face/Authentication ก่อนบริการ + บันทึก Disability ภายในวัน)
2. และยังปิดสิทธิ์อีกครั้งผ่าน NP System หรือ API (End-point)
→ ทำให้เกิดความขัดกัน ข้อมูลไม่ถูกนำมาประมวลผล

### จ่าย 0 บาท
- หากระบบประมวลผลและออก Statement แต่จ่าย 0 บาท หมายความว่ารายการไม่เข้าเงื่อนไขการจ่ายชดเชย
- ตรวจสอบว่าผู้รับบริการมีสิทธิ์ UC จริงหรือไม่
- ตรวจสอบว่ารายการอยู่ในประกาศกายภาพบำบัดหรือประกาศผู้ป่วยระยะกลางหรือไม่
- ตรวจสอบระบบบันทึกว่าใช้ถูกช่องทาง (EClaim, Disability, หรือระบบอื่น)
