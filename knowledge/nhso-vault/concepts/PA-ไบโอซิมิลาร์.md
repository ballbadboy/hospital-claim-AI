# PA ยาไบโอซิมิลาร์ — แนวทางชั่วคราวกรณีระบบขัดข้อง

## Definition

ระบบ Prior Authorization (PA) สำหรับยาไบโอซิมิลาร์ (Biosimilar / ยาไบโอซิมิลาร์) ในระบบสปสช. ที่ได้รับผลกระทบจากการปรับปรุงระบบ SSO (Single Sign-On) เมื่อต้นเดือนเมษายน 2569 ทำให้โปรแกรม PA เข้าไม่ได้หรือทำงานไม่ได้ชั่วคราว

## How It Works

### สาเหตุของปัญหา
- สปสช.ปรับปรุงระบบ Single Sign-On (SSO/ไทยดี) สำหรับโปรแกรมต่างๆ
- ส่งผลให้โปรแกรม PA สำหรับยาไบโอซิมิลาร์ได้รับผลกระทบ
- มีผลในช่วงต้นสัปดาห์ของ เมษายน 2569

### แนวทางชั่วคราว 2 กลุ่ม

**กลุ่มที่ 1 — ผู้ป่วยรายเก่า (บันทึกในระบบแล้ว รอการอนุมัติ):**
- ไม่ต้องดำเนินการใดๆ
- สปสช.จะดึงข้อมูลที่บันทึกไว้แล้วไปให้ Auditor/ผู้เชี่ยวชาญตรวจสอบ
- สปสช.จะบันทึกในระบบแทนหน่วยบริการ
- แจ้งผลการอนุมัติผ่าน **Email** ของโรงพยาบาล

**กลุ่มที่ 2 — ผู้ป่วยรายใหม่ (ยังไม่ได้ลงทะเบียน/ต่ออายุ/ยกเลิก):**
- ดาวน์โหลดแบบฟอร์ม 3 ประเภท (ผ่าน QR Code ที่ประกาศด่วน)
- กรอกรายละเอียดให้ครบถ้วน
- ส่งกลับผ่าน QR Code ให้ สปสช.
- สปสช.จะกรอกข้อมูลลงโปรแกรม PA แทนหน่วยบริการ
- ส่งให้ Auditor ตรวจสอบ แล้วแจ้งผลทาง Email

### ระยะเวลา
- แนวทางชั่วคราวนี้ใช้จนกว่าระบบจะปกติ
- คาดว่าระบบจะปกติก่อน **25 เมษายน 2569**

### ช่องทางติดต่อ
- ผู้ประสานงาน: **คุณปนัดดา** (หมายเลขโทรศัพท์แสดงในสไลด์)
