# MRA — Medical Record Audit คู่มือการตรวจประเมินคุณภาพการบันทึกเวชระเบียน

## Definition

MRA (Medical Record Audit Guideline) คือคู่มือการตรวจประเมินคุณภาพการบันทึกเวชระเบียน จัดทำโดย **สำนักตรวจสอบการชดเชยและคุณภาพบริการ สปสช.** ฉบับปัจจุบัน: **พ.ศ. 2563** (พิมพ์ครั้งที่ 1)

## ประวัติ

| ปี | ฉบับ |
|----|------|
| 2553 | MRA ฉบับแรก |
| 2557 | ฉบับปรับปรุง |
| 2563 | **ฉบับปัจจุบัน** (ล่าสุด) |

## วัตถุประสงค์

- ตรวจสอบความถูกต้องของข้อมูลที่บันทึกในเวชระเบียน
- ทบทวนกระบวนการเพื่อหาจุดบกพร่อง → ปรับปรุงคุณภาพบริการ
- ใช้เป็นหลักฐานทางการแพทย์ กฎหมาย และ **การเบิกจ่ายกองทุน** (สปสช., กรมบัญชีกลาง, ประกันสังคม)

## ความเชื่อมโยงกับ Claim สปสช.

เวชระเบียนที่ไม่ผ่าน MRA → **claim ถูก deny** เพราะ:
- Discharge summary ไม่สมบูรณ์ → ICD coding ผิด → DRG ผิด
- ไม่มี Informed consent → ไม่ได้รับค่าหัตถการ
- Progress note ไม่ครบ → audit ปฏิเสธ

## เจตคติที่จำเป็นในการทำ MRA

> การบันทึกเวชระเบียนต้องทำอย่างดีที่สุดในผู้ป่วยทุกราย **มิใช่ทำเพื่อรองรับการตรวจสอบเท่านั้น** แต่ทำเพื่อช่วยให้ได้บริการที่ดีที่สุดแก่ผู้รับบริการ

## ขอบเขตการตรวจประเมิน

| ประเภท | เนื้อหา |
|--------|---------|
| **OPD/ER** | ผู้ป่วยนอกทั่วไป + โรคเรื้อรัง + ฉุกเฉิน |
| **IPD** | ผู้ป่วยใน — 12 หัวข้อ แต่ละหัวข้อ 7-9 เกณฑ์ |
| **จิตเวช** | OPD/IPD สำหรับผู้ป่วยจิตเวช |

## เวชระเบียนอิเล็กทรอนิกส์ (EMR)

- ต้องสืบค้นใน log in ได้ว่าแพทย์ผู้ใดเป็นผู้บันทึก
- ระบบต้องไม่สามารถแก้ไขย้อนหลังโดยผู้อื่นได้
- กรณีพิมพ์เอกสารส่งตรวจ ใช้หนังสือนำส่งแทนการลงนามรับรองทุกแผ่น
