# Statement IP (ใบแจ้งรายการจ่ายเงินผู้ป่วยใน)

## Definition

Statement IP คือเอกสารสรุปรายการจ่ายเงินค่าบริการผู้ป่วยใน ที่สปสช.ออกให้แต่ละหน่วยบริการในแต่ละงวด บอกรายละเอียดว่าได้รับเงินจากส่วนใดบ้าง และถูกหักเงินคืนจากส่วนใดบ้าง

## โครงสร้าง Statement IP (รอบปิด Global Budget ปี 2568)

ไฟล์ Excel สรุปรายโรงพยาบาลประกอบด้วย 7+ คอลัมน์หลัก:

| คอลัมน์ | รหัส Statement | ความหมาย |
|---------|---------------|----------|
| 1 (H) | `6808_IP_01` | รายรับสิงหาคม: AdjRW ส.ค. × อัตราปิด Global Budget |
| 2 (I) | `6808_IP_01_อุทธรณ์` | ยอดอุทธรณ์เดือนสิงหาคม (บวกหรือลบ) |
| 3 (J) | `6809_IP_01` | รายรับกันยายน 1-15 ก.ย.: AdjRW × อัตราปิด |
| 4 (K) | `6809_IP_01_อุทธรณ์` | ยอดอุทธรณ์เดือนกันยายน |
| 5 (L) | `6809_IP_03` | ผลการสุ่มตรวจ Random Sampling 3% |
| 6 (M) | `0809_IP_04` | Recalculate 8,350 → อัตราจริง (ส่วนที่เกี่ยวกับ Extrapolation) |
| 7 (N) | `6809_IP_05` | Recalculate 8,350 → อัตราจริง (ส่วนอื่น: Newborn, Home Watch, MIS) |
| **O** | — | **ยอดสุทธิรวมทั้งหมด** (1+2+3+4+5+6+7) |

> **หมายเหตุ:** รหัส Statement ใช้รูปแบบ `ปีงบ_เดือน_ประเภท_ลำดับ` เช่น `6808` = ปี 68 เดือน 08

## วิธีดาวน์โหลด Statement

1. เข้าระบบ **EK Web** หรือ **EKB Component** ของสปสช.
2. Login ด้วย Username/Password ของโรงพยาบาล
3. เลือกเมนู Statement UC
4. เลือกปี พ.ศ., ประเภทบริการ (ผู้ป่วยใน), ช่วงเวลา
5. กดดาวน์โหลด

## การกระทบยอด

- ยอดสุทธิใน Excel คอลัมน์ O ต้องตรงกับยอดที่โอนจริงในระบบ SMT (Smart Money) / NHSO Budget
- ถ้ายอดไม่ตรง: ตรวจสอบในส่วนของ "การหักเงินเดือน" ซึ่งแสดงในหน้า Summary ของ Statement
- หน่วยบริการในสังกัด สป. ต้องคำนึงถึงค่า K ที่ทำให้อัตราจ่ายต่างจาก BR01

## ชีทภายใน Statement แต่ละรายการ

- **ชีทสรุป (Sheet 1):** ยอดรวม IP แบ่งตาม AE, High Cost, IP, ODS/MIS ฯลฯ
- **ชีทรายบุคคล (Sheet 4+):** ข้อมูลแต่ละเคส — TID, IEP, AdjRW ที่ผ่านการตรวจสอบ, อัตราจ่าย, การหักต่างๆ, ยอดสุทธิรายการ
- **IEP File:** ไฟล์แยกสำหรับเปรียบเทียบ AdjRW เก่าและใหม่ (ดาวน์โหลดเพิ่มเติม)

## ข้อควรระวัง

- **BR01 vs BR02:** หน่วยบริการสังกัด สป. จะเห็นอัตราจ่าย (BR02) ต่ำกว่าอัตรากลาง (BR01) เนื่องจากการถ่วงน้ำหนักค่า K
- **ODS:** กรณีให้บริการ ODS ไม่มีการถ่วงน้ำหนักค่า K จะได้อัตรา BR01 เท่ากับหน่วยบริการนอก สป.
- **ยอดงวดนี้ยังไม่ครบ:** งวดที่มีตัวเลขติดลบอาจเป็นเพราะยังไม่ได้รับก้อน IP Cleanup, DRG และ USEP

## อัปเดต — ครั้งที่ 88 (13 กุมภาพันธ์ 2569)

### REP ก่อน/หลัง Statement IP ออก
- **ก่อน Statement ออก (ระหว่างเดือน):** โหลด REP → ยอดชดเชย IP จะ**ยังไม่แสดง** เพราะต้องรอคำนวณอัตราจ่าย Global Budget รายเขตก่อน
- **หลัง Statement ออก (สิ้นเดือน):** โหลด REP ใหม่อีกครั้ง → ยอดชดเชย IP จะปรากฏพร้อมอัตราจ่ายจริง
- Remark Column: กรณี ODS/MIS หรือ Home Ward จะมีหมายเหตุพิเศษในคอลัมน์ Remark
