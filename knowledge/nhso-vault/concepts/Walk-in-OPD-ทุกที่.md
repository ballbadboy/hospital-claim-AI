# Walk-in / OP Anywhere — การเบิกจ่ายผู้ป่วยนอกข้ามพื้นที่

## Definition

**Walk-in** หรือ **OP Anywhere** คือการที่ผู้มีสิทธิ์บัตรทอง (UC) ไปรับบริการผู้ป่วยนอกที่หน่วยบริการซึ่งไม่ใช่หน่วยประจำของตน โดยไม่มีใบส่งตัว ระบบมีกติกาต่างกันตามว่าผู้ป่วยอยู่ **ในจังหวัดเดียวกัน** หรือ **ข้ามจังหวัด**

## ผู้มีสิทธิ์เบิก Walk-in

- สิทธิ์ UC (บัตรทอง)
- สิทธิ์ว่าง
- ผู้ประกันตน
- คนพิการ

## ประเภทและอัตราจ่าย

### กรณีที่ 1: Walk-in ข้ามจังหวัด (เบิกกองทุนกลาง)

| รายการ | รายละเอียด |
|--------|-----------|
| เงื่อนไข | ผู้รับบริการมีสิทธิ์ในจังหวัดอื่น มารับบริการข้ามจังหวัด |
| Project Code | Walk-in |
| แหล่งเงิน | กองทุนกลาง (เบิกที่ส่วนกลาง) |
| อัตราจ่าย | **0.5 Point** ต่อรายการ (50% ของ Free Schedule) |
| ตัวอย่าง | เบิกมา 100 บาท → จ่ายจริง 50 บาท |

### กรณีที่ 2: Walk-in ในจังหวัดเดียวกัน (เบิกภายในจังหวัด)

| รายการ | รายละเอียด |
|--------|-----------|
| เงื่อนไข | ผู้รับบริการมีสิทธิ์ในจังหวัดเดียวกัน อยู่คนละอำเภอหรือคนละหน่วย |
| Project Code | Walk-in (ใช้เหมือนกัน) |
| ยอดจาก สปสช. | **0 บาท** (ระบบจ่าย 0 โดยอัตโนมัติ) |
| วิธีเรียกเก็บ | ออกรายงานจากระบบ → นำไปเรียกเก็บกันตามหลักเกณฑ์จังหวัด |

## ข้อกำหนดการส่งข้อมูล (กรณีในจังหวัด)

สำหรับหน่วยบริการสังกัดกระทรวงสาธารณสุขที่ส่งผ่าน M-Financial Data Hub:

- ใน **แฟม 16** ของมาตรฐาน 43 แฟม ต้องระบุ **UUC = 2** (ไม่ใช้สิทธิ์ UC)
- ระบบจะออกรายงานสรุปค่าใช้จ่าย แต่จ่ายเป็น **0 บาท**
- หน่วยบริการนำรายงานไปเรียกเก็บกันในจังหวัด

## การเบิกต่อเนื่อง Walk-in โดยไม่ย้ายสิทธิ์

ผู้ป่วยที่มารับบริการต่อเนื่องนานเป็นปีโดยไม่ย้ายสิทธิ์:
- **สามารถเบิกได้** ตลอด แต่ได้เพียง **0.5 Point**
- แนะนำให้ผู้ป่วยย้ายสิทธิ์มาอยู่หน่วยปัจจุบัน — สามารถย้ายได้ถึง **4 ครั้งต่อปี** และเกิดสิทธิ์ทันที

## ข้อผิดพลาดที่พบบ่อย

| ข้อผิดพลาด | สาเหตุ | วิธีแก้ |
|-----------|--------|--------|
| ส่ง Walk-in แล้วได้ 0 บาท (ในจังหวัด) | ระบบตรวจพบว่าสิทธิ์อยู่ในจังหวัดเดียวกัน | ตั้งค่า UUC=2, ออกรายงานไปเรียกเก็บในจังหวัด |
| ส่ง Walk-in แล้วได้ 0 บาท (ข้ามจังหวัด) | ลืมกำหนด Project Code Walk-in ให้ถูกต้อง | ตรวจสอบสิทธิ์ว่าข้ามจังหวัดจริง |
