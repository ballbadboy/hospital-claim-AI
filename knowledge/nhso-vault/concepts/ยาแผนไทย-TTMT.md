# ยาแผนไทยและรหัสกลาง TTMT

## Definition

รหัสกลาง TTMT (Thai Traditional Medicine Terminology) คือรหัสมาตรฐานกลางสำหรับยาแผนไทยและยาสมุนไพร ที่พัฒนาโดยทีม สมพสท. (สมาคมสมุนไพรไทย) ร่วมกับกรมแพทย์แผนไทย เพื่อให้ทุกโรงพยาบาลส่งข้อมูลการเบิกจ่ายยาแผนไทยในรูปแบบมาตรฐานเดียวกัน

## How It Works

### ปัญหาเดิม
แต่ละโรงพยาบาลตั้งรหัสยาแผนไทยของตัวเอง เช่น ฟ้าทลายโจรชนิดเม็ดของโรงพยาบาล ก. อาจเป็นรหัส A แต่ของโรงพยาบาล ข. เป็นรหัส B ทำให้ สปสช. ไม่สามารถระบุได้ว่าโรงพยาบาลเบิกยาตัวไหน

### วิธีการ MAP Code
1. โรงพยาบาลต้องนำรหัสยาของตัวเองไป **MAP** กับรหัสกลาง TTMT ใน Drug Catalog
2. ส่งข้อมูลที่ MAP แล้วให้ สปสช.
3. สปสช. จะรู้จักรหัสยาและสามารถจ่ายชดเชยได้

### ผลกระทบหากไม่ MAP
- ข้อมูลที่ส่งเข้ามาจะมีสถานะ "หารหัสไม่เจอ" (TTMT not found)
- สปสช. ไม่สามารถจ่ายชดเชยได้
- ตัวอย่าง: ข้อมูลที่ยังค้างชำระอยู่ประมาณ 190,000 เรคคอร์ด คลีนแล้วเหลือ 10,000 ที่ยังจ่ายไม่ได้ เพราะหา TTMT ไม่เจอ

### งบประมาณปีงบ 68
- รวม 1,500 ล้านบาท แบ่งเป็น 2 กลุ่ม:
  - **หัตถการ:** นวด ประคบ ฝังเข็ม
  - **ยาแผนไทย:** ยาสมุนไพร 32 รายการที่มีประสิทธิภาพทดแทนยาแผนปัจจุบัน + ยาทั่วไปในบัญชีรายการ
- จ่ายเป็น **พอย์** (points) ทั้งหมด

### รอบการจ่าย
- **งวดที่ 1:** ผลงาน Q1–Q2 ปีงบประมาณ (ตุลาคม–มีนาคม)
- **งวดที่ 2:** ผลงาน Q3–Q4 (เมษายน–กันยายน) จ่ายต้นตุลาคมปีถัดไป

### การส่งข้อมูล
- **ภูมิภาค:** ใช้ข้อมูลจาก OPP Individual
- **กทม.:** ใช้โปรแกรม EKM บันทึก

### เงื่อนไขการเบิก ICD
- ไม่จำเป็นต้องใส่รหัส ICD โรคร่วม (co-morbidity) ในการเบิกยาสมุนไพร
- มีรหัสโรคหลักก็พอ
