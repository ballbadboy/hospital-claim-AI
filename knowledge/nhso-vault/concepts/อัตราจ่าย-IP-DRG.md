# อัตราจ่าย IP (ผู้ป่วยใน) / DRG ปีงบ 2569

## Definition

การจ่ายค่าใช้จ่ายสำหรับผู้ป่วยในใช้ระบบ DRG (Diagnosis Related Group) เวอร์ชัน 6.3 (TDRGs Version 6.3) เป็นมาตรฐานการจัดกลุ่มโรคและคำนวณค่า RW (Relative Weight) สำหรับการชดเชยค่าบริการ

## อัตราจ่ายปีงบ 2569

| ประเภทบริการ | อัตราจ่าย | หมายเหตุ |
|-------------|-----------|---------|
| บริการในเขต | **8,350 บาท/RW** | TDRGs v6.3 |
| บริการนอกเขต/ข้ามเขต | **9,600 บาท/RW** | TDRGs v6.3 |

- ค่า RW คำนวณโดยไม่ deduct DG และไม่มียาเคมีบำบัด

## ประเภทบริการภายใต้ IP

### IP ทั่วไป
- จ่ายตามค่า RW ตามอัตราข้างต้น
- ผู้มีสิทธิ์ UC ทุกราย, ผู้ป่วยประกันสังคม (D1)

### ODS (One-Day Surgery / ผ่าตัดวันเดียวกลับ)

| ปี 2568 | ปี 2569 (ใหม่) |
|---------|--------------|
| TDRGs Version 6 | **TDRGs Version 6.3** |
| ในเขต: อัตราเดิม | **8,350 บาท/RW** |
| นอกเขต: อัตราเดิม | **9,600 บาท/RW** |

- รายการโรค/หัตถการตามแนบท้ายประกาศ
- โปรแกรม: e-Claim

### MIS (Minimally Invasive Surgery / ผ่าตัดเล็ก)
- จ่ายตาม SSRW
- ในเขต: 8,350 บาท | ข้ามเขต: 9,600 บาท
- **ไม่มีการปรับลดค่าแรง** (ทั้งในเขตและนอกเขต)
- รายการตามแนบท้ายประกาศ

### CRRT (Continuous Renal Replacement Therapy)

เงื่อนไขหน่วยบริการ:
- บันทึกศักยภาพ **รหัส H50** ใน CP PP
- มีอายุรแพทย์โรคไต
- มีศักยภาพเฉพาะทางผู้ป่วยวิกฤต

เกณฑ์ข้อบ่งชี้ผู้ป่วย:
1. ภาวะสมองบวมหรือมีความเสี่ยงสมองตาย
2. ความผิดปกติ Metabolic ต่อเนื่องที่แก้ไขด้วยฟอกเลือดชั่วคราวไม่ได้
3. ปริมาณสารน้ำในร่างกายสูงเกินเกณฑ์

### STEM (Fast Track)

| ยา | ปี 2568 | ปี 2569 |
|----|---------|---------|
| Streptokinase | 10,000 บาท | **5,000 บาท** ⬇️ |
| Tenecteplase (TNK-TPA) | 22,660 บาท | **22,660 บาท** (คงเดิม) |

## เกณฑ์การจัดทำข้อมูล IP

1. รหัสวินิจฉัย: ICD-10 (WHO 2016) + ICD-10-TM
2. รหัสหัตถการ: ICD9-CM 2016
3. รับไว้ รพ. < 2 ชม. → บันทึกเป็น OP (ยกเว้น Dead/Refer/Escape/Against Advice)
4. อยู่ รพ. > 24 ชม. → Leave day → หักกลบออกจาก Admit ครั้งนั้น
5. ต้องมี Catalogue (Drug catalog + Service catalog)
6. ให้บริการ OP แล้วรับเป็น IP ภายใน 24 ชม. → เบิกเป็น IP
