# กองทุน Long-term Care ปีงบ 2569

## Definition

กองทุนการดูแลระยะยาวด้านสาธารณสุขสำหรับผู้มีภาวะพึ่งพิง (Long-term Care Fund) เป็นส่วนหนึ่งของการบริหารบริการสาธารณสุขร่วมกับ อปท. จัดสรรงบประมาณผ่านกองทุนท้องถิ่น เพื่อให้ผู้ที่ต้องพึ่งพิงผู้อื่นในชุมชนได้รับการดูแลใกล้บ้านและใกล้ใจ

## วงเงินและอัตราปี 2569

| รายการ | ปีก่อน | ปี 2569 |
|--------|--------|---------|
| วงเงินรวม | ~3,000 ล้านบาท | **5,114 ล้านบาท** (~2 เท่า) |
| อัตราต่อหัว | 6,000 บาท/คน | **1,442 บาท/หัวประชากร** |

## อาสาสมัครนักบริบาลชุมชน (Care Giver)

ปี 2569 ปรับโครงสร้างค่าตอบแทน **แบ่งตามชั่วโมงอบรม**:

| ชั่วโมงอบรม | ค่าตอบแทน/เดือน |
|-------------|----------------|
| 70 ชั่วโมง (ระดับต้น) | **5,000 บาท** |
| 120 ชั่วโมง (ระดับกลาง) | **6,000 บาท** |
| 420 ชั่วโมง (ระดับสูง / นักบริบาล) | **9,000 บาท** |

เดิมใช้อัตราเดิม 1,500–500 บาท/เดือน (ค่าอ้อม)

## เป้าหมาย Care Giver

- เร่งรัด **18,000 Care Giver** ทั่วประเทศ (งบกระตุ้นเศรษฐกิจ)
- อัปเกรดจาก อสม. เดิม → **อสม.นักบริบาลชุมชน**
- บูรณาการกับ MOU สปสช. + กรมอนามัย + กรมสบส.

## MOU 3 หน่วยงาน

ลงนามในการประชุมวันนี้ (15 ตุลาคม 2568):
- **สปสช.** — จัดสรรงบประมาณ
- **กรมอนามัย** — สนับสนุนวิชาการ, พัฒนาศักยภาพ, แลกเปลี่ยนข้อมูล
- **กรมสนับสนุนบริการสุขภาพ (กรมสบส.)** — พัฒนาและส่งเสริม อสม. ยุคใหม่

## How It Works

1. สปสช. จัดสรรงบ 1,442 บาท/หัวให้กองทุนท้องถิ่น
2. อปท. ใช้งบ + สมทบ จ้าง/สนับสนุน Care Giver ในพื้นที่
3. Care Giver (อสม.ที่ผ่านอบรม) ไปดูแลผู้มีภาวะพึ่งพิงที่บ้าน
4. ระบบข้อมูล: กรมอนามัยช่วยแลกเปลี่ยนข้อมูลผู้พึ่งพิง
5. กองทุนสามารถเกลี่ยข้ามกองได้ (เช่น ค่าผ้าอ้อมไม่พอ นำเงินจังหวัดมาเติม)
