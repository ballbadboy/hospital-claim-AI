# อัตราจ่ายผู้ป่วยใน (IP) ปีงบประมาณ 2569

## Definition

ปีงบประมาณ 2569 สปสช. กำหนดอัตราจ่ายเบื้องต้นสำหรับผู้ป่วยใน (IP) ที่ **8,350 บาท/1 Adjusted RW** สำหรับทุกหน่วยบริการทุกสังกัด โดยปรากฏมีข่าวว่าจ่ายเพียง 3,505 บาท ซึ่งเป็นความเข้าใจผิดเกี่ยวกับการบริหารจัดการภายในกระทรวงสาธารณสุข

## อัตราจ่ายตามสังกัด

| สังกัดหน่วยบริการ | อัตราจ่าย สปสช. (คอลัมน์ 7 Statement) | หักเงินเดือน |
|---|---|---|
| สำนักงานปลัดกระทรวง สธ. (สป.สธ.) | **8,350 บาท** (จ่ายเต็ม) | หักในภาพรวมระดับประเทศ → ช่องเงินเดือน = 0 |
| กรมต่างๆ ในกระทรวง สธ. (เช่น รพ.ราชวิถี) | **8,350 บาท** | หักตามเกณฑ์กรม |
| นอกสังกัด สธ. (กลาโหม, ยูฮอส ฯลฯ) | **8,350 บาท** | หักตามเกณฑ์สังกัดนั้น |
| ภาคเอกชน | **8,350 บาท** | ไม่หัก (จ่ายค่าแรงเอง) |
| บริการผู้ป่วยข้ามเขต (ทุกสังกัด) | **9,600 บาท** (อัตรากลาง) | ขึ้นอยู่กับสังกัด |

## ที่มาของ 3,505 บาท

อัตรา **3,505 บาท** ที่ปรากฏใน Statement ของหน่วย สป.สธ. ไม่ใช่อัตราที่ สปสช. ลดให้ แต่เกิดจากกลไกบริหารภายในกระทรวงดังนี้:

```
สปสช. จ่าย 8,350 บาท/AdjRW
    ↓
กระทรวงสธ. หักค่าแรงภาพรวมระดับประเทศ (ตามมติ 19 ธ.ค. 68)
    ↓
เหลือที่หน่วยรับ = 3,505 บาท (ช่องเงินเดือน = 0)
    ↓
ปลายปี กระทรวงเกลี่ยเงิน 6,326 ล้านบาท เติมให้ตามเกณฑ์
```

## วงเงินที่กระทรวงสธ. กันไว้เติมหน่วย สป.สธ.

| ประเภทบริการ | วงเงิน (ล้านบาท) |
|---|---|
| ผู้ป่วยนอก (OP) | 4,600 |
| ผู้ป่วยใน (IP) | 6,326 |
| สร้างเสริมสุขภาพ (PP) | ~1,000 |
| **รวม** | **~12,000** |

การบริหารวงเงิน 12,000 ล้านแบ่ง 3 ระดับ:
- **ระดับหน่วย (จ่ายตรง):** 8,400 ล้านบาท
- **ระดับเขต (สขจ. บริหาร):** 3,000 ล้านบาท (กระจาย 12 เขต)
- **ระดับประเทศ:** 600 ล้านบาท (สภาพคล่อง + นโยบาย สธ.)

## กลไกการตัดสินใจ

| วันที่ | เหตุการณ์ |
|---|---|
| 27 ตุลาคม 2568 | ประชุมร่วม สปสช.–ตัวแทน รพ.ศ./รพ.ท./สบทช. เสนอหักค่าแรงภาพรวม |
| พฤศจิกายน 2568 | บอร์ดเห็นชอบครั้งที่ 1 |
| 1 ธันวาคม 2568 | บอร์ดเห็นชอบครั้งที่ 2 — อนุมัติจัดสรร OP/PP/IP สป.สธ. |
| 19 ธันวาคม 2568 | อนุกรรมการการเงินการคลังเห็นชอบรายละเอียดการกันเงิน 12,000 ล้าน |

## เงินเสริม IP เพิ่มเติม 5,450 ล้านบาท

นอกจากวงเงิน 12,000 ล้านข้างต้น บอร์ดยังเห็นชอบให้กันงบจากก้อนอื่น **5,450 ล้านบาท** มาเสริม IP สำหรับหน่วย สป.สธ. โดยเฉพาะ เพื่อป้องกันรายรับต่ำกว่าปีที่แล้ว

## การตรวจสอบ Statement

หน่วยบริการทุกสังกัดดาวน์โหลด Statement ได้ผ่านระบบของ สปสช.
- **คอลัมน์ 7** = อัตราจ่าย (บาท/AdjRW)
- หน่วย สป.สธ.: คอลัมน์เงินเดือน = 0 (หักรวมระดับประเทศแล้ว)
- หน่วยภาคเอกชน: คอลัมน์เงินเดือน = 0 (จ่ายค่าแรงเอง)

## การติดตาม

สปสช. และกระทรวง สธ. ตั้งคณะทำงานร่วม 2 คณะเพื่อมอนิเตอร์:
1. คณะมอนิเตอร์งบเหมาจ่าย
2. คณะมอนิเตอร์งบนอกเหมาจ่าย

และกระจายคณะทำงานลงในเขตพื้นที่ทุกเขต ประชุมร่วมกันทุกสัปดาห์
