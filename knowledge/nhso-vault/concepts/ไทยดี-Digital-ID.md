# ไทยดี (ThaID / Digital ID)

## Definition

ไทยดี (ThaID) คือแอปพลิเคชันพิสูจน์และยืนยันตัวตนดิจิทัลของภาครัฐไทย พัฒนาโดยกรมการปกครอง กระทรวงมหาดไทย ใช้แทนบัตรประชาชนทางกายภาพในการยืนยันตัวตน สปสช. ใช้ไทยดีเชื่อมต่อกับระบบลงทะเบียนหน่วยบริการ (One Service) และการพิสูจน์ตัวตนผู้รับบริการ

## บทบาทในระบบ สปสช.

| การใช้งาน | รายละเอียด |
|-----------|-----------|
| ลงทะเบียนหน่วยบริการ (One Service) | ผู้บริหารหน่วยบริการต้องพิสูจน์ตัวตนผ่านไทยดีในการสมัครและแก้ไขข้อมูล |
| พิสูจน์ตัวตนผู้รับบริการ | ผู้รับบริการสามารถใช้ไทยดีแทนบัตรประชาชนในการเข้ารับบริการบางกรณี |
| ปิดสิทธิ์ | สปสช. รับรองไทยดีเป็นหนึ่งในวิธียืนยันตัวตนที่ถูกต้อง |

## ปัญหาที่พบบ่อย: เชื่อมต่อ One Service ไม่ได้

**อาการ:** กดลิงก์เชื่อมต่อไทยดีใน One Service แล้วขึ้น error ตลอด

**วิธีแก้:**
1. เปิดแอป **ไทยดี** บนมือถือก่อน
2. ทำการ **ล็อกอินแอปให้สำเร็จสมบูรณ์** (พิสูจน์ตัวตนผ่านแอปให้ครบ)
3. หลังจากล็อกอินแอปสำเร็จแล้ว จึงกลับมาเปิด **One Service** บนเว็บแอปพลิเคชัน
4. เลือกเมนูไทยดีอีกครั้ง → ระบบจะเชื่อมต่อได้

> สาเหตุ: ระบบ One Service ของ สปสช. เชื่อมต่อไปยัง API ของไทยดี ซึ่งต้องการ session ที่ active ในแอปก่อน

## การใช้แทนบัตรประชาชนในชีวิตประจำวัน

ณ เมษายน 2568 ไทยดียังไม่ได้รับการยอมรับจากทุกหน่วยงาน:
- ใช้ได้: คลินิก/โรงพยาบาลที่รองรับ, ระบบออนไลน์ภาครัฐ
- ยังติดปัญหา: หน่วยงานราชการบางแห่ง (เช่น ทำเนียบรัฐบาล) ยังต้องการบัตรประชาชนตัวจริง

## ข้อกำหนดการใช้บัตรทอง

ผู้รับบริการที่ไปคลินิกบัตรทองต้องยืนยันตัวตน:
- บัตรประชาชน (Smart Card) → เสียบเครื่องอ่านบัตร
- แอปไทยดี → สแกน QR หรือ NFC ตามระบบที่คลินิกรองรับ

> **สำคัญ:** หากไม่มีทั้งบัตรประชาชนและไทยดี → คลินิกไม่สามารถปิดสิทธิ์และเบิกจ่ายได้ตามมาตรฐาน
