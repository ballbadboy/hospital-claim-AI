# DS032 — เกินโควต้าบริการต่อวันต่อผู้ประกอบวิชาชีพ

## Definition

**DS032** คือรหัสปฏิเสธการจ่ายชดเชย เกิดขึ้นเมื่อหน่วยบริการนวัตกรรมให้บริการเกินจำนวนสูงสุดที่กำหนดต่อผู้ประกอบวิชาชีพ 1 ท่านต่อวัน สปสช. กำหนดโควต้าโดยทำงานร่วมกับสภาวิชาชีพแต่ละสาขา

## เหตุผลที่มีโควต้าต่อวัน

กฎหมายหลักประกันสุขภาพกำหนดให้บริการต้องมี **คุณภาพและมาตรฐาน** ดังนั้น สปสช. และสภาวิชาชีพจึงกำหนดจำนวนผู้ป่วยสูงสุดที่ผู้ประกอบวิชาชีพ 1 ท่านสามารถให้บริการได้อย่างมีคุณภาพ

## โควต้าตามวิชาชีพ

| วิชาชีพ | โควต้าต่อวันต่อผู้ประกอบวิชาชีพ 1 ท่าน |
|---------|----------------------------------------|
| กายภาพบำบัด | **13 คน/วัน** |
| การพยาบาล | **90 คน/วัน** (ตัวอย่างที่กล่าวถึง) |
| วิชาชีพอื่น | ตามที่สภาวิชาชีพกำหนดร่วมกับ สปสช. |

> **หมายเหตุ:** โควต้าคำนวณจากจำนวนผู้ประกอบวิชาชีพที่ขึ้นทะเบียนไว้ในหน่วย เช่น มีนักกายภาพบำบัด 2 ท่าน → ให้บริการได้ 26 คน/วัน

## ผลกระทบและการจ่ายชดเชย

- รายการที่อยู่ภายในโควต้า → จ่ายชดเชยตามปกติ
- รายการที่เกินโควต้า → **ติด DS032** → ไม่ชดเชย
- **ไม่มีระบบอุทธรณ์ DS032** ผ่าน On Screen Review (ณ พฤษภาคม 2568)

## วิธีตรวจสอบว่าติด DS032

1. เข้า Seamless for DMS (SATE for DMIS)
2. เลือก **รายงาน REP Summary** → ค้นหาตามรหัสหน่วยบริการ
3. ดูงวด IPEP ที่ต้องการ
4. ดูคอลัมน์ **ไม่ชดเชย** — รายการที่ติด DS032 จะระบุเหตุผล "ให้บริการเกิน X คนต่อวัน"

## แนวทางปฏิบัติ

1. **ตรวจสอบจำนวนผู้ประกอบวิชาชีพที่ขึ้นทะเบียน** ให้ครบถ้วนตามจำนวนจริง
2. **บริหารนัดหมายผู้ป่วย** ให้ไม่เกินโควต้า
3. ผู้ป่วยที่เกินโควต้า → แนะนำให้ไปรับบริการที่หน่วยบริการประจำ (โรงพยาบาล/รพสต.)
4. หากยืนยันว่าเคสมีความจำเป็นทางการแพทย์จริง → รอให้ สปสช. เปิดระบบอุทธรณ์

## ความแตกต่างจาก D032

DS032 ใช้ในระบบนวัตกรรม (Seamless/SATE for DMIS) ส่วน D032 อาจปรากฏในระบบอื่น — ตรวจสอบระบบที่ใช้เบิกให้ถูกต้อง
