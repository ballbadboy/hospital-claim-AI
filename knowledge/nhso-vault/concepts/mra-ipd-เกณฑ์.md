# MRA — เกณฑ์ตรวจประเมินเวชระเบียนผู้ป่วยใน (IPD)

## โครงสร้าง

IPD ประเมิน **12 หัวข้อ (Content)** แต่ละหัวข้อมี **7–9 เกณฑ์** คะแนนเกณฑ์ละ 1 คะแนน

---

## 1. Discharge Summary: Diagnosis, Operation
**เนื้อหาข้อมูลทางสถิติและการแพทย์** — 9 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | สรุป **Principal diagnosis** เป็น clinical term (ไม่ใช่ ICD code) ถูกต้อง มีเพียงโรคเดียว |
| 2 | สรุป **Comorbidity/Complication** เป็น clinical term สอดคล้องกับ chart (ใส่ NA ถ้าไม่มี) |
| 3 | สรุป **Procedure/Operation** ถูกต้องครบถ้วน (NA ถ้าไม่มี) |
| 4 | บันทึกวันเวลาเริ่ม–สิ้นสุดหัตถการใน OR (NA ถ้าไม่มี) |
| 5 | ไม่ใช้ตัวย่อ (ยกเว้น WHO ICD-10/9CM) ลายมืออ่านออกได้ |
| 6 | มี **Clinical summary** ครบ 5 ข้อ: สาเหตุแรกรับ, investigation สำคัญ, การรักษาและผล, แผนหลังจำหน่าย, Home medication |
| 7 | สรุปสาเหตุการตายสอดคล้องกับ chart (NA ถ้าไม่เสียชีวิต) |
| 8 | สรุป **Discharge status และ Discharge type** ถูกต้อง (ถ้า "by transfer" ต้องระบุชื่อ รพ.) |
| 9 | มีลายมือแพทย์ผู้รับผิดชอบ พร้อม **ชื่อ นามสกุล เลขใบประกอบวิชาชีพ** |

> ⚠️ **Claim impact:** ถ้า Principal diagnosis ผิด → DRG grouping ผิด → deny หรือได้ RW ต่ำกว่าความเป็นจริง

---

## 2. Discharge Summary: Others
**เนื้อหาอื่นๆ และส่วนประกอบ** — 7 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | ชื่อ นามสกุล เพศ อายุ (หรือวันเกิด) ถูกต้องครบ |
| 2 | เลขประจำตัวประชาชน / เลขต่างด้าว / หนังสือเดินทาง |
| 3 | ที่อยู่ปัจจุบันของผู้ป่วย |
| 4 | ชื่อ รพ., HN, AN ถูกต้องตรงกันทุกแห่งใน chart |
| 5 | วันเวลา Admit และ Discharge ถูกต้อง |
| 6 | LOS และ Leave days ถูกต้อง |
| 7 | ชื่อผู้ให้รหัสโรค (coder) และผู้ให้รหัสหัตถการ |

---

## 3. Informed Consent
**บันทึกการรับทราบและยินยอม** — 9 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | ชื่อ นามสกุล ผู้ป่วยถูกต้องชัดเจน |
| 2 | ลายมือชื่อผู้ให้คำอธิบาย (ชื่อ นามสกุล ตำแหน่ง) |
| 3 | ระบุชื่อหัตถการ/การรักษาที่ยินยอม |
| 4 | มีการอธิบายวัตถุประสงค์ ความเสี่ยง ผลแทรกซ้อน |
| 5 | มีทางเลือกอื่นในการรักษา |
| 6 | ผู้ป่วย/ญาติมีโอกาสซักถาม |
| 7 | ลายมือชื่อผู้ป่วย หรือผู้ปกครอง/ญาติ พร้อมวันเวลา |
| 8 | ลายมือชื่อพยาน |
| 9 | บันทึกก่อนทำหัตถการ (ไม่ใช่หลัง) |

---

## 4. History Taking & Physical Examination
**การซักประวัติและตรวจร่างกายแรกรับ** — 9 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | Chief complaint ชัดเจน |
| 2 | Present illness ครอบคลุมอาการปัจจุบัน |
| 3 | Past history (โรคประจำตัว ยา แพ้ยา) |
| 4 | Family/Social history (ถ้าเกี่ยวข้อง) |
| 5 | Physical examination ครบระบบ |
| 6 | Vital signs แรกรับ |
| 7 | Provisional/Working diagnosis |
| 8 | Investigation plan |
| 9 | ลายมือแพทย์ ชื่อ นามสกุล เลขใบประกอบวิชาชีพ วันเวลา |

---

## 5. Admission Note (Doctor's Note)
**บันทึกการรับผู้ป่วยเข้าโรงพยาบาล** — 9 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | วัน เดือน ปี เวลา |
| 2 | อาการสำคัญ (Chief complaint) |
| 3 | ประวัติปัจจุบัน (History of present illness) |
| 4 | ประวัติเดิม (Past history) |
| 5 | การตรวจร่างกาย (Physical examination) |
| 6 | Provisional diagnosis |
| 7 | แผนการตรวจเพิ่มเติม Investigation plan |
| 8 | แผนการรักษา (Treatment plan) |
| 9 | ลายมือแพทย์ ชื่อ นามสกุล เลขใบประกอบวิชาชีพ |

---

## 6. Progress Notes & Doctor's Order Sheet
**บันทึกความก้าวหน้าและคำสั่งการรักษา** — 9 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | ลงวันเดือนปีและเวลาทุกครั้ง |
| 2 | บันทึก**ทุกวัน**ใน 3 วันแรก |
| 3 | เนื้อหาครอบคลุม **SOAP** ใน 3 วันแรก |
| 4 | บันทึกทุกครั้งที่มีการเปลี่ยนแปลงอาการ การรักษา ยา invasive procedure หรือเปลี่ยนแพทย์ |
| 5 | เนื้อหาครอบคลุม SOAP ทุกครั้งที่มีการเปลี่ยนแปลง |
| 6 | บันทึกการแปลผล investigation สำคัญ พร้อม diagnosis และแผนการรักษา |
| 7 | บันทึกตรงตำแหน่งที่ รพ. กำหนด |
| 8 | ลายมือที่อ่านออกได้ + ลายมือชื่อแพทย์ระบุตัวได้ |
| 9 | Doctor's order: ลงวันเวลา + ลงนามกำกับ **ทุกครั้ง** (รวม order ทาง LINE/App) |

> ⚠️ **Claim impact:** Progress note ไม่ครบ 3 วันแรก หรือ SOAP ไม่ครบ → MRA ไม่ผ่าน → อาจถูก audit deny

---

## 7. Consultation Record
**บันทึกการปรึกษาระหว่างแผนก** — 9 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | วันเวลา ความเร่งด่วน หน่วยงานที่ขอปรึกษา |
| 2 | ระบุปัญหาที่ต้องการปรึกษาชัดเจน |
| 3 | ประวัติการตรวจร่างกายและการรักษาโดยย่อ |
| 4 | ลายมือชื่อแพทย์ผู้ขอปรึกษา (ชื่อ นามสกุล เลขใบฯ) |
| 5 | ผลการตรวจประเมินเพิ่มเติมและคำวินิจฉัยของผู้รับปรึกษา |
| 6 | ความเห็น/แผนการรักษา/คำแนะนำ |
| 7 | วันเวลาที่ผู้รับปรึกษามาตรวจ |
| 8 | ลายมือชื่อแพทย์ผู้ให้คำปรึกษา (ชื่อ นามสกุล เลขใบฯ) |
| 9 | บันทึกตรงตำแหน่งที่ รพ. กำหนด |

> หมายเหตุ: รพ.ชุมชนที่ไม่มีการแบ่งแผนก → ไม่ต้องประเมินหัวข้อนี้

---

## 8. Anesthetic Record
**บันทึกทางวิสัญญีวิทยา** — 9 เกณฑ์

ประเมินเฉพาะ: General anesthesia, Spinal block, Epidural block, Regional block  
**ไม่ประเมิน:** Local block

| # | เกณฑ์ |
|---|-------|
| 1 | Status ผู้ป่วยก่อนให้ยาและวิธีให้ยาระงับความรู้สึก |
| 2 | บันทึกโรคก่อนผ่าตัด (ตรงกับ Dx ของแพทย์) |
| 3 | ชนิดและชื่อการผ่าตัด (ตรงกับแพทย์) |
| 4 | Pre-anesthetic evaluation โดยทีมวิสัญญี (ระบุ history การได้รับยาสลบก่อนหน้า) |
| 5 | Vital signs + monitoring ระหว่างดมยาทุก 5 นาที |
| 6 | Intake, Output, Blood loss, Total intake, Total output |
| 7 | ดูแลผู้ป่วยหลังผ่าตัด 1 ชั่วโมง (Recovery room) |
| 8 | บันทึกภาวะแทรกซ้อนระหว่าง/หลังดมยา (ถ้ามี) |
| 9 | ลายมือชื่อวิสัญญีแพทย์ ชื่อ นามสกุล เลขใบฯ |

---

## 9. Operative Note
**บันทึกการผ่าตัด** — 9 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | วันเดือนปีและเวลาเริ่ม–สิ้นสุดผ่าตัด |
| 2 | Pre-operative diagnosis |
| 3 | Post-operative diagnosis |
| 4 | ชื่อหัตถการ/การผ่าตัดที่ทำ (ครบถ้วน) |
| 5 | ชื่อศัลยแพทย์ผู้ผ่าตัดและผู้ช่วย |
| 6 | Findings ระหว่างผ่าตัด |
| 7 | รายละเอียดขั้นตอนการผ่าตัด |
| 8 | Specimen ส่งตรวจ (ถ้ามี) |
| 9 | ลายมือชื่อแพทย์ผู้ผ่าตัด ชื่อ นามสกุล เลขใบฯ |

---

## 10. Nursing Note
**บันทึกทางการพยาบาล** — 9 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | วันเดือนปีและเวลาทุกครั้งที่บันทึก |
| 2 | Assessment แรกรับ (Nursing assessment) |
| 3 | Nursing diagnosis / ปัญหาการพยาบาล |
| 4 | กิจกรรมการพยาบาลสอดคล้องกับปัญหา |
| 5 | การตอบสนองของผู้ป่วยต่อการพยาบาล |
| 6 | Vital signs บันทึกสม่ำเสมอตามคำสั่งแพทย์ |
| 7 | บันทึกการให้ยาครบถ้วน (ชื่อยา dose route เวลา) |
| 8 | ลายมือที่อ่านออกได้ + ลายมือชื่อพยาบาลระบุตัวได้ |
| 9 | Discharge planning / การสอน/แนะนำผู้ป่วย |

---

## 11. Rehabilitation Record
**บันทึกการฟื้นฟูสมรรถภาพ** — 9 เกณฑ์  
*(NA ถ้าไม่มีการทำ rehabilitation)*

| # | เกณฑ์ |
|---|-------|
| 1 | วันเดือนปีและเวลา |
| 2 | Functional assessment แรกรับ |
| 3 | เป้าหมายการฟื้นฟู |
| 4 | แผนการฟื้นฟู |
| 5 | กิจกรรมการฟื้นฟูที่ทำ |
| 6 | การตอบสนองของผู้ป่วย |
| 7 | การประเมินผลซ้ำ |
| 8 | Home program / คำแนะนำเมื่อกลับบ้าน |
| 9 | ลายมือชื่อนักกายภาพ/นักกิจกรรมบำบัด ระบุตัวได้ |

---

## 12. Discharge Planning
**แผนการจำหน่ายผู้ป่วย** — 7 เกณฑ์

| # | เกณฑ์ |
|---|-------|
| 1 | ประเมิน discharge needs ตั้งแต่แรกรับ |
| 2 | แผนการดูแลต่อเนื่อง (Home care/Refer) |
| 3 | การสอนผู้ป่วย/ญาติ (ยา, อาหาร, กิจกรรม) |
| 4 | นัด Follow up |
| 5 | สัญญาณอันตรายที่ต้องกลับ รพ. |
| 6 | ผู้ป่วยและญาติเข้าใจและยอมรับแผน |
| 7 | ลายมือชื่อผู้รับผิดชอบ discharge planning |

---

## สรุป — จุดที่เชื่อมกับ Claim

| หัวข้อ MRA | ผลต่อ Claim |
|-----------|-----------|
| Discharge Summary #1 | PDx/SDx ผิด → DRG ผิด → RW ต่ำ → deny |
| Discharge Summary #2 | LOS ผิด → claim ผิดช่วงเวลา |
| Informed Consent | ไม่มี → deny ค่าหัตถการ |
| Progress Notes | ไม่ครบ → audit D001 deny |
| Operative Note | ไม่ครบ → deny ค่าผ่าตัด |
| Anesthetic Record | ไม่มี → deny ค่าดมยา |
