# การยืนยันตัวตนสำหรับกลุ่มเปราะบาง

## Definition

**กลุ่มเปราะบาง** ในบริบทของระบบ New Authentication ของ สปสช. หมายถึงกลุ่มผู้รับบริการที่ **ไม่สามารถสแกนใบหน้า (FVS) ได้** ซึ่งส่วนใหญ่คือ:

- เด็กอายุ **ต่ำกว่า 7 ปี** — ไม่มีบัตรประชาชน ไม่มีข้อมูลในฐานทะเบียนราษฎร
- เด็กอายุ **7–14 ปี** — มีข้อมูลแต่อาจสแกนไม่ผ่าน
- **ผู้สูงอายุ** (โดยเฉพาะผู้ที่ไม่ได้ทำบัตรประชาชนใหม่แล้ว)
- **ผู้พิการ** ที่มีปัญหาด้านใบหน้า

## กระบวนการ — Step by Step

### ขั้นตอนที่ 1: ลองสแกนใบหน้าก่อน (บังคับ)
- **ลอง FVS อย่างน้อย 2 ครั้ง** ทุกกรณี ยกเว้นเด็กต่ำกว่า 7 ปีที่ไม่มีข้อมูลแน่นอน
- การสแกน 2 ครั้งจะถูก **เก็บ log** ไว้ในระบบ สปสช. ว่า "ไม่สามารถสแกนได้จริง"
- Log นี้สำคัญมาก — ถ้าไม่มี log 2 ครั้ง อาจถูกปฏิเสธการจ่ายเงิน ([[concepts/N019-ข้อผิดพลาด]])

### ขั้นตอนที่ 2: ถ้าสแกนไม่ผ่าน — เลือกวิธีสำรอง

**วิธีที่ 2: จองผ่าน Line OA**
- ใช้ได้ทั้งกลุ่มที่มีและไม่มีบัตรประชาชน

**วิธีที่ 3: เสียบ Smart Card + ยืนยันตัวตนแทน**
เข้าระบบผ่าน OSS → endpoint ปิดสิทธิ์ → เลือก "ยืนยันตัวตนแทนผู้เข้ารับบริการ"

**ขั้นตอนย่อยในวิธีที่ 3:**

| ลำดับ | การกระทำ |
|-------|----------|
| 1 | ใส่เลขบัตรประชาชน **ของผู้รับบริการ** → กดค้นหา |
| 2 | เลือก **เหตุผล** ที่ไม่สามารถยืนยันด้วยตัวเอง (เด็ก / ผู้สูงอายุ / ผู้มีปัญหาด้านใบหน้า) |
| 3 | เลือก **ความสัมพันธ์** ของผู้ยืนยันแทน (พ่อ / แม่ / ปู่ / ย่า / ตา / ยาย / ผู้ดูแล) |
| 4 | เสียบ Smart Card ของ **ผู้รับบริการก่อน** |
| 5 | ถ้าผู้รับบริการไม่มีบัตร → ใช้บัตรผู้ดูแล/ปกครอง |
| 6 | กรอกข้อมูลปิดสิทธิ์ตามปกติ (ยอดเงิน ฯลฯ) |
| 7 | ติ๊กยืนยันว่าเป็นบริการนวัตกรรม |
| 8 | กด **บันทึก** |

## ตารางสรุปตามกลุ่มอายุ

| กลุ่ม | วิธีหลัก | วิธีสำรอง |
|-------|----------|-----------|
| < 7 ปี | ข้ามวิธีที่ 1 ได้เลย | Line OA หรือ Smart Card ของผู้ปกครอง |
| 7–14 ปี | FVS ก่อน 2 ครั้ง | Line OA หรือ Smart Card ของผู้ปกครอง |
| 15–59 ปี | FVS ก่อน 2 ครั้ง | ถ้าไม่ผ่านจริง → Smart Card + ยืนยันตัวตนแทน |
| ≥ 60 ปี | FVS ก่อน 2 ครั้ง | Smart Card + ยืนยันตัวตนแทน |
| ผู้พิการ | FVS ก่อน 2 ครั้ง | Smart Card + ยืนยันตัวตนแทน |

## ข้อสังเกตสำคัญ

- **ญาติสายตรง** สามารถสแกนใบหน้าแทนกันได้ (เช่น พ่อแม่สแกนแทนลูก หรือลูกสแกนแทนพ่อแม่) — มีตัวเลือกในระบบ
- **ห้ามให้คนอื่นสแกนแทน** โดยไม่มีความสัมพันธ์ที่ถูกต้อง
- กรณีที่ติด IN19 เพราะไม่ได้สแกน 2 ครั้งก่อน → สปสช. จะตรวจ log และดึงมาจ่ายใหม่วันจันทร์

## ญาติสายตรง — เงื่อนไขการสแกน Line OA แทน (Version 2)

ญาติสายตรงที่สามารถสแกน QR Line OA แทนผู้รับบริการได้ต้องมีเงื่อนไขครบ:
1. เป็น **บิดา หรือ มารดา หรือ บุตร** ของผู้รับบริการ
2. มี **นามสกุลตรงกัน**
3. มี **ทะเบียนบ้านที่อยู่เดียวกัน**

ถ้าญาติไม่ตรงเงื่อนไขข้างต้น (เช่น ปู่ ย่า หลาน) → ต้องใช้วิธีที่ 3 Smart Card

## เว็บ New Autent สำหรับ Smart Card วิธีที่ 3

- **URL:** autentsiceo.go.th/autent/ent code
- **เมนูที่ใช้:** "ยืนยันตัวตนแทนของผู้เข้ารับบริการโดยผู้ปกครอง/ผู้ดูแล/ผู้รับรอง"
- **เฉพาะ:** หน่วยบริการที่ขึ้นทะเบียนเป็นหน่วยนวัตกรรม UC เท่านั้น
- **ผลลัพธ์:** ประเภทการยืนยันตัวตนในรายงาน = "end point" (ไม่ใช่ FVS หรือ Line OA)
- ระบบตรวจสอบซ้ำอีกครั้งในขั้นตอน payment หลังบ้าน

## กรณีที่พบในไลฟ์ครั้งที่ 86

| กรณี | วิธีแก้ |
|------|---------|
| ผู้ป่วยผ่าตัดตา กระพริบตาลำบาก | ลอง FVS 2 ครั้ง → ถ้าไม่ผ่าน ใช้ Smart Card |
| ผู้ป่วยผ่าตัดสมอง/ใส่สาย NG tube | ลอง FVS 2 ครั้ง → ใช้ Smart Card (กลุ่มเปราะบาง) |
| เด็กกายภาพบำบัดที่บ้าน ไม่มีโทรศัพท์ | ใช้ระบบเดิม (Disability Portal) ไม่ต้องจองคิว |
| คนไข้สูงอายุ 72 ปี สแกนหน้าผ่าน | ยืนยัน FVS ใช้ได้กับผู้สูงอายุ ไม่ได้จำกัดที่ 60 ปี |
