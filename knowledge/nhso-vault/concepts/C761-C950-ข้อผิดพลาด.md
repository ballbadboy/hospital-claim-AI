# C761 / C950 — ข้อผิดพลาดโครงสร้างข้อมูล Financial Data Hub

## Definition

**C761** และ **C950** เป็นรหัสปฏิเสธการจ่ายที่มีแก่นสาระเดียวกัน เกิดจากหน่วยบริการส่งข้อมูลผ่าน **Financial Data Hub** แต่โครงสร้างข้อมูลไม่สมบูรณ์ตามมาตรฐาน 43 แฟ้ม ECM

| รหัส | ความหมาย | ปรากฏที่ |
|------|----------|---------|
| C761 | ข้อมูลไม่ผ่านการตรวจสอบโครงสร้าง | รายการ C ในระบบ ecam |
| C950 | หน่วยบริการส่งข้อมูลผ่าน Financial Data Hub ไม่สมบูรณ์ตามโครงสร้าง | FinancialData Hub (FDS) |

ทั้งสองรหัสนี้ **ตรวจสอบเฉพาะโครงสร้าง** ยังไม่ได้ตรวจเนื้อในข้อมูล

## วิธีแก้ไข

1. เข้าโปรแกรม **ecam System**
2. ไปที่เมนู **ดาวน์โหลด**
3. เลือก **รายงาน 4**
4. ตรวจสอบรายละเอียดโครงสร้างข้อมูลที่ไม่สมบูรณ์ และทำตามคำแนะนำที่ระบบแสดง
5. แก้ไขข้อมูลให้ครบตามโครงสร้าง 43 แฟ้ม ECM 16 man แล้วส่งใหม่

## สาเหตุที่พบบ่อย

- ข้อมูลบางฟิลด์ขาดหาย (null หรือ empty) ในแฟ้มที่ส่งผ่าน Financial Data Hub
- โครงสร้างไฟล์ไม่ตรงกับ schema ที่กำหนดในมาตรฐาน ECM 16 man
