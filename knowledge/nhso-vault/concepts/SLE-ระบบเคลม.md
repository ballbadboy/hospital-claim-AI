# SLE Claim System — ระบบประมวลผลข้อมูลเบิกจ่าย

## Definition

**SLE Claim System** (SLE CL System) คือระบบประมวลผลกลางของ สปสช. ที่รับข้อมูลจากทุกระบบบันทึกบริการของหน่วยนวัตกรรม นำมาตรวจสอบก่อนจ่าย (Pre-Audit) และส่งออกรายงานการชดเชยไปยัง Seamless for DMIS เพื่อวางฎีกาโอนเงิน

## วงจรการประมวลผล

```
ระบบบันทึกบริการต่างๆ
    ├── e-Claim (eKhlaim)
    ├── M-Care Plus
    ├── Krungthai Digital Health Platform (PP)
    └── Vendor Software (API)
           ↓  (เที่ยงคืน – 18:00 น. วันถัดไป)
    SLE CL System (ตรวจสอบก่อนจ่าย)
           ↓  (วันจันทร์สัปดาห์ถัดไป)
    Seamless for DMIS (รายงาน REP)
           ↓  (วันอังคาร–พุธ)
    วางฎีกา
           ↓  (วันพุธ/พฤหัสบดี ≤ 21:00 น.)
    Smart Money Transfer / NHO Budget (โอนเงิน)
```

## รายละเอียดขั้นตอน

| ขั้นตอน | เวลา | รายละเอียด |
|---------|------|-----------|
| รับข้อมูลจากระบบบันทึก | เที่ยงคืน – 18:00 น. | ระบบดึงข้อมูลจากทุก Vendor ผ่าน API |
| ประมวลผล Pre-Audit | วันจันทร์ถัดไป | ตรวจสอบสิทธิ์, ตรวจสอบโควต้า, ตรวจสอบ C-Code |
| ออกรายงาน REP | วันจันทร์ | ปรากฏใน Seamless for DMIS |
| วางฎีกา | วันอังคาร–พุธ | รอการอนุมัติโอนเงิน |
| โอนเงิน | วันพุธ/พฤหัสบดี ≤ 21:00 น. | เข้าบัญชีธนาคารของหน่วยบริการ |

> **หมายเหตุ:** ตั้งแต่กลางมกราคม 2569 การโอนเงินปรับเป็น**วันพฤหัสบดี** แทนพุธ เนื่องจากปรับระบบประมวลผล

## ระบบบันทึกบริการที่เชื่อมกับ SLE

1. **e-Claim / eKhlaim** — ระบบมาตรฐาน สปสช. สำหรับโรงพยาบาล
2. **M-Care Plus** — สำหรับหน่วยนวัตกรรม (คลินิก CI, OPD)
3. **Krungthai Digital Health Platform** — บันทึกงาน PP (ส่งเสริม/ป้องกัน)
4. **Vendor Software** — ระบบ HIS ของเอกชนที่เชื่อม API กับ สปสช.

## รหัสรายงานที่ออกจาก SLE

| รหัส | ประเภทรายงาน |
|-----|-------------|
| REP Summary | สรุปยอดรายงวดต่อหน่วยบริการ |
| REP Individual | รายละเอียดรายบุคคล |
| IEP Number | เลขงวดสำหรับโยงกับ Smart Money Transfer |

## C-Code ที่ SLE ตรวจสอบ

SLE ตรวจสอบ C-Code หลากหลายตัว รวมถึง:
- **C305** — ปิดสิทธิ์ไม่ถูกต้อง/ไม่สมบูรณ์ (ดู [[concepts/C305-กระบวนการอุทธรณ์]])
- **C30** — ปัญหาการปิดสิทธิ์ทั่วไป
- อื่นๆ ตามตาราง C ใน eClaim (ดู [[concepts/c-code-การแก้ไข]])
