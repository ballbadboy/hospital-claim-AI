# CI Claim (เคลมโรคเรื้อรัง)

**CI** = Chronic Illness — การเบิกจ่ายค่าบริการสำหรับผู้ป่วยโรคเรื้อรัง (เบาหวาน, ความดัน, CKD ฯลฯ) ที่ร้านยาให้บริการภายใต้โครงการ [[entities/บัตรทอง]]

## ขั้นตอนการส่งเคลม

1. ร้านยาให้บริการผู้ป่วยโรคเรื้อรัง
2. **บันทึกข้อมูล** ผ่านโปรแกรม **MC Care**
3. ข้อมูลถูกส่งเข้าระบบ สปสช. อัตโนมัติ
4. ระบบประมวลผลและแสดงใน [[concepts/seamless-for-dms]]

## เมื่อเคลม CI ไม่ได้รับการประมวลผล

**อาการ:** ข้อมูลบริการวันที่ X ไม่ปรากฏในระบบ

**วิธีตรวจสอบ:**
1. เข้า Seamless for DMS
2. เลือกเมนู REP → Individual (รายบุคคล)
3. กำหนดวันที่ที่สงสัย
4. ค้นหาและดูว่าข้อมูลปรากฏหรือไม่

**ถ้าพบข้อมูลใน Seamless for DMS:** แสดงว่าข้อมูลประมวลผลแล้ว (ไม่มีปัญหา)

**ถ้าไม่พบ:** ติดต่อ สปสช. เพื่อตรวจสอบเพิ่มเติม

## โปรแกรม MC Care

ซอฟต์แวร์ที่ร้านยาใช้บันทึกข้อมูลการให้บริการผู้ป่วยโรคเรื้อรัง ก่อนส่งออกรายงานเข้าระบบ สปสช.
