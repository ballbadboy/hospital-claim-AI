# ประชุมชี้แจงการบริหารกองทุนหลักประกันสุขภาพแห่งชาติ ปีงบประมาณ 2569 (บ่าย)

**Channel:** สปสช. | **Duration:** 2:25:16 | **Ingested:** 2026-04-13

## Summary

นี่คือช่วงบ่ายของการประชุมชี้แจงการบริหารกองทุนหลักประกันสุขภาพแห่งชาติ ประจำปีงบประมาณ 2569 จัดในรูปแบบ Hybrid ทั้ง onsite (500 คน) และ online ผ่าน Zoom และ Facebook Live โดยมีท่านรองนายกรัฐมนตรีท่านโสภลเป็นประธาน พร้อมผู้บริหารจากกระทรวงสาธารณสุขและสปสช. รวมถึงตัวแทนจากหน่วยงานสาธารณสุขทั่วประเทศ

ช่วงแรกประกอบด้วยพิธีมอบรางวัลสนับสนุนการใช้นวัตกรรมเครื่องมือแพทย์ไทยดีเด่น และการลงนาม MOU 3 หน่วยงาน (สปสช., กรมอนามัย, กรมสนับสนุนบริการสุขภาพ) เพื่อยกระดับอสม.ยุคใหม่ให้ดูแลผู้มีภาวะพึ่งพิง ช่วงสำคัญที่สุดคือการชี้แจงเป้าหมายการบริหารกองทุนปี 2569 โดยรองเลขาธิการสปสช.ทั้ง 4 ท่าน ครอบคลุมงบประมาณภาพรวม กระบวนการเบิกจ่าย การ audit และกองทุนท้องถิ่น/Long-term Care

ไฮไลท์สำคัญได้แก่: งบกองทุนหลักประกันปี 2569 สูงที่สุดในประวัติศาสตร์ (~4,100 บาท/หัว เพิ่ม 12%), กองทุน Long-term Care เพิ่มเป็น 5,114 ล้านบาท (จาก ~3,000 ล้าน), ปรับระบบ audit ใช้ AI, ยกเลิก error code "C" ใช้ "deny" แทน, และรองเลขาธิการประกาศ 4 แนวทางพัฒนาระบบเบิกจ่ายปี 2569

## Key Points

- **งบกองทุนปี 2569 New High**: ~4,100 บาท/หัวประชากร เพิ่มขึ้น 12% จากปีก่อน — สูงที่สุดตั้งแต่ก่อตั้งกองทุนมา 20+ ปี
- **MOU 3 หน่วยงาน**: สปสช. + กรมอนามัย + กรมสนับสนุนบริการสุขภาพ ลงนามร่วมมือยกระดับอสม.ยุคใหม่ดูแลผู้มีภาวะพึ่งพิง
- **กองทุนท้องถิ่น 3 กองทุน รวม 9,925 ล้านบาท** ปี 2569:
  - กองทุนตำบล (อบต./เทศบาล) — 45 บาท/หัวคงเดิม + เพิ่ม 4 บริการใหม่ ~800 ล้าน/ปี
  - กองทุนฟื้นฟูสมรรถภาพจังหวัด (อบจ.) — ขยายขอบเขตครบวงจร
  - กองทุน Long-term Care — 5,114 ล้านบาท (เพิ่มจาก ~3,000 ล้าน)
- **Long-term Care ปรับสำคัญ**: อัตราใหม่ 1,442 บาท/หัว; อาสาสมัครนักบริบาลชุมชนแบ่งตามชั่วโมงอบรม: 70 ชม. = 5,000 บาท/เดือน, 120 ชม. = 6,000 บาท/เดือน, 420 ชม. = 9,000 บาท/เดือน
- **4 แนวทางพัฒนาระบบเบิกจ่ายปี 2569** (นพ.อรรถพร):
  1. เชื่อมระบบข้อมูล — ทำระบบสำรอง สิทธิ์, เชื่อม SIS โรงพยาบาลกับ e-Claim โดยอัตโนมัติ (ทดลองกทม. กลางปี 69)
  2. ปรับระบบ Audit — ใช้ AI/OCR ตรวจสอบรหัส, ยกเลิก error "C" ใช้ "deny" แทน, ประกาศหลักเกณฑ์ 1 ครั้ง/ปี ต้นปีงบประมาณ
  3. ความโปร่งใส — URL สาธารณะดู Statement และ Smart Money Transfer, ปฏิทินจ่ายเงินรายปีทุกกองทุน
  4. CL Relation — เปิดเวทีหารือ, ไลฟ์ทุกวันศุกร์ตอบปัญหาโรงพยาบาล, ทีม Provider Center
- **C30 error wave**: ข้อมูลบริการปี 2568 — วันสุดท้ายส่งข้อมูล 31 ธันวาคม 2568; หลังจากนั้นเปิดอุทธรณ์ C30 ผ่านระบบ upload ไฟล์
- **Extrapolation ปี 69**: จะประกาศหลักเกณฑ์ล่วงหน้าและชัดเจนกว่าปีก่อน
- **ร่วมจ่าย (Co-payment)**: รับเป็นประเด็นสำหรับพิจารณา แต่ยังไม่มีมติ
- **เงินเดือน vs IP**: สำนักงบกำหนดสัดส่วนหักเงินเดือน ~50% ของเงินเดือนทั้งหมด คิดเป็นประมาณ 30-40% ใน UC Budget — ปรับได้โดยต้องหารือร่วมกับสำนักงบ
- **กทม. หน่วยนวัตกรรม เขต 13**: ระงับรับสมัครชั่วคราว ระหว่างวิเคราะห์ความหนาแน่น
- **สัดส่วน 1:20 ผู้ประกอบวิชาชีพ**: ปรับให้เป็น 20 เคส/ผู้ประกอบวิชาชีพ 1 คน (ไม่ใช่ต่อคลินิก)
