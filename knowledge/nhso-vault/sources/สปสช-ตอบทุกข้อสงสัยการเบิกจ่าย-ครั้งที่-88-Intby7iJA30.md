# สปสช. ตอบทุกข้อสงสัยการเบิกจ่าย ครั้งที่ 88

**Channel:** สปสช. | **Duration:** 1:23:40 | **Ingested:** 2026-04-13  
**Upload date:** 13 กุมภาพันธ์ 2569 | **Views:** 220

## Summary

ครั้งที่ 88 ของรายการสปสช.ตอบทุกข้อสงสัยการเบิกจ่าย เน้น 4 ประเด็นหลัก ได้แก่ การเข้าถึงประกาศบริหารกองทุนบนเว็บไซต์ของสปสช., แนวทางการยืนยันตัวตนผู้รับบริการสำหรับคลินิกพยาบาลชุมชนอบอุ่น (ครอบคลุมกรณีเด็กอายุต่ำกว่า 15 ปี และกลุ่มผู้สูงอายุ/เปราะบาง ที่ใช้ Smart Card ยืนยันตัวตนแทน), การอ่าน Statement และรายงาน REP สำหรับมือใหม่ และการตรวจสอบรายงานการโอนเงินกองทุน IP

เนื้อหาวิดีโอนี้ออกอากาศช่วงต้นหลังคลินิกพยาบาลเริ่มให้บริการในระบบ 30 บาทรักษาทุกที่ (9 กุมภาพันธ์ 2569) มีคำถามสดหลายร้อยข้อเกี่ยวกับโควต้าหมดเร็ว, การเลือก Service Code ผิด, และการอ่านไฟล์ REP. ช่วงท้าย ผู้เชี่ยวชาญฝ่ายตรวจสอบ (อาจารย์มานพ) ตอบคำถามจาก Google Form รวมถึงปัญหา tran_id แสดงค่า E+ ใน Excel และการเบิก IPD cancer anywhere

## Key Points

- **ประกาศบริหารกองทุน** ดูได้ที่ www.nhso.go.th → "ข้อมูลสำหรับหน่วยบริการ" → "ประกาศบริหารกองทุน" มีทั้งหมด 77 ฉบับ แบ่ง 7 หมวด ดาวน์โหลดแบบรวมเล่มหรือรายฉบับ สอบถาม 1330 ตลอด 24 ชั่วโมง
- **คลินิกพยาบาล** มีกว่า 2,000 หน่วยเข้าร่วมระบบ 30 บาทรักษาทุกที่ เริ่มให้บริการจริง 9 กุมภาพันธ์ 2569
- **Service Code** แยก 3 รหัส: PG 019002 (CI/รักษาโรคเบื้องต้น), PG 019003 (พยาบาลพื้นฐาน/ทำแผล), PG 006001 (PP) ต้องเลือกให้ตรงกับบริการที่ให้จริงทั้งในระบบ FVS/Line และตอนส่งเบิก
- **Smart Card ยืนยันตัวตนแทน** เฉพาะกลุ่มพิเศษ (เด็ก ≤15 ปี, ผู้สูงอายุ ≥60 ปีที่ Face Scan ไม่ผ่าน, ผู้พิการด้านใบหน้า, กลุ่มเปราะบาง) ผ่านระบบ New Auth (authenservice.go.th) → เสียบบัตรญาติสายตรง + เลือกประเภทบริการนวัตกรรม
- **โควต้าคลินิกพยาบาล** แบ่งเป็น 2 ประเภท: CI และ OPD Nursing ต้องเลือกให้ตรงก่อนใช้โควต้า มีปัญหาโควต้าหมดเร็วในช่วงเปิดใหม่ สปสช.กำลังปรับ
- **Flow การเบิกจ่าย** หน่วยบริการส่งข้อมูลผ่าน Financial Data Hub หรือ e-Claim → ประมวลผลรายวัน → REP (จันทร์/พุธ/ศุกร์) → Statement (OP ทุก 15 วัน, IP ทุกสิ้นเดือน) → โอนเงิน
- **REP Excel 6 ชีท**: Detail, Summary, Data Instrument (ถ้ามีเบิกอุปกรณ์), Data Drug, Deny (รายการติด C), Data 0 บาท
- **คอลัมน์ชดเชยสุทธิ OP** = ผลรวมคอลัมน์ BC–CX + DG (กองทุน Free Schedule) ดูรหัส C/Deny ในคอลัมน์ Error Code
- **REP ก่อน Statement ออก** ยอด IP จะยังไม่แสดงอัตราจ่ายจนกว่าจะสิ้นเดือน เมื่อ Statement ออกแล้วให้โหลด REP อีกครั้งเพื่อดูยอดชดเชยสุทธิ IP
- **Statement IP** ตรวจสอบผ่าน Smart Money Transfer (nhho.go.th) เปรียบเทียบยอดกับเลขงวดใน Seamless for DMIS
- **tran_id แสดง E+** ใน Excel เป็นเพียงการแสดงผลของโปรแกรม ไม่กระทบการส่งเคลม ให้ตรวจสอบค่าจริงใน Cell แทน
- **IPD cancer anywhere** รหัส 545 เป็นบริการ OP Refer ไม่ใช่ IP ถ้าส่งต่อผู้ป่วยนอกให้ใช้รหัสโครงการพิเศษแทน
- **V993 CAG/PCI** จะเปิดให้ยื่นคำขออนุมัติล่วงหน้าผ่านระบบ PPFS ดูแนวทางใน Data Form ประกอบ
- **ข้อมูลช่วงดัง** กรณีที่หน่วยบริการไม่พบชื่อผู้ป่วยใน Seamless แต่ได้รับเงินแล้ว ให้ตรวจสอบ REP Number และวันที่โอนตามตาราง
