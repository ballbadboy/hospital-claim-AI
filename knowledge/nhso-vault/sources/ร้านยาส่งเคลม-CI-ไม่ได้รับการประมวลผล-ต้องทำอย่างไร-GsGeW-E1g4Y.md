# ร้านยาส่งเคลม CI ไม่ได้รับการประมวลผล ต้องทำอย่างไร ?

**Channel:** สปสช. | **Duration:** 2:07 | **Ingested:** 2026-04-13

## Summary

วิดีโอนี้สาธิตขั้นตอนที่ร้านยาควรทำเมื่อพบว่าข้อมูลเคลม CI (Chronic Illness) ที่ส่งผ่าน MC Care ไม่ได้รับการประมวลผลในระบบ สปสช.

แนวทางแก้ไขหลักคือการเข้าตรวจสอบใน **ระบบ Seamless for DMS** (บูรณาการติดตามข้อมูลการจ่ายชดเชยโรคเฉพาะ) โดยใช้รหัสผู้ใช้งานที่สำนักงานกำหนดให้ เพื่อดูรายงานแบบ Individual ว่าข้อมูลบริการวันที่ 6-7 ปรากฏอยู่ในระบบแล้วหรือไม่

ระบบจะแสดงข้อมูลรายบุคคล พร้อม timestamp งวดปัจจุบัน สามารถ export รายงานออกมาตรวจสอบความครบถ้วน (67 ราย) ได้

## Key Points

- เข้าระบบ Seamless for DMS ด้วย user ของร้านยา + รหัสที่สำนักงานกำหนด
- ไปที่เมนู REP → รายงานแบบ Individual (รายบุคคล)
- กำหนดวันที่ที่ต้องการตรวจสอบ → กดปุ่มค้นหา
- ดูว่าบริการวันที่ 6-7 ปรากฏในรายงานหรือไม่
- ถ้าปรากฏ → แสดงว่าข้อมูลได้รับการประมวลผลแล้ว
- ส่งออกรายงาน (ปุ่ม Export สีส้ม) เพื่อตรวจสอบความครบถ้วน
