# สปสช. ตอบทุกข้อสงสัยเรื่องการเบิกจ่าย ครั้งที่ 25

**Channel:** สปสช. | **Duration:** 1:01:06 | **Ingested:** 2026-04-13

## Summary

รายการ Facebook Live "สปสช.พบหน่วยบริการ ตอบทุกข้อสงสัยเรื่องการเบิกจ่าย" ครั้งที่ 25 ออกอากาศวันที่ 1 พฤศจิกายน 2567 ดำเนินรายการโดยเจ้าหน้าที่สปสช. 2 ท่าน (ท่านรองและอาจารย์มานพ) ผ่าน Facebook Live มีผู้ชมราว 107 คน

ตอนต้นของรายการมีการประกาศเปลี่ยนแปลงสำคัญ: **คลินิกนวัตกรรม 7 วิชาชีพ** (เวชกรรม ทันตกรรม ร้านยา กายภาพบำบัด แพทย์แผนไทย เทคนิคการแพทย์ พยาบาล) จะปรับรอบจ่ายจาก "ทุก 3 วัน" เป็น **ทุกวันพุธ** เพื่อความสม่ำเสมอ ทั้งในและนอกจังหวัดนำร่อง และมีการแจ้งชะลอจ่ายบริการ OPW (OP Anywhere ในจังหวัด / เหตุสมควร) เพื่อตรวจสอบการทับซ้อนงบเหมาจ่ายรายหัวกับงบกรณี OPW

ช่วงถาม-ตอบครอบคลุมปัญหาหลากหลาย ได้แก่ ปัญหาการโอนเงินล่าช้าของคลินิกกายภาพบำบัดในและนอกจังหวัดนำร่อง ปัญหา error code C305/C349/C911/B03 ขั้นตอนการอุทธรณ์ผ่านระบบ EAL วิธีอ่าน Statement ใน Smart Money Transfer การส่งข้อมูล ODS (One Day Surgery) การเบิกบริการ HD (ฟอกไต) การเบิกยาสมุนไพร การเบิก Palliative Care (P Care) รอบจ่ายเงินเยี่ยมบ้านของคลินิกพยาบาล และสิทธิ์บัตรทองของ รพ.สต.ถ่ายโอน

## Key Points

- **รอบจ่ายเงินนวัตกรรมเปลี่ยนเป็นทุกวันพุธ** ทั้งในและนอกจังหวัดนำร่อง โดย IEP ออกวันจันทร์ → ตั้งฎีกาวันอังคาร → โอนเงินไม่เกิน 21:00 น. วันพุธ
- **ชะลอจ่าย OP Anywhere ในจังหวัด** เนื่องจากงบอาจทับซ้อนระหว่างงบเหมาจ่ายรายหัวกับงบกรณี OPW มีหนังสือเวียน ว 4841 แจ้งผู้อำนวยการหน่วยบริการ
- **ข้อมูลตกค้างที่ยังไม่จ่าย** หมายความว่าถูกปฏิเสธ สาเหตุหลักคือไม่ได้ปิดสิทธิ์ภายในวันให้บริการ → ติด C305 → ต้องอุทธรณ์
- **Error B03** เกิดจากปริมาณยาหรือหน่วยในการเบิกไม่ตรงกับ Fee Schedule ของสปสช. แก้ไขโดยปรับ drug catalog ให้ถูกต้องตามขนาดบรรจุและหน่วย
- **ODS (One Day Surgery)** ต้องผ่านการขึ้นทะเบียนจากเขตสุขภาพ ส่งข้อมูลเป็น Ambulatory Care (SV_type = A) ใน 16 แฟ้มมาตรฐาน ไปยัง Final Data Hub
- **HD ฟอกไต** รอบจ่ายทุก 15 วัน: งวด 1-15 → โอนวันที่ 22 / งวด 16-สิ้นเดือน → โอนวันที่ 6-7 ตรวจสอบผ่าน Simulate for DMS ใน CKD section
- **P Care (Palliative Care)** จ่ายเหมา 1,000 บาท/เดือน สูงสุด 6 เดือน + 3,000 บาทเมื่อผู้ป่วยเสียชีวิต บันทึกผ่าน e-Claim
- **Statement ยอดไม่ตรง** เกิดจากตัวเลขในคอลัมน์ Excel เป็น Text ไม่ใช่ตัวเลข ต้องแปลงก่อน SUM
- **C911** (สิทธิ์เกิดใหม่ทันที) หากผู้ป่วยเป็นสิทธิ์ว่างและลงทะเบียนใหม่ในวันรับบริการ สามารถเบิกเป็น AE Walk-in ได้ 1 ครั้ง
- **รพ.สต. ถ่ายโอน** ยังอยู่ในระบบหลักประกันสุขภาพแห่งชาติ ไม่สามารถเก็บค่ารักษาจากผู้มีสิทธิ์บัตรทองได้
