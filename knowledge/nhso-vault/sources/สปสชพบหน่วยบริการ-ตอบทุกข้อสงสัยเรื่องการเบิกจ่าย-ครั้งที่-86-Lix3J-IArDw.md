# สปสช.พบหน่วยบริการ ตอบทุกข้อสงสัยเรื่องการเบิกจ่าย ครั้งที่ 86

- **อาจารย์มานพ โลเฮือง** — ผู้เชี่ยวชาญฝ่ายตรวจสอบก่อนการจ่ายเงินชดเชยค่าบริการ สปสช.

## Summary

ครั้งที่ 86 ของรายการสปสช.พบหน่วยบริการครอบคลุม 2 ประเด็นหลัก ประเด็นแรกคือการอธิบายระบบยืนยันตัวตนเวอร์ชัน 2 สำหรับหน่วยบริการนวัตกรรม ที่เริ่มใช้จริงตั้งแต่วันที่ 12 มกราคม 2569 โดยเฉพาะการแก้ปัญหาสำหรับกลุ่มเปราะบาง (ผู้สูงอายุ เด็กเล็ก ผู้ป่วยที่สแกนใบหน้าไม่ได้) ด้วยการนำ Smart Card มาเป็นทางเลือกสำรองลำดับที่ 3 อย่างเป็นทางการ พร้อมแสดงขั้นตอนการใช้งานบนเว็บ autentsiceo.go.th/autent/ent code

ประเด็นที่สองครอบคลุมคำถามค้างจากการประชุมหน่วยบริการทั่วประเทศ (~2,000 แห่ง) เมื่อสัปดาห์ก่อนเกี่ยวกับรหัส C305 โดย อาจารย์มานพ โลเฮือง สรุปปัญหาเป็น 5 กลุ่ม ได้แก่ (1) ปัญหาระบบ vs. ความเข้าใจผิดของหน่วยบริการ (2) ปัญหารอยต่อระหว่างวัน (3) การปิดสิทธิ์ซ้ำหลายครั้งผ่าน EDC (4) รหัส error IN22 และ C30 (5) ประเด็นเฉพาะการฟอกไต และแจ้งว่าจากข้อมูล C305 ทั้งหมด 1,500,000 records พบว่า 700,000 records เกิดจากระบบ (สปสช.แก้อัตโนมัติ) ส่วนที่เหลืออีก 800,000 records เปิดให้อุทธรณ์ผ่าน ap349.nhso.go.th ระหว่างวันที่ 1–28 กุมภาพันธ์ 2569

ช่วงท้ายตอบคำถาม Live รวมถึงกรณีคลินิกพยาบาล IN23 (ยังไม่ได้เชื่อม API สภาการพยาบาล คาดเสร็จ 2 กุมภาพันธ์ 2569), กายภาพบำบัดที่บ้านกลุ่ม IMC4, และการสแกนหน้าไม่ผ่านสำหรับผู้ป่วยที่ผ่าตัดสมอง/ใส่สาย NG

## Key Points

1. **ระบบยืนยันตัวตน Version 2 เริ่ม 12 ม.ค. 2569** — เปลี่ยนจากการเสียบบัตรประชาชนอย่างเดียวมาเป็น Face Recognition เป็นวิธีหลัก เพราะงบกองทุนฯ ปีละ >200,000 ล้านบาท ต้องมี check-and-balance

2. **ลำดับความสำคัญของการยืนยันตัวตน 3 วิธี:**
   - **วิธีที่ 1 (หลัก):** Face Recognition (FVS) — ลองก่อนเสมอ ทุกกลุ่มอายุ รวมถึงอายุ >60 และ <15
   - **วิธีที่ 2:** Line OA สปสช. + QR Code — สำหรับกลุ่มอายุ >60 หรือ ≤15 ที่สแกนหน้าไม่ผ่านและมีมือถือ (รวมถึงญาติสายตรง = บิดา/มารดา/บุตร ที่มีนามสกุลตรงกันและทะเบียนบ้านเดียวกัน สามารถสแกน QR แทนได้)
   - **วิธีที่ 3 (อนุโลม):** Smart Card เสียบบัตรประชาชน — เฉพาะกลุ่มที่ไม่สามารถทำวิธี 1 และ 2 ได้จริงๆ เช่น เด็กอายุ <7 ปีที่ไม่ได้อยู่กับพ่อแม่, ผู้สูงอายุที่ตาบอด/ใบหน้าเปลี่ยนมาก

3. **Smart Card วิธีที่ 3 ต้องมีหลักฐานว่าลอง FVS ก่อนแล้วไม่ผ่าน** — ระบบเก็บ log FVS ทุกครั้ง ถ้าไม่มี log และเสียบบัตรเลย จะถูกปฏิเสธการจ่าย

4. **เว็บ New Autent:** `autentsiceo.go.th/autent/ent code` — ใช้เมนู "ยืนยันตัวตนแทนของผู้เข้ารับบริการโดยผู้ปกครอง/ผู้ดูแล/ผู้รับรอง" เฉพาะหน่วยนวัตกรรม UC เท่านั้น; ผู้รับรองแบ่งเป็น ผู้ปกครอง / ผู้ดูแล / เจ้าหน้าที่ (ควรใช้ญาติก่อน); ประเภทการปิดสิทธิ์จะเป็น "end point" (ไม่ใช่ FVS หรือ Line OA)

5. **C305 ทั้งหมด ~1,500,000 records แยกเป็น 2 ส่วน:**
   - **700,000 records** — เกิดจากความผิดพลาดของระบบ สปสช.แก้ให้อัตโนมัติ
   - **800,000 records** — แยกเป็นรายหน่วยบริการ ขึ้นเว็บไซต์ให้ตรวจสอบ เปิดอุทธรณ์ 1–28 กุมภาพันธ์ 2569

6. **ปัญหารอยต่อระหว่างวัน (Midnight Boundary Problem):** ผู้ป่วยรับบริการ 23:00 เสร็จหลังเที่ยงคืน → ระบบตัดรอบก่อนเที่ยงคืน → ต้องส่งข้อมูลภายในวันนั้น ระบบจะขยายเวลาตรวจสอบเป็น 24 ชั่วโมง

7. **ปัญหาปิดสิทธิ์ซ้ำผ่าน EDC:** ผู้ป่วยไปหลายแผนกได้ KM Code หลายเลข → หน่วยบันทึกหลายเลขคั่นด้วยอักขระพิเศษ → ระบบตรวจพบ KM Code ไม่ถูกต้องติด C305 → สปสช. จะคัดข้อมูลที่ 15% ของ C305 ทั้งหมดของหน่วยนั้นมาตรวจ ถ้าพบ pattern นี้จะเลือกเฉพาะ KM Code ครั้งสุดท้ายมาประมวลผล

8. **IN22:** ปิดสิทธิ์ไม่ผ่าน FVS หรือ Line OA — เกิดกับบริการส่งเสริมป้องกัน (PP) ที่ปิดสิทธิ์ผ่านระบบ Krungthai Digital Health Platform ซึ่งยังไม่รองรับ สปสช. จะ waive และ reprocess ให้

9. **C30:** ส่งเบิกผ่าน eClaim แต่ปิดสิทธิ์ไม่ถูกเงื่อนไข หรือวัน/เวลาคลาดเคลื่อน — ต้องให้วันและเวลาปิดสิทธิ์ตรงกับเวลาสิ้นสุดบริการจริง ไม่ใช่เวลาที่ส่งข้อมูล

10. **ช่องทางอุทธรณ์ C305:** `ap349.nhso.go.th` — เปิด 1 กุมภาพันธ์ – 28 กุมภาพันธ์ 2569; ต้องอัปโหลดเอกสารแยกรายบุคคล (ไม่ใช่ชุดรวม); ระยะเวลาตรวจสอบมาตรฐาน 30 วัน; หลังอนุมัติออก Statement ปกติ

11. **กรณีฟอกไต (AVF/AVG):** อุทธรณ์ผ่านระบบ eClaim → ระบบ EPL; ฟอกไตต้องปิดสิทธิ์หลังจากฟอกเสร็จเท่านั้น ปิดก่อนไม่ได้; เอกสาร ฮีโมชีส ต้องอัปโหลดแยกรายบุคคล

12. **คลินิกพยาบาล IN23:** API สภาการพยาบาลยังไม่เสร็จ (วางแผนเสร็จ 2 ก.พ. 2569); ให้บริการได้เฉพาะเคสเก่า (ส่งข้อมูลเข้าระบบ eClaim ตั้งแต่พฤศจิกายน–ธันวาคม 2568); 11 เคสที่ถามน่าจะเป็นเคสใหม่ที่ยังไม่สามารถให้บริการได้

13. **กายภาพบำบัดที่บ้าน (IMC4):** ไม่ต้องจองคิว ใช้ระบบเดิม — โรงพยาบาลส่งต่อผ่าน Disability Portal ก่อน แล้วคลินิกไปให้บริการที่บ้านได้ไม่เกิน 20 ครั้ง/คน/6 เดือน

14. **ผู้ป่วยสแกนหน้าไม่ผ่าน (ผ่าตัดสมอง/ใส่สาย NG/กระพริบตาไม่ได้):** ใช้วิธีที่ 3 Smart Card ได้ แต่ต้องมี log FVS ว่าลองสแกนแล้วไม่ผ่านก่อน

15. **ปีงบประมาณ 2569 ตั้งแต่ 1 มกราคม 2569:** ไม่มีการติด C305 แล้ว ระบบแจ้ง error ทันทีตอนส่งข้อมูล หน่วยบริการสามารถแก้ไขได้เลยก่อนการตรวจสอบ
