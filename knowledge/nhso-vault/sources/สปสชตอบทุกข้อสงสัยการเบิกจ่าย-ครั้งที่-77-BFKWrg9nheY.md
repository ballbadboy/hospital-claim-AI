# สปสช.ตอบทุกข้อสงสัยการเบิกจ่าย ครั้งที่ 77

**Channel:** สปสช. | **Duration:** 1:14:08 | **Ingested:** 2026-04-13

**อัปโหลด:** 18 พฤศจิกายน 2568 | **วิทยากร:** ท่านรองเลขาธิการ + อาจารย์มานพ โยเฮือง (ผู้เชี่ยวชาญฝ่ายตรวจสอบก่อนการจ่ายชดเชย)

## Summary

ครั้งที่ 77 ออกอากาศกลางเดือนพฤศจิกายน 2568 โดยมีเรื่องแจ้งสำคัญ 2 เรื่องก่อนเริ่มตอบคำถาม ได้แก่ (1) สปสช. ชะลอรับสมัครขึ้นทะเบียนหน่วยบริการรับส่งต่อเฉพาะด้าน 7 ประเภท (นวัตกรรม) ตั้งแต่ 10 พฤศจิกายน–31 ธันวาคม 2568 เพื่อปรับปรุงระบบยืนยันตัวตน Face Recognition และจัดเครือข่ายโรงพยาบาลแม่ข่าย-คลินิกบริวาร และ (2) พบปัญหาข้อมูลบริการวันที่ 9 พฤศจิกายน 68 ของหน่วยนวัตกรรมไม่ถูกประมวลผลออกรายงานวันจันทร์ เนื่องจากปัญหาการเชื่อมข้อมูลปิดสิทธิ์

ส่วนตอบคำถามครอบคลุมหลายหัวข้อ ได้แก่ ปัญหาการตกหล่นของข้อมูลคลินิกพยาบาลจาก Care/Care Plus, การอุทธรณ์ C305 ในเดือนมกราคม 2569 ผ่าน 349.nhso.go.th, กรณี Walk-in ในจังหวัดเดียวกันจ่าย 0 บาท, กรณี IN002 ผิดสัญชาติจาก SRM migration, วิธีอ่านรายงาน IEP/REP ในระบบ Seamless for DMIS, และปัญหาหลักประกันสัญญาคลินิกพยาบาลใน One Service

## Key Points

- **ชะลอขึ้นทะเบียนนวัตกรรม** 10 พ.ย.–31 ธ.ค. 68 เพื่อเตรียม Face Recognition + จัดเครือข่ายแม่ข่าย
- **เครือข่ายโรงพยาบาล-คลินิก:** โรงพยาบาลส่งคำสั่งแลบหรือกายภาพไปคลินิกล่วงหน้า ลดความอัด ใช้ HeiLink (ต่างจังหวัด) หรือ M-Refer (กทม.)
- **ปัญหาประมวลผลตกหล่น:** วันที่ 9 พ.ย. 68 (วันอาทิตย์) ข้อมูลคลินิกพยาบาลบางส่วนเชื่อมกับระบบปิดสิทธิ์ไม่ครบ → ไม่ผ่าน On Screen Review → ไม่ออกรายงานวันจันทร์
- **C305 อุทธรณ์ ม.ค. 69:** เปิด 349.nhso.go.th ประมาณ 1 เดือน ส่งเอกสาร PDF (2 ฉบับ) + CSV (2 คอลัมน์: Tran ID + เหตุผล)
- **Walk-in ในจังหวัด = 0 บาท:** ต้องตั้งค่า UUC=2 ใน 16 แฟม แล้วออกรายงานไปเรียกเก็บกันในจังหวัด
- **Walk-in ข้ามจังหวัด:** เบิก Project Code Walk-in ที่กองทุนกลางได้ แต่ได้ค่า Point 0.5 เท่า
- **IN002** ที่เกิดช่วง 26 ก.ย.–8 ต.ค. 68: เกิดจาก migration ERM→SRM ทำให้สิทธิ์ไม่อัปเดต → ทำหนังสือแจ้ง สปสช. ขอแก้ไข
- **รายงาน REP/IEP ใน Seamless DMIS:** REP Number = "ถุง" ชุดข้อมูลรายสัปดาห์ ออกวันจันทร์ มีเลขงวด (IEP) แยกย่อย ดาวน์โหลด Excel ได้จาก Unit Report
- **หลักประกันสัญญาคลินิกพยาบาล:** ปัญหาระบบ One Service ค้นหาเลขสัญญาไม่เจอ → สปสช. อัปโหลดสัญญาขึ้นระบบใหม่ให้ทีละหน่วย
- **งบนวัตกรรม 3,700 ล้านบาท (ปีงบ 69):** กำหนดปลายปิด มีโควต้าบริการต่อผู้ให้บริการ ต้องจัดคิวไม่ให้เกินที่กำหนด
