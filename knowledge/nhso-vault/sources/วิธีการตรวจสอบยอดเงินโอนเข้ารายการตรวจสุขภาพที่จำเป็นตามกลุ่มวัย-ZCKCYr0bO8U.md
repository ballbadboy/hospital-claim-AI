# วิธีการตรวจสอบยอดเงินโอนเข้ารายการตรวจสุขภาพที่จำเป็นตามกลุ่มวัย

**Channel:** สปสช. | **Duration:** 4:38 | **Ingested:** 2026-04-13

## Summary

วิดีโอ Live จาก สปสช. ตอบคำถามหน่วยบริการในกทม. (เชิงรุก) เรื่องการตรวจสอบยอดเงินโอนเข้าจากการตรวจสุขภาพที่จำเป็นตามกลุ่มวัย

**คุณมานพ โยเฮือง** (ผู้เชี่ยวชาญ ฝ่ายตรวจสอบก่อนการจ่ายชดเชย) สาธิตขั้นตอนจริงบนหน้าจอ: เริ่มจาก Portal สปสช. → ไปยัง **Krungthai Digital Health Platform** → เข้า **Seamless for DMIS** → เมนู ข้อมูล → **RBF** → เลือก SUMMARY หรือ INDIVIDUAL → ดูรายงานการชดเชยรายการตรวจสุขภาพกลุ่มวัย

ระบบแสดงรายการตรวจสุขภาพพร้อมยอดชดเชย เช่น **80 บาท** และ **120 บาท** ต่อรายการ (3 รายการในตัวอย่าง) โดยมี PS CODE กำกับแต่ละรายการ

## Key Points

- **ระบบบันทึก:** กรุงไทย — ปิดสิทธิ์ → สร้างใบเบิกจ่าย → ตัดเที่ยงคืนวันอาทิตย์ → ออกรายงานวันจันทร์
- **URL ระบบหลัก:** `seamlessforme.nhso.go.th`
- **ชื่อระบบเต็ม:** ระบบบูรณาการข้อมูลสิทธิการรักษาพยาบาล (Seamless for DMIS)
- **เส้นทางนำทาง:** Portal icon grid → คลิก **Krungthai Digital Health Platform** → DMIS → เมนู **ข้อมูล** → **RBF**
- **รายงาน RBF มี 2 แบบ:** "RBF : ราคา RBF และ SUMMARY" หรือ "RBF : ราคา RBF และ INDIVIDUAL"
- **อัตราชดเชยตัวอย่าง:** 80.00 บาท, 120.00 บาท ต่อรายการตรวจ (ราคาเบิกอาจแสดง 0.00 แต่รวมเงินจ่ายจริง)
- **คอลัมน์รายงานหลัก:** รายการตรวจสุขภาพที่จำเป็น, จำนวน, ราคากลาง, ราคาเบิก, รวมเงิน, PS CODE
- ถ้าไม่พบใน path หลัก → ลองไอคอน **บัตรประชาชนใบเดียว** แทน
