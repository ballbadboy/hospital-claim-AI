# สปสช. ตอบทุกข้อสงสัยเรื่องการเบิกจ่าย ครั้งที่ 24

**Channel:** สปสช. | **Duration:** 1:02:43 | **Ingested:** 2026-04-13

**ผู้ดำเนินรายการ:**
- **ทพ.อรรถพร ลิ้มปัญญาเลิศ** — รองเลขาธิการ สปสช.
- **คุณมานพ โยเฮือง** — ผู้เชี่ยวชาญ ฝ่ายตรวจสอบก่อนการจ่ายชดเชยค่าบริการ

## Summary

รายการ "สปสช. พบหน่วยบริการ ตอบทุกข้อสงสัยเรื่องการเบิกจ่าย" ครั้งที่ 24 ออกอากาศสด 25 ตุลาคม 2567 เป็น Q&A สดระหว่างเจ้าหน้าที่ สปสช. กับหน่วยบริการนวัตกรรม (คลินิกเอกชน) ที่เข้าร่วมโครงการ 30 บาทรักษาทุกที่ด้วยบัตรประชาชนใบเดียว รูปแบบรายการเป็นการตอบคำถามสดจากแชทผู้ชมและคำถามที่ส่งมาล่วงหน้า โดยทีมงานหลังบ้านค้นข้อมูลในระบบแบบ real-time เพื่อตอบรายกรณี

ประเด็นสำคัญที่ประกาศในตอนนี้ได้แก่ การปรับรอบการจ่ายเงินสำหรับหน่วยบริการนวัตกรรมในปีงบประมาณ 2568 เป็นโอนทุกวันพุธ (จากเดิมทุก 3 วัน) สำหรับจังหวัดนำร่อง และกำลังทดสอบขยายไปจังหวัดที่ไม่ใช่นำร่องอีก 31 จังหวัด รวมถึงการชี้แจงปัญหา error code ยอดนิยม ได้แก่ C305, C438, C911, C172, C152 พร้อมวิธีแก้ไข

นอกจากนี้ยังมีการสาธิตการใช้งานระบบ eCam System สำหรับค้นหารหัส C (error code) และแนวทางปฏิบัติสำหรับคลินิกพยาบาล คลินิกกายภาพบำบัด คลินิกแพทย์แผนไทย ร้านยา โรงพยาบาลส่งเสริมสุขภาพตำบล (รพสต.) และโรงพยาบาลสังกัดรัฐที่เบิกจ่ายกรณีพิเศษ (HBOT, DSS/DCS)

## Key Points

- **ปรับรอบจ่ายเงินนวัตกรรมปี 68:** หน่วยนวัตกรรมในจังหวัดนำร่อง → จ่ายทุกวันพุธ (ถ้าพุธตรงวันหยุด เลื่อนเป็นวันก่อนหรือหลัง 1 วัน) เพิ่งจ่ายรอบแรกของปี 68 เมื่อวันที่ 25 ตุลาคม 2567
- **ขยายรอบพุธไปนอกจังหวัดนำร่อง:** กำลังทดสอบ 31 จังหวัด คาดว่าจะประกาศอย่างเป็นทางการเร็วๆ นี้ ปัจจุบันหน่วยนอกนำร่องยังอยู่รอบ 15 วัน (โอนวันที่ 7 และ 25 ของเดือน)
- **วิธีปิดสิทธิ์:** แนะนำให้หน่วยบริการนวัตกรรมปิดสิทธิ์ทุกครั้งที่ให้บริการผ่านเว็บ New Attend หรือ smart card เพื่อป้องกัน C305 และปัญหาการโอนเงิน
- **C305 — ให้บริการแต่ไม่ปิดสิทธิ์ภายในวัน:** กรณีเสียบบัตร (smart card) แต่ไม่ได้กดปิดภายในวัน สปสช. จะนำมาจ่ายในรอบถัดไปโดยอัตโนมัติ; กรณีไม่ได้เสียบบัตรเลยต้องทำหนังสืออุทธรณ์เป็นรายกรณี
- **C305 กรณีเด็กไม่มีบัตรประชาชน:** แนบสูติบัตร + เวชระเบียน + หนังสือปะหน้าอุทธรณ์ ขออนุมัติจากเลขาธิการเป็นรายๆ
- **C438 — สิทธิ์ไม่ตรงกับที่เบิก:** ต้องตรวจสอบสิทธิ์ ณ วันรับบริการว่าเป็นสิทธิ์ UC / ข้าราชการ / อปท. / ประกันสังคม ให้ตรงกับที่เบิก
- **C911 — ข้อมูลไม่ครบสำหรับการประมวลผล:** ขาดรายการย่อยหรือเงื่อนไข (ฉุกเฉิน, walk-in, ส่งต่อข้ามจังหวัด ฯลฯ) ต้องแก้ไขก่อนส่งใหม่
- **C172 — ใช้รหัส IP กับผู้ป่วยนอก:** รหัส IP002–IP006 ใช้ได้เฉพาะ IPD ไม่ใช้กับ OPD
- **C152 — เบิกซ้ำซ้อน OPD:** ผู้ป่วยคนเดียวกันมี 2 รายการ OPD ในวันเดียว ต้องตรวจสอบว่ารวม visit ได้หรือไม่
- **ระบบ eCam System:** ดาวน์โหลดเอกสาร "รายละเอียดการผิดพลาดของข้อมูลรหัสติด C" ได้ที่ eclaim.nhso.go.th → ดาวน์โหลด ระบบอัปเดตรหัส C อย่างต่อเนื่อง
- **OSR (On Screen Review / osr appeal):** ระบบตั้งเตือน LINE Alert แจ้งเมื่อมีข้อมูลต้องตรวจสอบ ต้องตอบสนองภายใน 10 วัน หน่วยบริการควรเข้าตรวจทุกวัน ก่อนดู Seamless for DMS
- **คลินิกกายภาพบำบัด:** ต้องดูรายงานใน "ระบบสารสนเทศฟื้นฟู" (ไอคอนคนพิการ) ไม่ใช่ไอคอน 30 บาท หรือ KTP หรือ iMet
- **HBOT (Hyperbaric Oxygen Therapy):** กรณีเบิก 2 ชั่วโมงแต่จ่าย 1 ชั่วโมง ต้องตรวจสอบข้อมูลที่บันทึก เพราะสปสช. จ่ายตามที่บันทึกในระบบ
- **ร้านยาโมเดล 3:** ต้องปิดสิทธิ์ทุกครั้งที่จ่ายยา ผ่านเว็บ New Attend โดยใช้ leader card
- **รพสต. บริการเชิงรุก (PP Free Schedule):** ออกให้บริการยาธาตุเหล็กเชิงรุกได้ แต่ถ้าต้องออกนอกสถานที่ต้องขออนุญาต สสจ. และ สปสช.เขตก่อน
- **Smart Money Transfer:** ใช้ตรวจสอบยอดโอนเงิน เข้าผ่าน nhso.go.th → หน่วยบริการ → บริการออนไลน์ → NHO Budget → Smart Money Transfer
