# สปสช. ตอบทุกข้อสงสัยเรื่องการเบิกจ่าย ครั้งที่ 40

**Channel:** สปสช. | **Duration:** 1:50:25 | **Ingested:** 2026-04-13

**ออกอากาศ:** 17 กุมภาพันธ์ 2568 | **บันทึก Live ณ:** ปั๊ม Bangchak สาขาอ่อนนุช 55

**วิทยากรหลัก:**
- ท่านรอง (ผู้บริหาร สปสช.) — เปิดรายการ บรรยายภาพรวม
- คุณมานพ โยเฮือง — ผู้เชี่ยวชาญ ฝ่ายตรวจสอบก่อนการจ่ายชดเชยค่าบริการ

## Summary

การ Live ครั้งที่ 40 นี้ถ่ายทำ ณ ปั๊ม Bangchak อ่อนนุช 55 ซึ่งเป็นสถานที่เปิดคลินิกเวชกรรมในปั๊มน้ำมัน ในฐานะหน่วยบริการนวัตกรรมภายใต้โครงการ 30 บาทรักษาทุกที่ รายการนี้เน้นตอบคำถามจากหน่วยบริการ (รพสต., คลินิก, โรงพยาบาล) ที่ส่งมาผ่าน Google Form และ Facebook Live Chat เกี่ยวกับ **การ Authentication (พิสูจน์ตัวตนก่อนรับบริการ / Auth Point)** และ **การปิดสิทธิ์ (End Point)** ซึ่งเป็น 2 แนวคิดที่มักถูกสับสน

ครั้งที่ 40 นับว่าเป็นครั้งที่ทำมาครบ 10 เดือน และยังพบว่าความสับสนระหว่างคำว่า "Authentication" กับ "การปิดสิทธิ์" ยังคงมีอยู่ในหน่วยบริการจำนวนมาก ผู้ดำเนินรายการจึงอธิบายอย่างละเอียดทั้ง Flow กระบวนการ, 7 วิธีปิดสิทธิ์, กรณียกเว้น และปัญหาที่พบบ่อยอย่าง Error C305

วิดีโอครอบคลุมตั้งแต่คำนิยามพื้นฐานไปจนถึงสถานการณ์จริงหลากหลายกรณี เช่น ผู้ป่วยเรื้อรัง, บริการ Telemedicine (emet), การคัดกรองมะเร็งปากมดลูก HPV DNA Test, การจองเคลม, OP Anywhere, AE ข้ามจังหวัด รวมถึงอธิบายรอบการจ่ายเงิน (Statement) 15 วัน และวิธีดูปฏิทินการจ่าย

## Key Points

- **Authentication (อทน) ≠ การปิดสิทธิ์:** Authentication คือการพิสูจน์ตัวตน "ก่อน" รับบริการ (Auth Point); การปิดสิทธิ์คือการยืนยันสิทธิ์ "หลังสิ้นสุด" บริการ (End Point) — ทั้งสองเป็นคนละขั้นตอนกัน
- **Error C349 vs C305:** ไม่ผ่าน Authentication → ติด C349; ไม่ผ่านปิดสิทธิ์ → ติด C305
- **นโยบาย 30 บาทรักษาทุกที่ (1 มกราคม 2568):** เปลี่ยน Focus จาก Authentication (ก่อนบริการ) มาเน้น **การปิดสิทธิ์** (หลังบริการ) เป็นหลัก
- **7 วิธีปิดสิทธิ์:** มี 6 วิธีตามประกาศ + วิธีที่ 7 (Minimal Data Set) ที่ซักซ้อมกับ สป.
- **วิธีที่ 6 มีข้อจำกัด:** ใช้ Authentication + ส่งข้อมูลเบิกจ่ายภายในวัน (ก่อนเที่ยงคืน) — ถ้าส่งข้อมูลไม่ทันในวันจะติด C305
- **วิธีที่ 1–5 และ 7 ไม่จำเป็นต้องส่งข้อมูลในวัน:** ยืดหยุ่นกว่า แนะนำให้ใช้แทนวิธีที่ 6 หากมีปัญหา
- **การจองเคลม (Claim Reservation):** = วิธีที่ 7 — ต้องทำ Authentication + ส่ง Minimal Data Set ผ่าน API → Financial Data Hub
- **OP Anywhere และ AE ข้ามจังหวัด:** ต้องปิดสิทธิ์ทุกครั้ง เบิกจากกองทุนกลาง ไม่ใช่งบ Capitation จังหวัด
- **Walk-in ในจังหวัด:** ไม่ต้องเบิกกองทุนกลาง ให้เรียกเก็บกันเองตามหลักเกณฑ์จังหวัด เพราะแต่ละหน่วยได้รับ Capitation แล้ว
- **รอบชดเชย:** ผู้ป่วยนอก (OP/PP) จ่าย 2 รอบต่อเดือน คือรอบ 1–15 (Statement วันที่ ~17) และรอบ 16–สิ้นเดือน (Statement วันที่ 3 เดือนถัดไป) — โอนเงินประมาณวันที่ 20–25 และ 10 ของเดือนถัดไปตามลำดับ
- **เลข Auth Code = เลขปิดสิทธิ์:** เป็นเลขเดียวกันในระบบ เรียกว่า "Approve Code"
- **กรณีปิดสิทธิ์ด้วยบัตรผู้ปกครอง:** ใช้สำหรับเด็กอายุต่ำกว่า 7 ปีที่ไม่มีบัตรประชาชน — เสียบบัตรผู้ปกครองแล้วพิมพ์เลข 13 หลักของเด็ก
- **HPV DNA Test (คัดกรองมะเร็งปากมดลูก):** ปิดสิทธิ์เมื่อเก็บ Specimen เสร็จ ไม่ต้องรอผล 7–14 วัน — บันทึกบริการย้อนหลังใน KTP ได้
