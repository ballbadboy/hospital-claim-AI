# ตอบทุกข้อสงสัยเรื่องการเบิกจ่าย ครั้งที่ 79

**Channel:** สปสช. | **Duration:** 1:00:58 | **Ingested:** 2026-04-13

## Summary

ไลฟ์ครั้งที่ 79 ออกอากาศวันศุกร์ที่ 28 พฤศจิกายน 2568 ดำเนินรายการโดยท่านรองเลขาธิการ สปสช. และอาจารย์มานพ โยเฮือง เปิดด้วยการแจ้งข่าวสำคัญ 2 เรื่อง ได้แก่ การยกเว้นการปิดสิทธิ์สำหรับหน่วยบริการในพื้นที่อุทกภัย 7 จังหวัดภาคใต้ และรายละเอียดกระบวนการอุทธรณ์ข้อมูลปีงบประมาณ 2568 ที่ต้องดำเนินการให้แล้วเสร็จภายใน 31 ธันวาคม 2568

จากนั้นตอบคำถามจากหน่วยบริการหลากหลาย ครอบคลุมประเด็นทางเทคนิคของ C-code ต่างๆ (C30, C305, C438, C512, C514, C998), ปัญหาการโอนเงินล่าช้าของ UC IP และ อปท., การใช้ระบบ FVS, ข้อผิดพลาดในการคัดลอก Approve Code และการตรวจสอบ D001 จาก Auditor

## Key Points

- น้ำท่วม 7 จังหวัดภาคใต้ (พัทลุง นราธิวาส ยะลา สงขลา ปัตตานี ตรัง สตูล): ยกเว้นการปิดสิทธิ์และการส่งข้อมูลล่าช้า ตั้งแต่ 21 พ.ย. 2568 จนกว่าจะประกาศยกเลิก
- อุทธรณ์ปีงบ 2568: ต้องส่งเอกสารถึง สปสช. ภายใน 31 ธ.ค. 2568 (อ้าง ข้อ 8.4.6 ประกาศบริหารกองทุน)
- C998 หลัง 31 ธ.ค.: ทำหนังสืออ้างข้อมูลอุทธรณ์ชุดเดิม เพื่อให้ทีมงานปลดล็อค
- C30 แก้ไข: เปิด 1–31 มกราคม 2569 (เตรียมข้อมูลได้เลย)
- HPV/L43 LBC: คนไข้ไม่ได้มาแต่ specimen มา → ยกเว้นปิดสิทธิ์ (ประสานระบบ Krungthai DHP)
- ทันตกรรมนวัตกรรม: สปสช. ไม่ได้แบ่งงบ จ่ายตามผลงานจริง
- C438 อปท.: เกิดจากระบบ SIM ช่วง 26–28 ก.ย. คลาดเคลื่อน → ส่งข้อมูลยืนยันสิทธิ์อปท.ใหม่
- UC IP งวดแรก ล่าช้า: รอหารือ สธ. เรื่องระบบ audit และ monitoring ปี 69
- อปท. ค้างจ่าย: ปี 68 ค้างจาก มี.ค. ประมาณ 6,000 ล้าน ได้งบปี 69 มา 4,000 ล้าน ยังขาด 2,000 ล้าน (National Clearing House)
- Approve Code: ห้ามมีช่องว่างติดมาเมื่อคัดลอกจากระบบ New AEN
- C305: ปิดสิทธิ์ก่อนเวลาเข้ารับบริการ → time series ย้อนแย้ง
- D001: ถูกทักท้วงจาก Auditor มีเวลา 10 วัน เพื่อเข้าระบบตรวจสอบและอัปโหลดเวชระเบียน
- C514: ส่งต่อในจังหวัดเดียวกัน เบิกพาหนะไม่ได้
- แพทย์แผนไทย: ออกเลขงวดแล้ว โอนเงินคืนวันนั้น
- FVS ใบหน้า: ต้องใช้โปรแกรมของ สปสช. เท่านั้น (เชื่อมข้อมูลกับเครือข่าย สปสช.)
