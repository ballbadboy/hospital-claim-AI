# สปสช. ตอบทุกข้อสงสัยการเบิกจ่าย ครั้งที่ 87

**Channel:** สปสช. | **Duration:** 1:02:53 | **Ingested:** 2026-04-13  
**Upload date:** 6 กุมภาพันธ์ 2569 | **Views:** 220

## Summary

รายการสดทุกวันศุกร์เวลา 13:00–14:00 น. ครั้งที่ 87 มีวิทยากร 4 ท่าน ได้แก่ (1) คุณทวัชัย เรืองโรจน์ — ผู้ช่วยเลขาธิการ สปสช. สายงานบริหารกองทุน (2) อาจารย์มานพ โยเฮือง — ผู้เชี่ยวชาญสำนักก่อนจ่าย (3) คุณกฤตพล ไพสัน — ผู้อำนวยการฝ่ายพัฒนาแพลตฟอร์มและนวัตกรรมดิจิตอล (4) ผอ.อัครัต อย่างไพบูลย์ — ผู้อำนวยการฝ่ายบริหารนโยบายดิจิตอลและความมั่นคงปลอดภัยไซเบอร์

ประเด็นหลักที่ 1 คือการชี้แจงข่าวลือที่ว่า สปสช. จ่ายผู้ป่วยใน (IP) เพียง 3,505 บาท/AdjRW แทนที่จะเป็น 8,350 บาท ตามประกาศบอร์ด ซึ่งที่จริง สปสช. ยังคงจ่าย 8,350 บาท ให้ทุกสังกัด แต่หน่วยบริการในสังกัดสำนักงานปลัดกระทรวงสาธารณสุข (สป.สธ.) มีการบริหารจัดการภายในโดยหักค่าแรงภาพรวม 12,000 ล้านบาท และกันเงิน 6,326 ล้านบาทสำหรับ IP เพื่อเติมตามเกณฑ์ของกระทรวง ซึ่งเป็นเรื่องภายในกระทรวง ไม่ใช่การที่ สปสช. ลดอัตราจ่าย

ประเด็นหลักที่ 2 คือการเปิด Appeal C305 ผ่านระบบ appel349.nhso.go.th โดยชี้แจงวิธีค้นหาข้อมูลที่ติด C305 ทั้งในระบบ eKhlaim (eClaim) และระบบอื่น (DMS for Simulate สำหรับแผนไทย, Disability, ฟื้นฟู) และวิธีอัปโหลด Excel 2 คอลัมน์ (Tran ID + เหตุผล) พร้อมข้อควรระวังเรื่อง format ของ Tran ID ต้องเป็น text ไม่ใช่ตัวเลข ประเด็นที่ 3 คือ Reactivate ของคลินิกพยาบาล — สภาการพยาบาลเพิ่งเชื่อม API กับ สปสช. เมื่อวันที่ 2 กุมภาพันธ์ 2569 จึงเพิ่งเปิดให้คลินิกพยาบาล Reactivate ได้ โดยปิดรับเที่ยงคืนวันนี้ (6 ก.พ.) มีผู้เข้า Reactivate แล้ว 2,512 หน่วยจากทั้งหมดกว่า 5,000 หน่วย

## Key Points

1. **สปสช. ยังจ่าย IP อัตรา 8,350 บาท/AdjRW ทุกสังกัด** — ยืนยันด้วย Statement จริงของ รพ.ศรีนครินทร์, รพ.ราชวิถี, รพ.ค่ายจีรประวัติ, และ รพ.เอกชนเชียงใหม่ Medical (คอลัมน์ที่ 7 ใน Statement IP)

2. **3,505 บาทคืออัตราของ สป.สธ. เท่านั้น** — เกิดจากมติที่ประชุมร่วม สปสช.–กระทรวง 19 ธันวาคม 2568 ให้หักค่าแรงภาพรวมระดับประเทศออกก่อน แล้วกันเงิน 6,326 ล้านบาทไว้เติมให้ตามเกณฑ์ สธ. ในช่วง 9.5 เดือน (ธ.ค. 68 เป็นต้นไป)

3. **วงเงิน 12,000 ล้านบาท ที่ สธ. กันไว้** ประกอบด้วย OP 4,600 ล้าน + IP 6,326 ล้าน + PP 1,000 ล้าน บริหารจัดการ 3 ระดับ: ระดับหน่วย 8,400 ล้าน, ระดับเขต 3,000 ล้าน, ระดับประเทศ 600 ล้าน

4. **2.5 เดือนแรก (ก.ค.–ก.ย. 68)** — สป.สธ. ได้รับ 8,350 แต่ยังไม่มีการกันเงินเติม ระหว่างรอบอร์ดเห็นชอบหลักเกณฑ์ โดยบอร์ดเห็นชอบ 2 ครั้ง (พฤศจิกายน + 1 ธันวาคม 2568)

5. **Statement สป.สธ. — คอลัมน์เงินเดือน = 0** เพราะหักเงินเดือนในภาพรวมระดับประเทศแล้ว (ไม่หักรายหน่วยเหมือนสังกัดอื่น) ปลายปีกระทรวงจะเกลี่ยจ่ายคืน

6. **เพิ่มเงิน IP อีก 5,450 ล้านบาท** — บอร์ดเห็นชอบให้กันงบจากก้อนอื่นมาเสริม IP สำหรับหน่วย สป.สธ. เพื่อไม่ให้รายรับต่ำกว่าปีที่แล้ว

7. **Appeal C305 ผ่าน appel349.nhso.go.th** — URL คือ `349.nhso.go.th` เข้าด้วย username เดิม ข้อมูลที่ติด C305 ตัดข้อมูล ณ วันที่ 29 มกราคม 2569 ระบบโหลดรายการล่าสุดให้หน่วยบริการดาวน์โหลด

8. **การค้นหาข้อมูล C305 แบ่งตามระบบ**: (ก) ข้อมูลใน eClaim → ดาวน์โหลด IP ใน Appeal 349 ได้เลย (ข) ข้อมูลนอก eClaim (แผนไทย/Disability/ฟื้นฟู/KTB) → ค้นใน DMS for Simulate และมีรายงาน TT305, DS305 แยกตามประเภท

9. **รูปแบบ Excel อัปโหลด C305**: 2 คอลัมน์ — คอลัมน์ A = Tran ID (ต้องแปลงเป็น Text ก่อน ห้ามเป็น Number) คอลัมน์ B = เหตุผล ชื่อไฟล์แนะนำ: `[รหัส 5 หลัก][DDMMYYYY].xlsx` ไม่มีบังคับ format ชื่อไฟล์แต่ต้องถูก format

10. **Reactivate คลินิกพยาบาล** — สภาการพยาบาลเชื่อม API กับ สปสช. เมื่อ **2 กุมภาพันธ์ 2569** (สภาอื่น 6 สภาเชื่อมก่อนแล้ว) เปิดให้ reactivate เฉพาะคลินิกพยาบาลจนถึง **เที่ยงคืน 6 กุมภาพันธ์ 2569** ยอดเข้า reactivate แล้ว **2,512 หน่วย** จากกว่า 5,000 หน่วย

11. **ประเภทบุคลากรพยาบาลใน Reactivate** — เลือก **4.2.26** สำหรับพยาบาลเวชปฏิบัติ (ให้บริการ CI ได้) หรือ **4.1** สำหรับพยาบาลวิชาชีพทั่วไป (ล้างแผล ฯลฯ เท่านั้น) ระบบ API ดึงข้อมูลจากสภาการพยาบาลอัตโนมัติ

12. **หลัง Reactivate ครบทุกวิชาชีพแล้ว** — สปสช. จะปิด Reactivate ทุกวิชาชีพ รอบถัดไปต้องรอจัดเครือข่ายพื้นที่โดย สปสช. เขตเสร็จก่อน

13. **IN19 (IN9) = เลือกประเภทบริการไม่ตรง** — ตัวอย่าง: ร้านยาเบิก CI ร้านยา แต่ไปเลือกประเภท PP แทน → ติด IN19 หลังระบบเปลี่ยนใหม่ สปสช. เคย waive ให้ในช่วงต้น แต่ภายหน้าจะไม่ waive แล้ว กำลังปรับระบบให้แสดงเฉพาะประเภทที่เหมาะกับหน่วย

14. **คลินิกแพทย์แผนไทย ค้างชำระ 3 งวด** — จะออกรายการวันจันทร์ที่ 10 กุมภาพันธ์ 2569 และจ่ายเงินวันพุธ (ประมาณ 30,000–40,000 เคส)

15. **DS027/DX027** = ติดเรื่องประเภทไม่ได้ทำ Face/Line สำหรับ Disability (กายภาพบำบัด) — ประมวลผลใหม่วันจันทร์ที่ 10 กุมภาพันธ์ 2569 | **IN22** กายภาพบำบัด — ประมวลผลใหม่สัปดาห์หน้า (จันทร์)
