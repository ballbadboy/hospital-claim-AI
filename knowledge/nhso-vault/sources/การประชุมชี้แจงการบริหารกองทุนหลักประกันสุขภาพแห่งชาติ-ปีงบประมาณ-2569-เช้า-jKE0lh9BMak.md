# การประชุมชี้แจงการบริหารกองทุนหลักประกันสุขภาพแห่งชาติ ปีงบประมาณ 2569 (เช้า)

**Channel:** สปสช. | **Duration:** 2:34:44 | **Ingested:** 2026-04-13

## Summary

การประชุมระดับประเทศเพื่อชี้แจงแนวทางการบริหารจัดการกองทุนหลักประกันสุขภาพแห่งชาติในปีงบประมาณ 2569 ซึ่งเป็นเวทีใหญ่ที่รวมผู้บริหารจากทุกหน่วยงาน ผู้แทนเขตสุขภาพ 13 เขต โรงพยาบาล สปสช.เขต และภาคีเครือข่ายสาธารณสุขทั่วประเทศ ถ่ายทอดสดผ่าน Facebook Live และ Zoom Meeting วิทยากรหลักคือ นพ.สมฤกษ์ จิงสมาน ปลัดกระทรวงสาธารณสุข, นพ.จเด็จ ธรรมทัศน์อารี เลขาธิการ สปสช., รองเลขาธิการ นพ.อรรถพร ลิ้มปัญญาเลิศ, แพทย์หญิงลฤิตยา กองคำ และนักวิชาการจาก TDRI

ช่วงเช้าประกอบด้วย 3 ส่วนหลัก: (1) การนำเสนอผลการศึกษาการประเมินการบริหารจัดการกองทุนโดย TDRI ซึ่งใช้กรอบ Triple E (Efficiency, Evidence-based, Execution) และ Principal-Agent Theory วิเคราะห์ประสิทธิภาพของบอร์ด สปสช. และกลไกการทำงาน, (2) การแลกเปลี่ยนนโยบายระหว่างปลัด สธ. และเลขาธิการ สปสช. เกี่ยวกับทิศทางระบบสุขภาพไทย 20 ปีหน้า รวมถึงนโยบาย One Region One Province One Hospital และปัญหา IP underfunding, (3) การชี้แจงรายละเอียดงบประมาณปี 2569 โดยสรุปโครงสร้างงบเหมาจ่ายรายหัว 190,000 กว่าล้านบาท และงบนอกเหมาจ่าย ทั้งหมดรวมกองทุนฯ ปี 2569 ประมาณ 265,295.5821 ล้านบาท

การประชุมครั้งนี้ถือเป็นครั้งแรกในรอบหลายปีที่จัดในรูปแบบ Policy-level briefing ร่วมกับ Detailed fund management—เดิมจะเป็นแค่การชี้แจงกติกาการเบิกจ่ายรายหมวด แต่ปีนี้เพิ่มช่วงนโยบายเพื่อให้ทุกภาคส่วนรับทราบทิศทางร่วมกันก่อนลงรายละเอียด หลังเสร็จช่วงเช้า สปสช.เขตรับหน้าที่นำไปชี้แจงต่อในพื้นที่

## Key Points

- **งบกองทุนหลักประกันสุขภาพแห่งชาติปี 2569 รวม ~265,295.58 ล้านบาท** แสดงบนสไลด์ขณะชี้แจงภาพรวม
- **งบเหมาจ่ายรายหัว (หมวด 1) ~190,000 กว่าล้านบาท** ดูแลประชากรสิทธิ์บัตรทอง ~47 ล้านคน เพิ่มจากปี 68 ราว 16,000 กว่าล้านบาท
- **งบนอกเหมาจ่าย (กลุ่มเฉพาะ) HIV/AIDS ปี 69 ~4,000 กว่าล้านบาท**
- **งบผู้ป่วยไตวายเรื้อรัง ปี 68 ~13,000 กว่าล้านบาท → ปี 69 ~16,000 กว่าล้านบาท** เพิ่มขึ้นจากนโยบาย "ฟอกไตฟรีทุกแห่ง" งบเตรียมรองรับ ~50,000 ราย
- **งบป้องกันโรคเรื้อรัง (NCD) ~1,700 กว่าล้านบาท** ครอบคลุม DM, HT, จิตเวชเรื้อรัง, Asthma, COPD — มี PPNCD Global Budget ระดับประเทศใหม่
- **งบปฐมภูมิ (PCC/หมอครอบครัว) ~3,700 กว่าล้านบาท** จ่ายแบบ Global Budget ไม่ตัดจากค่าหัวโรงพยาบาล
- **งบ Long-term Care (Intermediate/LTC) ~5,000 กว่าล้านบาท** ปี 2569
- **งบกองทุนท้องถิ่น อปท. 45 บาท/หัวประชากร**
- **กระทรวงสาธารณสุขเสนอเป้าหมายอัตราจ่าย IP 10,000 บาท/RW สำหรับปี 70** และในระยะยาว 13,000 บาท/RW (ปัจจุบัน ~8,000–8,350 บาท/RW)
- **TDRI ประเมินกองทุนฯ ด้วยกรอบ Triple E**: Efficiency / Evidence-based / Execution และ Principal-Agent Theory — บอร์ดปีงบ 4–6 มีกรรมการมากกว่า 3 คณะ ทำให้เกิด Role Conflict
- **ปัญหาเชิงโครงสร้าง**: งบขาขึ้น–ขาลงไม่สัมพันธ์กัน, บางกรณีต้องใช้งบกลาง, disclosure ข้อมูลยังไม่เพียงพอ
- **นโยบาย One Region One Province One Hospital** ของ สธ.: ใช้ทรัพยากรร่วมกันทั้งเขต/จังหวัด ลดเวลารอคอย เพิ่มห้องผ่าตัด/ICU ในโรงพยาบาลอำเภอ
- **ปัญหา IP Underfunding**: โรงพยาบาลกระทรวงให้บริการบัตรทองแต่รายได้ ~56% ติดลบ โรงพยาบาลขาดทุนจากกองทุนนี้ บุคลากรสมองไหล
- **กลไก 7x7**: กลไกระดับประเทศรับประเด็นระหว่างปี ทำงานร่วม สปสช.–เขตสุขภาพ–กระทรวง
- **งบ PP ปี 69**: แยกออกจากเหมาจ่ายรายหัวมาแล้ว 2–3 ปี ปีนี้เพิ่ม PPNCD Global Budget
- **กลไก Electronic Audit / AI-Audit**: เป้าหมายปี 2569 พัฒนาระบบ Electronic Audit ลดภาระหน่วยบริการ — Pre & Post Audit ครอบคลุม IP, คลินิกบริกรรม, บริการ HD, CAG/PCI, CA anywhere
- **UHOSNET**: ระบบโครงข่ายโรงพยาบาล เชื่อมข้อมูลระดับเขต/จังหวัด/ท้องถิ่น รองรับนโยบาย One Region
- **นโยบายป้องกัน NCD ในอนาคต**: เน้น self-test (HIV, sugar) แทนการลงชุมชน, Telemedicine, ลด P&P แบบ passive
- **งบ P&P เปลี่ยนรูปแบบ**: จาก "เดินเข้าโรงพยาบาลมาเจาะเลือด/ฉีดวัคซีน" → แจกชุดตรวจให้ประชาชนทำเอง, ผลลัพธ์ดีกว่าในกรณี HIV
- **สปสช. 20 ปี**: เลขาธิการ นพ.จเด็จ ยืนยันระบบมีปัญหาตลอด แต่แก้ได้ ไม่เคยไร้ปัญหาแม้แต่วันเดียว
- **งบขาขึ้นปี 70**: สปสช.ต้องการ input จากพื้นที่ ไม่ใช่เพียงตารางส่วนกลาง — เขตส่งเป้าหมายขึ้นมา สำนักงานคำนวณงบเพื่อเจรจากับสำนักงบประมาณ
