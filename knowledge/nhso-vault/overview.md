# สปสช — Knowledge Wiki Overview

สำนักงานหลักประกันสุขภาพแห่งชาติ (สปสช) คือหน่วยงานภายใต้กระทรวงสาธารณสุข ทำหน้าที่บริหารจัดการกองทุนหลักประกันสุขภาพแห่งชาติ เพื่อให้คนไทยทุกคนเข้าถึงบริการสาธารณสุขที่มีคุณภาพ

---

## โครงสร้างระบบหลักประกันสุขภาพไทย

```
รัฐบาล / กระทรวงสาธารณสุข
    │
    ▼
สปสช (บริหารกองทุน)
    │
    ├── บัตรทอง / UC Scheme  ← คนไทยทั่วไป
    ├── ประกันสังคม          ← ลูกจ้างเอกชน (สปส.)
    └── สิทธิ์ข้าราชการ     ← ข้าราชการ (กรมบัญชีกลาง)
          │
          ▼
    โรงพยาบาล / หน่วยบริการ
    (ส่งเบิกผ่าน e-Claim)
```
